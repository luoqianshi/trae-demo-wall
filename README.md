# TRAE Demo Wall

TRAE AI 创造力大赛初赛作品一站式体验平台

## 技术栈

- Vue 3 + Vite + TailwindCSS
- Pinia 状态管理
- Canvas 2D 粒子动画
- Python 爬虫
- GitHub Pages 部署

## 本地开发

```bash
npm install
npm run dev
```

## 爬取数据

```bash
pip install -r crawler/requirements.txt
npm run crawl
```

## 构建

```bash
npm run build
```
