# 高中生 AI 学习陪跑教练 - 使用说明

## 一、功能概览

| 功能 | 说明 |
|------|------|
| 📷 拍照上传 | 拍摄错题照片，AI 自动识别题目内容 |
| ✏️ 手动录入 | 直接输入错题信息 |
| 🧠 AI 错因诊断 | 智能分析错误原因，给出复习建议 |
| 📖 分步引导 | 逐步引导理解解题思路 |
| 🎯 变式训练 | 生成同类型练习题，检验是否真正学会 |
| 📅 复习计划 | 科学安排复习时间（当天、3天、7天、考前） |
| 🖨️ 打印错题本 | 按科目筛选，生成 A4 错题本 |
| 📊 学习画像 | 了解自己的学习状况和薄弱点 |

---

## 二、快速开始

### 1. 启动应用

**方式一：双击启动脚本**
1. 解压压缩包到文件夹
2. 双击「启动服务.bat」
3. 等待依赖安装完成（首次）
4. 浏览器访问 http://localhost:5173

**方式二：命令行启动**
1. 进入项目文件夹
2. 运行 `npm run dev`
3. 浏览器访问 http://localhost:5173

### 2. 首次使用

首次进入会有一道示例错题，可以：
- 点击「查看详情」了解功能
- 点击「删除」删除示例错题
- 点击「录入错题」添加自己的错题

---

## 三、核心功能使用

### 3.1 录入错题

**方式一：拍照上传**
1. 点击「录入错题」→「拍照上传」
2. 拍摄或选择错题图片
3. AI 自动识别题目内容
4. 核对识别结果，修改错误之处
5. 可补充：错误答案、困惑描述
6. 点击「确认录入」保存

**方式二：手动录入**
1. 点击「录入错题」→「手动录入」
2. 填写：科目、知识点、题目内容
3. 可选填：错误答案、困惑描述
4. 点击「保存并开始 AI 诊断」

### 3.2 错因诊断与学习

进入错题详情页后：

1. **查看 AI 错因诊断**
   - 分析错误原因
   - 复习建议

2. **开始分步引导**
   - 点击「开始引导」进入引导模式
   - AI 逐步提示，引导你独立解题
   - 点击「继续引导」查看下一步提示

3. **变式训练**
   - 生成 3 道同类型练习题
   - 提交手写答案
   - AI 批改判断对错
   - 正确 2 道以上可进入「待复测」状态

### 3.3 复习计划

每道错题会自动生成复习计划：

| 复习节点 | 复习内容 | 目的 |
|---------|---------|------|
| 当天订正 | ✅ 原题 | 当天巩固 |
| 3天后复测 | ✅ 原题 | 短期记忆强化 |
| 7天后综合复习 | ✅ 变式题 | 检验迁移能力 |
| 考前复习 | ✅ 原题 + 变式题 | 综合检验 |

进入「复习计划」页面：
- 查看今日复习任务
- 点击任务进入对应错题
- 完成任务后标记「已完成」

### 3.4 打印错题本

1. 进入「错题生命线」页面
2. 点击「🖨️ 打印错题本」
3. 选择要打印的科目（可多选）
4. 点击「🖨️ 打印」或「📥 下载 HTML」

**打印选项**：
- 按科目筛选（数学、物理、化学等）
- 支持全选/清空
- 打印内容包含：题目、错误答案、困惑、订正区域

### 3.5 学习画像

进入「学习画像」页面：
- **错题统计**：各科目错题数量
- **错因分布**：错误原因的占比
- **薄弱知识点**：高频出错的知识点
- **学习建议**：根据你的情况给出的个性化建议

## 四、数据管理

### 4.1 数据存储

所有数据保存在浏览器本地（localStorage），不会上传到服务器。

### 4.2 导出备份

1. 进入「设置」页面
2. 点击「导出数据」
3. 保存 JSON 文件到本地

### 4.3 导入数据

1. 进入「设置」页面
2. 点击「导入数据」
3. 选择之前导出的 JSON 文件

### 4.4 清空数据

进入「设置」页面，点击「清空所有数据」可删除所有错题和复习计划（不可恢复）。

## 五、常见问题

### Q1：AI 识别显示 Mock 模式？
说明无法连接豆包 API，请检查：
- 电脑能否访问：https://ark.cn-beijing.volces.com/api/v3
- 网络是否正常

### Q2：打印时没有显示科目选择器？
刷新页面后重试。

### Q3：如何知道复习计划的时间？
进入「复习计划」页面，已过期的任务会显示红色提醒。

### Q4：数据会丢失吗？
- 删除浏览器缓存会删除数据
- 建议定期导出备份
- 换电脑需要重新导入数据
