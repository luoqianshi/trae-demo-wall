# Michael 的趋势交易艺术

> 基于 TRAE AI 的利弗莫尔法则 A 股交易系统 · TRAE AI 创造力大赛参赛作品

**世界很大，放手去造** — 用一句话触发"选股 → 买入 → 持有 → 卖出"全流程，把交易大师的方法论装进一个对话窗口。

---

## 一句话介绍

把杰西·利弗莫尔的交易法则（关键点突破 + 三阶段建仓 + 三重止损 + 趋势跟踪）系统化为 **4 大独立模块 + 2 大跨模块工作流**，借助 TRAE 的 SKILL / MCP / Agent 三件套，让 AI 真正成为交易秘书，而不是代码生成器。**系统绝不自动交易，所有买卖建议必须人工确认。**

---

## 核心亮点（评委导向）

### 1. 真实痛点 · 系统化解法
A 股散户贡献 80% 交易量，却拿走不足 30% 利润。问题不在选股，在**执行**。本系统把"该买/该卖/该加仓"的决策从情绪驱动改为规则驱动，30+ 阈值全部参数化可调。

### 2. TRAE 三件套的"非典型"用法
- **SKILL.md**：把交易工作流封装成可被自然语言触发的模块化任务（"帮我巡检" → 自动跑完 5 个 Step）
- **TDX MCP**：AI 直接调用 `tdx_quotes` / `tdx_kline` / `tdx_indicator_select` 拿 A 股实时数据，无需写爬虫
- **Agent 自动化**：每日 15:30 收盘后自动触发巡检任务，全量检查并等待人工确认

### 3. 完整的交易闭环
| 模块 | 输入 | 输出 |
|------|------|------|
| 选股 | 429 只自选股 | Top 5 候选 + 关键点信号 |
| 买入 | 候选标的 | 入场五条件验证 + 三阶段建仓计划 |
| 持有 | 持仓 + 行情 | 止损检查 + 加仓/移动止损建议 |
| 卖出 | 触发条件 | 8 类退出决策 + 禁忌清单 + 冷静期 |
| 巡检 | 自动触发 | 结构化报告 + 待确认建议 |
| 回测 | 历史 K 线 | 收益/夏普/胜率/盈亏比 |

### 4. 风控铁律（不是花架子）
- 三重止损：单日 -7% / 累计 4 日 -10% / 跌破 MA55
- 高位回撤锁利：盈利后回撤 5% 上移止损至成本，12% 全清
- 8 条禁忌清单：不抄底 / 不补仓亏损 / 不追高 9.5% 以上 / 不同时持 5 只以上...
- **零自动交易**：所有建议必须人工 [确认/暂缓/调整/跳过]

## 快速开始

### 环境要求
- Python 3.10+
- Windows / macOS / Linux
- （可选）Tushare Pro Token 用于实时行情

### 启动步骤

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置环境变量（可选，无 Token 也可演示离线数据）
copy .env.example .env
# 编辑 .env 填入 TUSHARE_TOKEN

# 3. 一键启动
start.bat            # Windows
# 或
python -m backend.api.app
```

启动后访问：
- **本机**：http://127.0.0.1:5000
- **局域网**：http://<本机IP>:5000（手机也可访问）

## 设计哲学

> "关键点就是价格告诉你阻力最小的方向。当价格突破关键点，就跟上去；当它没能突破，就等待。"
> —— 杰西·利弗莫尔

> "赚大钱的秘诀不在于频繁操作，而在于看对方向后坐得住。"

本系统的核心不是预测，而是**响应**：用规则识别关键点，用纪律执行止损，用耐心持有趋势。AI 的角色是**执行秘书**，不是预言家。
