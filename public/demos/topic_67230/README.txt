ColorSnap Demo 使用说明
==========================

1. 解压 ColorSnap_Demo.zip。

2. 推荐使用 Chrome 浏览器打开。

3. 双击 index.html 即可体验主要交互流程。

4. 如果浏览器限制部分功能（如相机权限），请使用本地服务器打开：

   npx serve .
   或
   python -m http.server 8080

   然后访问：
   http://localhost:8080

5. Demo 主要流程：
   Daily Pack 抽取今日色卡 → 现实寻色 → 拍照 / 模拟拍照 → 选择照片作为卡背 → 生成双面色卡 → 保存图鉴 → 发布地图 / 地图捕获。

6. 相机功能说明：
   真实相机需要浏览器权限，并且通常需要 localhost 或 HTTPS 环境。
   如果相机无法打开，可使用上传照片或模拟拍照继续体验完整流程。

7. 首次打开会显示新手引导（3 页），可跳过。
   如需重新查看，进入「我的」页面点击「玩法说明」。

8. 网络资源说明：
   地图底图使用 Leaflet + OpenStreetMap，需要联网加载。
   模拟照片使用 Unsplash 图床，需要联网加载。
   如果离线打开，地图和模拟照片可能无法显示，但不影响核心交互流程。
