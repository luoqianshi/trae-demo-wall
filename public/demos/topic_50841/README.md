## 快速开始

**无需安装任何环境，无需后端服务，双击即可体验。**

1. 解压 ZIP 文件
2. 用浏览器（推荐 Chrome / Edge）打开 `tonight-eat-what-demo.html`
3. 首次打开会显示 3 步引导教程，跟随引导即可了解核心操作
4. 点击首屏的「一键填充演示数据」按钮，瞬间可视化所有报表和成就

## 核心功能

### 食材选择与 AI 推荐
- 56 种食材，按蔬菜、肉类、蛋奶、主食、调料 5 大分类展示
- 两阶段推荐算法：
  - 第一阶段（本地粗排）：主食材匹配 70% + 调料匹配 30%
  - 第二阶段（AI 精排）：口味偏好 + 营养均衡 + 时令因素 + 今日已摄入量综合评分
- 支持口味筛选：清淡、红烧、快手、汤品、辣味
- 推荐结果标注匹配度、缺失调料、营养标签

### 菜谱详情
- 56 道家常菜谱，每道含完整步骤和烹饪技巧
- 营养数据：热量、蛋白质、脂肪、碳水化合物
- 智能提示：缺少哪些调料，可否用替代品
- 收藏功能：常吃菜谱一键收藏

### 营养追踪系统
- 每日营养摄入统计（热量、蛋白质、脂肪、碳水）
- 本周摄入趋势图表（ECharts 可视化）
- 个人营养目标设置：均衡 / 减脂 / 增肌 / 自定义数值
- 选食材页实时显示今日已摄入热量提示条
- 营养感知推荐：根据今日已摄入量调整推荐权重

### 三级营养报告
- **周报**：本周饮食分析，含评级、日均摄入、最高/最低日、AI 建议
- **月报**：月度统计，含上月对比数据，展示进步趋势
- **季报**：季度汇总，3 个月柱状对比图，长期趋势分析
- 所有报告支持导出为 PNG 图片（纯 Canvas 实现，无需外部库）
- 支持分享到微信、微博等社交平台（Web Share API + 剪贴板降级）

### 成就系统
- 10 个成就徽章：
  - 常规 6 个：初出茅庐、坚持一周、月度达人、季度王者、营养均衡、美食探索家
  - 隐藏 4 个：深夜食堂、五彩缤纷、轻食达人、蛋白之王
- 解锁成就时弹出庆祝动画 + Toast 提示
- 每个成就可生成专属分享海报（带二维码，扫码打开 Demo）
- 总览海报：展示所有已解锁成就的拼图

### 评委体验优化
- 3 步引导教程（首次打开自动触发，可跳过）
- 首屏「今日推荐」：根据季节和时间推荐 5 道菜，打开即有内容
- 一键填充演示数据：84 天营养历史 + 7 个成就 + 25 道菜谱 + 5 个收藏
- 数据持久化：localStorage 存储，关闭浏览器不丢失

## 技术架构

### 技术栈
- 纯原生 HTML + CSS + JavaScript（零框架依赖）
- ECharts：数据可视化图表
- Canvas API：图片导出和海报生成
- localStorage：数据持久化

### 技术亮点
- **单文件自包含**：所有代码、数据、样式集中在 1 个 HTML 文件中，双击即开
- **纯 Canvas 图片导出**：不依赖 html2canvas，直接用 Canvas API 绘制报告卡片
- **纯 JS 二维码生成**：不依赖 QRCode.js，自实现 25x25 模块 QR 码
- **两阶段推荐算法**：本地粗排 + AI 精排，支持低卡模式和营养感知模式
- **跨天数据归档**：自动按日期归档营养历史，支持周/月/季聚合统计

### 数据规模
- 56 种食材（5 分类，含 emoji 图标）
- 56 道菜谱（含步骤、技巧、营养数据、标签）
- 10 个成就（6 常规 + 4 隐藏）
- 6 种食材颜色分类（红/绿/黄/白/紫/棕）
