# 地球村超级联盟 Demo 使用说明

## 方式一：直接打开（推荐现代浏览器）
双击 demo.html 即可在浏览器中打开。

注意：由于浏览器安全策略，部分ES模块功能可能需要通过HTTP服务器访问。

## 方式二：本地HTTP服务器（推荐，功能最完整）

### Windows用户：
双击 启动服务器.bat，然后在浏览器中访问 http://localhost:8000

### 或者手动启动：
使用 Python:
`
python -m http.server 8000
`

使用 Node.js:
`
npx serve
`

## 包含角色
- 消费者
- 司机
- 骑手
- 餐饮商家
- 超市商家
- 电商商家
- 广告主
- 服务商家

## 功能说明
所有角色的我的页面均支持页面模式导航，点击菜单项可进入对应页面，左上角返回按钮可返回。
