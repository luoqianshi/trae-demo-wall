# 周末灵感生成器 · Demo 说明文档

> TRAE AI 创造力大赛 · 生活娱乐赛道参赛项目

## 一句话介绍

输入城市、预算、心情，AI 为你生成 3 套差异化的周末活动方案，含分时段时间线、真实地点推荐、预算汇总，可一键生成分享图。

## 核心功能演示流程（5分钟内）

### 1. 偏好输入（30秒）
- 选择城市（支持16城，可点击"📍定位"自动检测）
- 选择预算区间（0-100 / 100-300 / 300-500 / 500+ 元/人）
- 点击"获取实时天气"自动适配天气
- 选择心情标签（放松/探索/社交/运动/文艺）
- 选择同行人（独自/情侣/亲子/朋友）

### 2. 方案生成（10-15秒）
- 点击"生成方案"按钮
- 显示加载动画和趣味文案
- 可随时点击"取消"中断生成
- 生成完成后展示 3 套差异化方案

### 3. 方案展示与交互（2分钟）
- 顶部标签切换 A/B/C 方案
- 查看分时段时间线（上午/下午/晚上）
- 每个活动含时间、名称、地点、费用、交通方式
- 点击"换"按钮替换单个活动
- 点击"收起/展开详情"折叠时间线
- 查看备选建议和总预算汇总

### 4. 收藏与历史（1分钟）
- 点击"收藏"保存满意方案
- 刊头"历史"按钮查看生成历史（最多20条）
- 点击历史项可回看方案

### 5. 分享图片（30秒）
- 点击"生成分享图"按钮
- Canvas 绘制杂志风竖版分享图
- 点击下载保存为 PNG

## 技术亮点

| 特性 | 实现 |
|------|------|
| AI 方案生成 | LLM API + Prompt 工程，支持 DeepSeek/OpenAI 等 |
| 多层降级策略 | LLM → 规则引擎 → 模板兜底 → 保底模板 |
| 天气智能适配 | Open-Meteo API，雨天/高温/低温自动推荐室内活动 |
| GPS 定位 | 浏览器 Geolocation API + Haversine 距离匹配最近城市 |
| 16 城覆盖 | 北上广深 + 杭成南武西渝苏长沙津青厦昆，共290+ POI |
| 质量校验 | Schema 校验 + 预算合理性 + 地点真实性 + 天气适配性 |
| 请求缓存 | 偏好 hash 缓存，相同偏好毫秒级返回 |
| 分享图片 | Canvas 2D 绘制杂志风竖版图，高 DPI 适配 |
| 本地持久化 | localStorage 收藏(50条) + 历史(20条) + 偏好回填 |

## 快速开始

```bash
cd weekend-app
npm install
npm start
# 打开 http://localhost:3000
```

## AI 配置（可选）

未配置 API Key 时使用规则引擎生成方案（可体验完整功能）。
配置后启用 AI 生成：

1. 点击页面右上角"规则引擎"按钮
2. 填入 API Key / Base URL / 模型名
3. 点击"测试连接"验证
4. 保存后即启用 AI 生成

支持的服务商：DeepSeek / OpenAI / 通义千问 / Moonshot

## 部署

### 本地运行
```bash
npm install && npm start
```

### 线上部署（Railway / Render / Vercel）
```bash
# 设置环境变量
PORT=3000
LLM_API_KEY=sk-xxx  # 可选，不填用规则引擎
LLM_API_BASE=https://api.deepseek.com
LLM_MODEL=deepseek-chat
# ADMIN_TOKEN=your-token  # 可选，保护管理端点
```

## 技术栈

- **前端**：HTML + CSS + 原生 JavaScript（无构建步骤）
- **后端**：Node.js + Express
- **AI 层**：可配置 LLM API + Prompt 工程 + 规则引擎 + 模板兜底
- **数据层**：JSON 文件 + 内存缓存 + localStorage
- **外部 API**：Open-Meteo（天气）、LLM API（方案生成）
