周末灵感生成器 · The Weekend Dispatch
TRAE AI 创造力大赛 · 生活娱乐赛道
=============================================

【快速体验】
1. 确保已安装 Node.js 18+（https://nodejs.org）
2. 解压本压缩包
3. 在解压后的 weekend-app 文件夹中打开终端
4. 运行：npm install
5. 运行：npm start
6. 浏览器打开：http://localhost:3000

【功能说明】
- 输入城市/预算/天气/心情/同行人，30秒生成3套周末方案
- 支持16个城市，GPS自动定位
- 实时天气获取（Open-Meteo API）
- AI方案生成（需配置API Key，否则使用规则引擎）
- 方案收藏、历史记录、分享图生成

【可选：启用AI生成】
不配置API Key时使用规则引擎生成方案（可体验完整功能）。
如需启用真实AI生成：
1. 启动服务后，点击页面右上角"规则引擎"按钮
2. 填入 API Key / Base URL / 模型名
3. 点击"测试连接"验证后保存

支持的服务商：DeepSeek / OpenAI / 通义千问 / Moonshot

【技术栈】
- 前端：HTML + CSS + 原生 JavaScript（无构建步骤）
- 后端：Node.js + Express
- AI层：可配置 LLM API + 规则引擎 + 模板兜底

【演示流程】
1. 选择城市（或点击"📍定位"自动检测）
2. 点击"获取天气"自动适配
3. 选择预算/心情/同行人
4. 点击"生成方案"
5. 查看方案、切换标签、替换单个活动
6. 生成分享图、收藏、查看历史
