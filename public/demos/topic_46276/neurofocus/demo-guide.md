### v29 交互体验要点

- **佩戴即监测**：点击「模拟戴上耳机」后自动完成耳机连接、个人基线校准并开始实时监测

- **深蓝渐变 hero 卡**：专注页顶部深蓝渐变卡片，白色文字 + EEG 线条绘制动画 + 渐变分数

- **场景符号动画**：Focus Mix 波形线 / White Noise 噪声点 / Breath Guide 呼吸圆环 / Calm Ambient 柔和光斑

## 评审 60 秒体验路径

评审可在 60 秒内按以下 6 步完成完整体验，建议按此顺序录制演示：

1. 点击"开始实时体验" → 自动完成佩戴检测、个人基线建立并开始实时监测（`startRealtimeExperience()` 高层入口）
2. 查看实时 EEG 趋势
3. 连接音乐软件
4. 查看推荐声音场景
5. 触发 Brain Break
6. 生成每日脑状态报告

每一步对应首页「60 秒评审体验路径」模块中的一个节点，跑完后即可在报告页查看完整的脑状态与音乐恢复摘要。

### 扩展体验：自动睡眠脑状态洞察

评审在生成每日脑状态报告时，报告中会自动包含睡眠脑状态摘要：
1. 生成报告后，查看"自动睡眠脑状态摘要"模块
2. 查看夜间记录窗口、入睡过渡时间、睡眠稳定度
3. 查看深睡趋势参考和 REM-like 趋势参考
4. 查看夜间 EEG 频段趋势和睡眠状态时间线
5. 查看推荐睡眠声音场景和科学边界说明

## 截图技巧

1. **先进入截图模式**：点击右下角「评审快速体验」→「截图模式」，隐藏悬浮按钮和 Toast
2. **使用浏览器全屏截图**：Chrome DevTools → Capture full size screenshot
3. **确保数据已加载**：等待专注会话运行几秒后再截图，或使用「生成示例报告」
4. **推荐分辨率**：1280×800 或更高，保证清晰度
5. **宽屏布局**：3 张深色核心模块卡片在宽屏（>768px）下横向排列，窄屏下纵向排列
6. **完整流程截图**：建议使用「一键体验完整流程」跑通后再截图，数据最完整
