📖 使用教程

① 密码工具箱

  • 移除编辑保护：上传带保护的 Excel → 点击"开始处理" → 下载已解锁文件。原理：读取 OOXML 的 sheetProtection/workbookProtection 节点并移除。
  • 暴力穷举打开密码：选择模式 → 设置密码长度范围 → 开始穷举。仅支持 1~8 位纯数字。

② 数据整理工具

  • 合并：上传多个 Excel → 自动按表头合并到一张新表。
  • 拆分：指定一个列名，按该列不同值拆分成多个文件。
  • 去重：按指定列或整行去重。
  • 对比：上传两份文件，按关键字对比差异。

③ 格式转换

  • Excel → CSV/JSON：直接导出可读格式。
  • CSV/JSON → Excel：结构化数据转成 .xlsx 文件。

④ 辅助工具

  • 日期格式化、文本提取、统一行高列宽、生成目录（含超链接）。
