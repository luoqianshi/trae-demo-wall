外设开关

连接蓝牙设备，让实体物件成为你的专注开关

蓝牙外设
通过 Web Bluetooth 连接支持的 BLE 设备
推荐方案
小米碰碰贴
BLE NFC
贴在桌面任意位置，轻触即触发开关。无需充电，一枚纽扣电池可用一年。
1
购买小米碰碰贴（MI/NFC 标签款）
2
贴在办公桌显眼处，如显示器边框或桌面
3
在本页点击「扫描设备」并配对碰碰贴
4
碰一下 = 切换专注/休息，比点手机更快
Labubu IP 公仔
创意玩法
把喜欢的 Labubu 公仔变成专属专注开关——内置 BLE 模块，拿起放下的仪式感拉满。
1
准备一个 Labubu 公仔（任意款）
2
在内底缝入微型 BLE 加速度模块（如 DY-Sensor 或 ESP32-BLE）
3
模块配置为检测震动/翻转时发送切换信号
4
配对后，拍一拍 Labubu 或翻转它就能切换状态
ESP32 自定义开关
极客方案
用 ESP32 开发板 + 任意物理开关，打造完全个性化的外设。适合喜欢 DIY 的独立工作者。
1
准备 ESP32 开发板 + 物理开关/传感器
2
烧录 BLE HID 固件，配置开关触发广播
3
用 3D 打印或手工制作外壳，融入桌面美学
4
连接后按一下物理开关即可切换专注状态
基于 Web Bluetooth API，支持 Chrome / Edge 浏览器。
Safari 需使用 App Store 应用实现。
