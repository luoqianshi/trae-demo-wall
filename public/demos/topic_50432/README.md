# 社区共享工具地图

## 项目简介
社区共享工具地图是一个基于H5的社区工具共享平台，支持工具发布、预约借用、信用评价等功能，适配手机端和微信小程序WebView。

## 功能特性

### 核心功能
- 🗺️ **工具地图** - 地图模式展示社区内共享工具
- 📋 **工具列表** - 列表模式浏览，支持分类筛选和排序
- 🔧 **工具分类** - 电动工具、手动工具、园艺工具、清洁工具、厨房电器等
- 📍 **距离计算** - 按距离当前楼栋排序
- 📅 **预约借用** - 完整的预约状态流转
- ⭐ **信用评价** - 多维度互评（守时、工具爱护、沟通顺畅）
- 👤 **个人中心** - 信用分展示、借用记录管理

### 预约状态流转
```
可借 → 已预约(待确认) → 待取件 → 使用中 → 已归还
         ↓              ↓
       已取消         已拒绝
```

### 信用评分体系
- **守时** (35%) - 约定时间内完成取还
- **工具爱护** (40%) - 工具归还时的完好程度
- **沟通顺畅** (25%) - 交流及时、态度友好

## 技术栈
- HTML5 + CSS3 + JavaScript (ES6+)
- 响应式设计，移动端优先
- QRCode.js (二维码生成)
- 无框架依赖，纯静态页面

## 快速开始

### 本地运行
```bash
# 方式一：使用Python
python -m http.server 8080

# 方式二：使用Node.js
npx http-server -p 8080

# 方式三：使用PHP
php -S localhost:8080
```

然后访问 http://localhost:8080

### 部署上线

#### 1. 静态页面托管（推荐）
将 `index.html` 上传到任意静态托管平台：
- **GitHub Pages** - 免费，适合开源项目
- **Vercel / Netlify** - 免费额度充足，部署简单
- **阿里云OSS / 腾讯云COS** - 国内访问快
- **Nginx / Apache** - 自有服务器

#### 2. 部署步骤
1. 将 `index.html` 上传到服务器
2. 配置HTTPS（微信小程序要求必须HTTPS）
3. 配置域名解析
4. 测试访问

#### 3. 微信小程序接入
本H5页面可直接嵌入微信小程序的 `web-view` 组件中使用：

```xml
<!-- 小程序页面 -->
<web-view src="https://your-domain.com/index.html"></web-view>
```

**注意事项：**
- 域名必须配置为小程序业务域名
- 必须使用HTTPS协议
- 域名必须在微信公众平台备案
