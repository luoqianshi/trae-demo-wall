# 数字员工办公区体验版

打开方式：

1. 解压本 Zip。
2. 用浏览器打开 `index.html`。

如果浏览器安全策略限制本地文件访问，也可以在解压目录执行 `python3 -m http.server 8765`，然后访问 `http://localhost:8765`。

## 体验内容

这个 Zip 是评审可直接打开的静态体验版，内置演示 Job 数据，可体验：

- 3D 数字员工办公区；
- 员工状态、岗位和思考文本；
- 任务中心；
- Job 流转图；
- 节点详情和员工分工；
- 演示最终报告与脚本产物下载。

## 关于本地 CLI

完整版本会通过 Python Sidecar 调用本机 CLI / Agent 工具。静态 HTML 环境不会直接执行本地 CLI，这是为了保证评审打开即能体验，不受本地环境、权限和进程配置影响。
