VersePC

一个懂 AI 的 Minecraft 启动器。让不懂技术的玩家也能轻松管理游戏版本、安装整合包， 还能跟内置 AI 助手聊天、一键优化内存，玩游戏更省心。

核心功能

VersePC 不只是启动游戏，更是你的 Minecraft 智能管家

🎮

一键启动 & 版本管理

自动检测已安装的 Minecraft 版本，支持 Forge/Fabric 等主流模组加载器，一键切换不同版本，告别手动配置。

📦

整合包一键导入

支持 CurseForge、Modrinth 等平台整合包直接导入安装，自动解析依赖、下载模组，省去繁琐的手动配置步骤。

🤖

内置 AI 助手

遇到游戏问题不用到处搜攻略，直接问内置 AI。支持联网搜索、代码生成、故障排查，还能帮你写模组配置。

⚡

一键内存优化

玩游戏前一键清理系统内存，释放更多资源给 Minecraft，减少卡顿。底层调用 Windows API，安全高效。

🔧

JVM 参数智能调优

根据你的电脑配置自动推荐最优 JVM 参数，支持 G1GC 优化、内存预热等高级功能，让游戏运行更流畅。

🎨

精美现代化 UI

采用 Fluent Design 设计语言，支持自定义主题、背景图，界面简洁美观，操作直观易懂。

技术栈

全栈 Electron 桌面应用，前后端分离架构

Electron Node.js Vanilla JS HTML5 / CSS3 PowerShell Windows API AI 对话引擎
