# 智链笔记（SmartLink Notes）项目计划

## 项目概述

智链笔记是一款基于浏览器的Web应用，核心功能为实时语音转写、AI智能摘要、自动提取知识点并构建可视化知识图谱。本项目将基于现有的HTML demo进行升级，打造一个完整的、可演示的产品原型。

## 技术方案

### 架构选择
- **前端**: 原生HTML/CSS/JavaScript + TailwindCSS + D3.js
- **后端**: Python FastAPI（使用D盘的uv进行依赖管理）
- **AI能力**: 集成OpenAI API或国内AI API（如讯飞星火、百度文心一言）
- **语音识别**: Web Speech API（浏览器原生）+ 备用方案

## 运行方式

### 开发环境
```bash
# 进入项目目录
cd SmartLink Notes

# 启动后端服务（使用uv）
cd server
uv run uvicorn main:app --reload --host 0.0.0.0 --port 8000

# 前端直接打开index.html或使用静态服务器
```

### 演示模式
- 提供模拟数据，无需真实语音输入
- 点击"开始记录"自动播放预设转写内容
- AI功能使用模拟响应，展示界面交互

## 关键API设计

### 后端API

| 接口 | 方法 | 功能 |
|------|------|------|
| /api/summary | POST | 生成摘要 |
| /api/keywords | POST | 提取关键词 |
| /api/graph | POST | 生成知识图谱数据 |

### 前端API
- Web Speech API: 语音转文字
- localStorage: 笔记持久化

## 注意事项

1. **AI API密钥**: 需要在环境变量中配置API密钥，不硬编码
2. **跨域处理**: FastAPI需要配置CORS
3. **演示模式**: 确保无API密钥时也能正常演示（使用mock数据）
4. **浏览器兼容性**: Web Speech API在不同浏览器支持度不同，提供降级方案
