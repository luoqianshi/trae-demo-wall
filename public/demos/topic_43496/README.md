# 智链笔记 SmartLink Notes

> 智能记录，知识互联。让每一次学习都留下清晰的脉络。

智链笔记是一款基于浏览器的Web应用，核心功能为实时语音转写、AI智能摘要、自动提取知识点并构建可视化知识图谱。

## 🏆 项目背景

本项目是 **TRAE-AI创新大赛** 参赛作品，旨在解决学生和职场人士在课堂/会议中"记了前面漏了后面"的普遍困扰，打造一款能"边听边懂、边记边联"的智能工具。

## ✨ 核心功能

| 功能模块 | 描述 |
|---------|------|
| **实时语音转写** | 模拟实时语音转文字，自动添加时间戳和说话人标记 |
| **AI智能摘要** | 自动生成会议/课堂摘要，提取关键知识点和关键词 |
| **知识图谱** | 使用D3.js构建力导向图，可视化展示知识关联结构 |
| **笔记编辑** | 富文本编辑器，支持大纲导航、格式编辑和自动保存 |
| **场景切换** | 支持课堂模式和会议模式，各有专属配置和演示数据 |

## 🎯 目标用户

- **核心用户**：高校学生、职场新人及中层管理者、线上课程学习者
- **扩展用户**：行政人员、企业培训员工、听障人士

## 🛠️ 技术栈

### 前端
- **HTML5** - 页面结构
- **CSS3 + TailwindCSS** - 样式设计
- **JavaScript (ES6+)** - 前端逻辑
- **D3.js** - 知识图谱可视化
- **Phosphor Icons** - 图标库

### 后端
- **Python 3.12** - 编程语言
- **FastAPI** - Web框架
- **Uvicorn** - ASGI服务器
- **uv** - 依赖管理工具

## 🚀 快速开始

### 环境要求

- Python 3.12+
- uv (Python包管理工具)

### 安装依赖

```bash
cd server
uv install
```

### 启动服务

```bash
cd server
uv run uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 访问应用

打开浏览器访问：http://localhost:8000

## 🎮 使用说明

1. **选择场景**：在欢迎页选择"模拟课堂场景"或"模拟会议场景"
2. **开始记录**：点击"开始记录"按钮，观看实时转写演示
3. **生成摘要**：转写完成后，点击"生成摘要与图谱"
4. **查看图谱**：在右侧知识图谱面板查看可视化的知识关联
5. **编辑笔记**：切换到"笔记编辑"标签页，编辑和导出笔记

## 🌟 功能演示

### 场景选择
- 课堂模式：机器学习导论课程演示
- 会议模式：Q3产品路线图评审会

### 实时转写
- 自动播放预设转写内容
- 高亮显示当前正在转写的内容
- 支持标记重要内容

### AI摘要
- 生成5条核心要点
- 提取8-10个关键词标签
- 点击关键词在图谱中高亮对应节点

### 知识图谱
- 力导向图布局
- 节点分类：核心概念（橙色）、相关概念（蓝色）、细节节点（绿色）
- 支持拖拽、缩放、悬停查看详情

## 🌐 API接口

| 接口 | 方法 | 功能 |
|------|------|------|
| `/api/health` | GET | 健康检查 |
| `/api/scenes` | GET | 获取场景列表 |
| `/api/data/{scene_type}` | GET | 获取场景数据 |
| `/api/history` | GET | 获取历史记录 |
| `/api/summary` | POST | 生成摘要 |
| `/api/graph` | POST | 生成知识图谱数据 |
| `/api/keywords` | POST | 提取关键词 |
