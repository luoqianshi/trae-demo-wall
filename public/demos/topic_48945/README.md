# 声瞳Live

声瞳Live 是一个面向真实户外步行场景的实时助行原型。系统会固定采样胸前前视画面，在识别到高风险、路线节点或不确定情况时，优先通过耳机播报简短且可执行的中文提醒。

## 高效环境直述模式

当前实时模式优先追求“更快、更直接、更清楚地说出前方环境”：

- 主要描述脚下到前方约 3 米内的环境
- 优先表达地面与障碍、通道通畅度、方位关系、辅助环境
- 默认一句直述前方环境，只有必要时才补一句
- 建议只在明显风险或不确定时出现，而且保持极短
- 重复内容尽量由前端门控压制，减少打扰感

## 本地运行

```bash
npm install
npm run dev:full
```

- 前端页面默认运行在 `http://localhost:5173/`
- 本地代理默认运行在 `http://localhost:8787/`

## 真实大模型接入

页面内提供“真实识别设置”面板，你需要填写：

- Base URL
- API Key
- 模型名

第一版规则如下：

- 配置只保存在当前浏览器 localStorage
- 前端不会直接调用外部大模型服务
- 页面会通过同仓本地代理 `/api/vision/analyze` 发起真实分析请求
- 真实服务失败时，系统会自动回退到模拟模式并明确提示当前状态

## 测试连接

- 保存配置后，可点击“测试连接”
- 本地代理会请求 OpenAI 兼容接口的 `/models`
- 若模型名存在，则页面会标记“连接可用”
