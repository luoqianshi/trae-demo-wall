💡 创意简介

想解决什么问题：朋友之间想打麻将却凑不齐人？身处异地无法面对面打牌？现有的麻将游戏要么太复杂，要么缺少熟人社交氛围。我们需要一个简单直接、能快速约局的在线麻将平台。


为什么会想到做这个：周末和朋友聚会时经常有人临时有事缺席，导致麻将局开不起来。而且现在大家分布在不同城市，想一起玩牌更难。于是就想做一个可以在线创建私人房间、快速邀请好友的麻将游戏。


大概是什么产品：基于 Cocos Creator 的跨端在线棋牌游戏，支持手机和 PC，主打私人房间社交体验。

🎯 目标用户及痛点

面向哪些用户：

  • 喜欢打麻将的年轻人（20-40岁）
  • 异地工作的朋友群体
  • 想随时随地打牌的休闲玩家

当前痛点：

  • 线下凑人难，缺一个就开不了局
  • 传统麻将游戏匹配陌生人，缺少熟人社交感
  • 断线后重连困难，容易丢局
  • 战绩记录不完整，无法回顾精彩对局

✨ 价值与意义

社交娱乐价值：打破地域限制，让分散在各地的朋友、家人可以随时线上聚在一起打麻将。私人房间机制保持了熟人社交的氛围，让打牌不仅仅是娱乐，更是维系感情的方式。


用户体验提升：快速创建房间、一键邀请好友；智能断线重连，确保游戏不中断；完整战绩回放，随时回顾精彩对局；跨端支持，手机和 PC 都能玩。

🎮 核心功能

✓ 完整麻将玩法（吃碰杠胡）
✓ 私人房间 + 邀请好友
✓ 选飘系统（飘分玩法）
✓ 听牌提示
✓ 智能断线重连
✓ 战绩回放
✓ 跨端支持（Android/iOS/Windows/Web）

⚙️ 技术架构

三服协同架构：

Account Server（认证服务）→ Hall Server（大厅服务）→ Game Server（游戏服务）

技术栈：

Cocos Creator 2.4 Node.js 16 Socket.IO MySQL 阿里云 ECS PM2
