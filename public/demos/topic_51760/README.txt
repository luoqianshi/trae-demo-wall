旅行搭子 Easy Trip - AI 旅行计划协作小程序 Demo
====================================================

【如何使用】
1. 用浏览器（推荐 Chrome / Edge / Safari）直接打开 index.html 文件
2. 即可体验完整的交互 Demo

【Demo 包含功能】
- 欢迎页：品牌介绍 + 核心功能入口
- 行程规划：北京4天3晚多日行程 + 高德地图路线可视化
- 攻略导入：支持攻略链接、大交通、酒店住宿、门票预约分类导入
- 多人协作：评论、投票、行程确认功能
- 攻略广场：热门攻略展示 + 筛选功能
- 复刻行程：一键复刻 + 日期自定义 + 交通酒店导入
- 行后反馈：AI 标签 + 自由评论 + 发布到攻略广场

【技术说明】
- 单文件 HTML，无需安装依赖
- 地图使用高德地图 JS API
- 所有数据为 mock 数据，仅用于演示
