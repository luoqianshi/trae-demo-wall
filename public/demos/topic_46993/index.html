📋 使用说明

  1. 启动后端服务：运行项目中的 启动Demo.bat 脚本，或执行 python -m pulse_diagnosis.server
  2. 等待服务就绪：后端服务启动后会自动打开Demo页面
  3. 选择体质类型：在首页点击"快速开始"中的任意体质类型
  4. 观察脉象采集：实时查看六通道脉搏波形和采集进度
  5. 查看分析报告：采集完成后自动跳转到健康分析页面
  6. 探索更多功能：尝试不同的脉象预设，调节参数观察波形变化
