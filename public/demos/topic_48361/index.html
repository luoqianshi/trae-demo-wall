星轨乐影

1. 创意介绍
想解决什么问题
听歌时只有简单的播放器界面，缺少视觉沉浸感。歌词显示形式单一，无法带来互动乐趣。喜欢多位歌手的用户需要频繁切换歌单，缺少统一的歌手管理体验。
为什么会想到做这个
灵感来自星空的浪漫感和音乐的律动性。想让听歌不只是"听"，还能"看"和"玩"，把歌手、歌词、星空融为一体，创造独特的音乐体验。
大概是什么产品
基于 Web 的沉浸式音乐可视化播放器，结合星空背景、3D 环形歌词和歌手轨道展示。
2. 目标用户及痛点
面向哪些用户
音乐爱好者，特别是喜欢多位歌手、追求个性化体验的年轻人群体。
在什么场景下使用
日常听歌放松、深夜独处、聚会氛围营造、音乐分享等场景。
当前痛点
传统音乐播放器界面单调，歌词只是静态文本。用户无法直观管理喜欢的歌手，也缺少能与音乐产生互动的视觉体验。
3. 价值与意义

提升用户体验：打破传统播放器的单一形式，让听歌变成沉浸式视听体验。3D 环形歌词可拖动旋转，逐字跳动与演唱同步，带来"边听边玩"的乐趣。

技术创新价值：融合星空动画、3D 文字渲染、网易云音乐 API 接入等技术，展示了前端可视化与音乐服务的创新结合，为音乐应用提供新的交互范式。

4. 核心功能亮点

✨ 星空背景与炫彩流星

动态星空营造沉浸氛围，炫彩流星增添视觉美感

🌍 歌手轨道展示

五位歌手以星球形式环绕旋转，点击查看歌单播放歌曲

➕ 歌手管理

点击中心星球添加新歌手，点击歌手信息可删除

🎵 网易云音乐接入

支持网易云账号登录，获取版权歌曲播放权限

🎭 3D 环形歌词

歌词字符围绕成环形，渐变色彩，可拖动旋转

💃 逐字跳动同步

播放到的歌词字符带跳动效果，节奏与演唱同步

🔄 英文歌词自动滚动

长句英文歌词随进度向左旋转，完整显示不拥挤

🎨 可拖动播放器

播放器支持自由拖动位置，透明设计带专辑封面
