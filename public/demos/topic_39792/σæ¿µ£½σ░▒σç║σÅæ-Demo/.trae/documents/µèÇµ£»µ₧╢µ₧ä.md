# 周末就出发 - 技术架构文档

## 1. 技术选型

### 框架
- 纯原生 HTML + CSS + JavaScript
- 单文件实现，无外部依赖
- 可直接在浏览器打开

### 字体
```css
font-family: -apple-system, BlinkMacSystemFont, "SF Pro Display", "SF Pro Text", "Helvetica Neue", sans-serif;
```

### 图标
- 使用 Unicode 符号或 SVG 内联
- 不依赖外部图标库

## 2. 页面结构

### 页面列表
| 页面 | ID | 描述 |
|------|-----|------|
| 首页 | `#page-home` | 启动页/输入需求 |

| 弹窗 | ID | 描述 |
|------|-----|------|

| 预订成功 | `#toast-success` | 成功提示 |

## 3. 组件清单

### 手机外壳
- 固定宽度 375px，高度 812px（iPhone X 尺寸）
- 圆角 40px
- 顶部灵动岛/刘海
- 底部 home bar
- 阴影效果

### 状态栏
- 左侧：时间
- 右侧：电池图标 + 信号
- 高度 44px

### 底部 Tab 栏
- 高度 83px（含 safe area）
- 3 个标签：首页、行程、我的
- 图标 + 文字
- 当前页面高亮

### 卡片组件
- 白色背景
- 圆角 16px
- 阴影：0 2px 12px rgba(0,0,0,0.08)
- 内边距 16px

### 按钮
- 主按钮：橙红渐变背景，白色文字
- 次按钮：白色背景，橙色边框
- 圆角 12px
- 点击态：透明度变化

### 时间轴
- 垂直线连接
- 左侧时间，右侧内容
- 圆点节点

## 4. 色彩变量

```css
:root {
  --primary-start: #FF6B35;    /* 橙红 */
  --primary-end: #FFA500;      /* 橙黄 */
  --primary-gradient: linear-gradient(135deg, #FF6B35, #FFA500);

  --bg-main: #F8F9FA;
  --bg-card: #FFFFFF;
  --bg-dark: #1A1A2E;

  --text-primary: #1A1A2E;
  --text-secondary: #6B7280;
  --text-light: #9CA3AF;

  --border: #E5E7EB;
  --success: #10B981;
  --shadow: rgba(0, 0, 0, 0.08);
}
```

## 5. 动画规范

### 页面切换
- 淡入淡出 300ms ease
- 可使用 transform + opacity

### 按钮点击
- scale(0.98) 100ms
- 背景透明度变化

### 弹窗
- 从底部滑入 300ms
- 背景遮罩 fade 200ms

### 时间轴加载
- 依次显示 stagger 100ms
