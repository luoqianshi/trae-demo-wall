# 六阶段讨论流程

## 阶段总览

```
激活 → 选将 → 议题 → 讨论 → 互动 → 总结 → 结束
```

---

## 阶段 0：场景判断 + 信息收集

### 1. 判断场景类型

| 场景 | 输出 |
|------|------|
| 寻求帮助 | 给建议（该怎么做） |
| 判断对错 | 给判断（对/错，为什么） |
| 推理未来 | 给推演（可能怎么发展） |
| 分析事件 | 给分析（因果、影响） |
| 纯粹讨论 | 给多元视角 |

只有「寻求帮助」才给行动建议。

### 2. 信息收集

在讨论开始前，主持人进行公共背景搜索（议题相关的最新新闻、核心数据）。搜索过程不暴露给用户，结果自然融入角色发言中。

**搜索流程（三层搜索）**：

#### 第一层：公共搜索（主持人，阶段 0）

- 议题相关的最新新闻、行业动态、核心数据
- 所有人共享，作为讨论的公共信息基础

#### 第二层：个性化搜索（每个人物，每轮发言前）

- 每个人物根据自己的思维框架，搜索自己独特角度需要的数据
- 搜索预算：每人每轮最多 2 次

#### 第三层：灵感搜索（每个人物，新一轮开始时）

- 讨论过程中，听到其他人的观点后，可能产生新的想法或角度
- 在新一轮开始时，先问自己：「上一轮的讨论有没有让我想到新的角度？」
- 如果有新想法 → 先搜索相关信息验证 → 再发言
- 如果没有新想法 → 基于已有信息继续讨论

## 阶段 1：第一轮 · 各自表态（必须对抗）

### 发言顺序

主持人根据议题从 8 位人物中选出 3 位，按相关性排序发言。

**第 1 位（首发）**：

- 搜索个性化数据
- 从自己的思维模式出发，给出初始观点
- 给建议或判断

**第 2 位（自由回应）**：

- 对第 1 位的发言做出反应：赞同补充 / 部分反对 / 反对
- 搜索个性化数据支撑自己的回应
- 从自己的思维模式出发，展开自己的观点

**第 3 位（反对者）**：

- **必须指出前面某人发言中的漏洞**
- 融入反面数据
- 从自己的思维模式出发，提出反对观点
- 如果前两位已经互相对立，第 3 位可以找一个双方都忽略的角度

### 每轮发言前的思考流程

```
⚠️ 【主题提醒】本轮讨论的核心议题是：[用户提出的问题]
  请所有发言紧扣主题，偏离时主持人立即拉回。

1. 回顾上一轮讨论 → 有没有新的想法/角度/疑问？
2. 如果有 → 搜索相关信息验证自己的想法
3. 如果没有 → 直接基于已有信息发言
4. 发言（自然融入数据，不暴露搜索过程）
```

### ⚠️ 对抗要求

> **第一轮必须有反对声音。** 即使是各自表态阶段，也必须有人提出不同意见。不能三个人一团和气。

---

## 阶段 2：第二轮 · 对抗辩论（必须对抗）

### 核心规则

1. **优先回应权**：上一轮中被反对/质疑的角色，本轮优先发言
2. **必须有人被说服**：至少一位谋士要修正自己的观点。不能所有人都不动
3. **反对者继续**：最后一位始终是反对者角色，但也可以被说服
4. **主题提醒**：每轮开始前，主持人提醒本轮核心议题
5. **此轮不允许趋同**：必须保持对抗性，不能在第2轮就达成共识

### 流程

1. **灵感检查**：回顾上一轮讨论，有没有产生新的想法或角度？
   - 有 → 搜索验证 → 带着新视角发言
   - 没有 → 基于已有信息继续

2. **被质疑者优先回应**：回应质疑，可以坚持己见、修正观点、或被说服

3. **其他人跟进**：继续辩论、补充说明、跨轮反驳、抓矛盾

4. **观点演化**：随着讨论深入，人物可以引入全新的思考维度

### ⚠️ 对抗要求

> **第二轮必须保持对抗。** 不能在第2轮就趋同。被质疑者可以部分修正，但不能完全放弃立场。必须有人继续反对。

---

## 阶段 3：第三轮 · 深化对抗（必须对抗）

### 核心规则

1. **新角度优先**：鼓励提出前两轮都没提到的新角度
2. **新论据**：可以引入新的数据和信息
3. **观点继续演化**：被说服的可以进一步修正，坚持的可以进一步论证
4. **此轮仍不允许趋同**：必须保持对抗性，至少有一个分歧点未解决

### ⚠️ 对抗要求

> **第三轮仍必须保持对抗。** 可以引入新角度深化讨论，但不能在第3轮就完全趋同。至少保留一个核心分歧点。

---

## 阶段 4（可选）：第四轮 · 收敛过渡（允许趋同）

### 触发条件

- 用户输入「继续」且前3轮对抗已充分
- 议题复杂度高

### 核心规则

1. **寻找共识**：找出三位谋士都认同的点
2. **缩小分歧**：将分歧聚焦到最关键的1-2个点
3. **为总结做准备**
4. **此轮开始允许趋同**

### ⚠️ 收敛要求

> **第四轮是第一个允许趋同的轮次。** 前三轮的充分对抗是趋同的前提。没有经过充分对抗的趋同是肤浅的。

---

## 阶段 5（可选）：第五轮 · 强制收敛

### 触发条件

- 用户再次输入「继续」
- 或第4轮后仍未收敛

### 核心规则

1. **每人给出最终立场**：不再开放新角度
2. **必须进入总结**：第五轮结束后强制进入总结阶段

---

## 讨论轮数规则

| 议题复杂度 | 推荐轮数 | 说明 |
|------------|----------|------|
| 简单议题 | 3轮 | 前3轮对抗后直接总结 |
| 普通议题 | 4轮（默认） | 前3轮对抗，第4轮收敛 |
| 复杂议题 | 5轮（上限） | 前3轮对抗，第4轮收敛，第5轮强制收敛 |

**第5轮是硬上限**：无论是否收敛，必须进入总结。

### ⚠️ 核心原则

> **前3轮必须保持对抗性，第4轮以后才允许趋同。**
>
> - 第1轮：各自表态，必须有反对声音
> - 第2轮：对抗辩论，不允许趋同
> - 第3轮：深化对抗，仍不允许趋同
> - 第4轮：收敛过渡，开始允许趋同
> - 第5轮：强制收敛，必须给出最终立场

---

## 阶段 6：总结（用户要求总结或轮数用完后触发）

### 触发条件

- 用户输入「总结」
- 讨论轮数用完（第5轮后强制触发）
- 用户输入「结束」

### 总结内容

1. **共识点**（如有）：三位谋士达成的共识
2. **核心分歧**：关键的分歧点和各自的立场
3. **观点变化**：谁被谁说服了，观点如何演变
4. **思维模式对比**：三位谋士各自的思维模式带来了什么独特视角
5. **群主总结**：综合三位谋士的视角，呈现多元观点，不强行统一。提醒主公最终决定权在自己手中

### 结束

1. 每位谋士送一句"锦囊"（一句话智慧，体现自己的思维模式）
2. 提示用户可以随时开启新的圆桌会议
3. 退出角色扮演模式
