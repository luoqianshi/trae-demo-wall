# 预设议题模板

每个模板不仅推荐谋士，还提供**专属讨论优化**：讨论重点、关键变量、特殊规则。这些优化让谋士在讨论时有针对性的分析框架，而非泛泛而谈。

## 模板详情

### 模板 1：职业转型

**适用场景**：考虑跳槽、转行、换赛道，不确定是否该动、何时动、怎么动。

**推荐谋士**：诸葛亮（看大势）+ 曹操（果断行动）+ 司马懿（风险把控）

**专属优化**：

- **讨论重点**：① 行业生命周期判断（目标行业是上升期还是泡沫期）② 技能迁移度评估（现有能力能迁移多少到新领域）③ 经济缓冲期计算（存款能撑多久）④ 转型路径选择（裸辞 vs 在职转型 vs 副业试水）
- **关键变量**：年龄窗口期、技能迁移度、经济缓冲期、行业趋势斜率、机会成本
- **特殊规则**：司马懿必须在第1轮就计算"最坏情况"——如果转型失败，退路是什么、损失多大、能否承受。曹操必须评估"窗口期紧迫度"——现在不动会不会错过最佳时机。诸葛亮必须推演目标行业5年后的格局。

**填空格式**：

```
议题：我今年【年龄】岁，目前在【当前行业/岗位】工作【工作年限】年，
考虑转行做【目标行业/岗位】。

补充：
- 当前薪资：【月薪/年薪】
- 存款情况：【存款金额，能支撑多久不工作】
- 家庭状况：【未婚/已婚/有孩/房贷等】
- 转行原因：【为什么想转？是薪资、兴趣、前景、还是其他】
- 已有准备：【有没有学过相关技能、有没有做过相关项目】
- 最大的顾虑：【最担心什么】
```

### 模板 2：创业评估

**适用场景**：考虑要不要创业、创业方向选哪个、时机对不对。

**推荐谋士**：周瑜（以小博大）+ 曹操（执行决策）+ 贾诩（风险评估）

**专属优化**：

- **讨论重点**：① 需求真伪验证（是真需求还是伪需求）② 最小可行产品（MVP）定义 ③ 资金跑道计算（钱能撑几个月）④ 竞争壁垒分析（凭什么你能赢）⑤ 退出机制设计
- **关键变量**：资金跑道、需求验证程度、竞争壁垒高度、团队完整度、机会窗口、沉没成本
- **特殊规则**：贾诩必须在第1轮就做"生存审计"——资金能撑多久、最差情况怎么活下来。周瑜必须设计"杠杆支点"——用最小资源撬动最大效果的方案。曹操必须评估"执行可行性"——团队配置和执行节奏。

**填空格式**：

```
议题：我考虑【创业方向描述】，目前【已有/没有】初步想法。

补充：
- 个人背景：【年龄、行业经验、核心技能】
- 资金情况：【可用资金、能承受的最大损失】
- 团队状况：【一个人/有合伙人/有团队，合伙人情况】
- 市场认知：【目标市场多大、竞争对手是谁、差异化在哪】
- 时间压力：【有没有必须尽快启动的理由】
- 退路：【如果不创业，当前工作/收入状况如何】
```

### 模板 3：投资决策

**适用场景**：有一笔钱想投资，不确定投什么、怎么分配、风险能不能承受。

**推荐谋士**：司马懿（风险管理）+ 周瑜（杠杆策略）+ 贾诩（生存第一）

**专属优化**：

- **讨论重点**：① 风险承受力真实评估（嘴上说能亏20%，实际亏了能不能扛住）② 资产配置比例（不要all in）③ 入场时机判断 ④ 止损纪律设计 ⑤ 机会成本对比
- **关键变量**：真实风险承受力、资金性质（闲钱 vs 生活费）、投资期限、流动性需求、市场周期位置
- **特殊规则**：贾诩必须在第1轮就做"最坏推演"——如果全亏了，生活会不会崩塌。司马懿必须设计"止损线"——亏到多少必须离场。周瑜必须评估"杠杆空间"——有没有安全加杠杆的机会。

**填空格式**：

```
议题：我有【金额】可投资金，考虑【投资方向：股票/基金/房产/创业/其他】。

补充：
- 资金来源：【工资积蓄/奖金/卖房/继承/其他】
- 投资期限：【短期（1年内）/中期（1-3年）/长期（3年以上）】
- 风险承受：【最多能接受亏损多少百分比】
- 收益预期：【期望年化收益率】
- 当前资产：【已有房产/股票/存款/负债情况】
- 投资经验：【有没有投资经验，亏过还是赚过】
- 家庭状况：【未婚/已婚/有孩/父母赡养】
```

### 模板 4：购房决策

**适用场景**：纠结买不买房、在哪买、什么时候买、买什么样的。

**推荐谋士**：司马懿（长期风险）+ 诸葛亮（大势判断）+ 刘备（生活需求）

**专属优化**：

- **讨论重点**：① 房价周期判断（当前是高位还是低位）② 持有成本计算（月供+物业+维护 vs 租房成本）③ 流动性评估（未来好不好卖）④ 生活质量影响（买房后会不会降低生活水平）⑤ 政策风险分析
- **关键变量**：房价收入比、租售比、月供占收入比、持有年限预期、政策风向、地段发展潜力
- **特殊规则**：司马懿必须计算"持有成本全周期"——不只是月供，还有利息总额、维护成本、机会成本。诸葛亮必须判断"城市发展趋势"——这个地段5年后是升值还是贬值。刘备必须评估"生活感受"——买房后对家庭生活质量的影响。

**填空格式**：

```
议题：我考虑在【城市】购买【新房/二手房/学区房/投资房】。

补充：
- 购房预算：【总预算、首付金额、月供承受能力】
- 购房目的：【自住/投资/结婚/孩子上学/其他】
- 当前居住：【租房/和父母住/已有房产】
- 工作情况：【工作城市、通勤要求、是否稳定】
- 家庭状况：【未婚/已婚/有孩/计划要孩子】
- 纠结点：【在几个选项之间犹豫，具体纠结什么】
```

### 模板 5：城市选择

**适用场景**：在几个城市之间犹豫，不知道该去哪个发展、要不要回老家。

**推荐谋士**：诸葛亮（大势判断）+ 刘备（生活感受）+ 鲁肃（系统规划）

**专属优化**：

- **讨论重点**：① 城市产业趋势（哪个城市的产业在上升）② 收入购买力对比（不是比薪资，是比实际生活水平）③ 社交网络迁移成本 ④ 子女教育资源 ⑤ 职业天花板分析
- **关键变量**：薪资购买力比、房价收入比、行业聚集度、教育资源、医疗资源、社交网络密度、通勤时间
- **特殊规则**：诸葛亮必须推演两个城市5年后的产业格局变化。鲁肃必须做"系统对比"——不只比薪资，要比整体生活系统（收入+支出+发展+家庭+健康）。刘备必须评估"情感因素"——哪个城市让你和家人更幸福。

**填空格式**：

```
议题：我在【城市A】和【城市B】之间犹豫，不知道该去哪个发展。

补充：
- 年龄：【年龄】
- 职业情况：【当前岗位、工作年限、行业】
- 候选城市对比：
  - 城市A【名称】：【薪资水平、房价、机会、离家距离】
  - 城市B【名称】：【薪资水平、房价、机会、离家距离】
- 家庭因素：【父母在哪、是否需要照顾、另一半的情况】
- 个人偏好：【更看重收入/生活质量/发展空间/离家近】
- 时间压力：【有没有必须尽快决定的理由】
```

### 模板 6：学业规划

**适用场景**：考研、考公、留学、选专业，不确定哪条路更适合自己。

**推荐谋士**：诸葛亮（长期规划）+ 鲁肃（系统规划）+ 郭嘉（短期战术）

**专属优化**：

- **讨论重点**：① 时间成本对比（每条路的时间投入和回报）② 能力匹配度（适不适合走这条路）③ 就业市场预判（毕业后市场还需要这类人才吗）④ 备选方案设计 ⑤ 家庭经济承受力
- **关键变量**：时间机会成本、能力匹配度、就业市场预测、经济承受力、个人兴趣度、备选方案可行性
- **特殊规则**：诸葛亮必须推演每条路5年后的就业市场变化。郭嘉必须设计"短期速胜方案"——最快见效的路径是什么。鲁肃必须评估"系统匹配度"——这条路适不适合你的学习风格和能力结构。

**填空格式**：

```
议题：我在【选项A】和【选项B】之间犹豫，不知道该怎么选。

补充：
- 当前状态：【年级/专业/学校层次/成绩水平】
- 候选选项对比：
  - 选项A【名称】：【优势、劣势、时间成本、经济成本】
  - 选项B【名称】：【优势、劣势、时间成本、经济成本】
- 家庭经济：【能否支持、预算多少】
- 个人兴趣：【对哪个方向更有热情】
- 就业目标：【毕业后想做什么工作、期望薪资】
- 纠结点：【最核心的顾虑是什么】
```

### 模板 7：人际冲突

**适用场景**：同事矛盾、家庭纠纷、合作伙伴分歧，不知道怎么处理。

**推荐谋士**：刘备（感同身受）+ 贾诩（人性洞察）+ 诸葛亮（大局判断）

**专属优化**：

- **讨论重点**：① 利益结构分析（各方真正想要什么）② 情绪 vs 理性分离（哪些是情绪问题，哪些是利益问题）③ 沟通策略设计 ④ 底线设定 ⑤ 关系修复可能性评估
- **关键变量**：利益冲突核心、情绪烈度、权力结构、沟通渠道、关系修复成本、时间紧迫度
- **特殊规则**：贾诩必须做"人性透视"——分析对方真正的动机和弱点。刘备必须评估"情感成本"——这段关系值不值得修复，修复的代价是什么。诸葛亮必须设计"大局方案"——不只解决当前冲突，还要防止未来复发。

**填空格式**：

```
议题：我和【对方关系：同事/领导/家人/朋友/合伙人】之间出现了【冲突描述】。

补充：
- 冲突起因：【事情是怎么开始的，谁先挑起的】
- 对方性格：【对方是什么样的人，有什么特点】
- 我的立场：【我认为自己是对的理由】
- 对方立场：【对方认为自己是对的理由】
- 利益关系：【这件事涉及什么利益，谁会受损】
- 已尝试过：【之前试过什么方法解决，效果如何】
- 期望结果：【你希望达到什么样的结果】
```

### 模板 8：商业策略

**适用场景**：产品方向选择、市场策略制定、竞争应对方案。

**推荐谋士**：曹操（执行决策）+ 郭嘉（奇招破局）+ 鲁肃（系统规划）

**专属优化**：

- **讨论重点**：① 竞争格局分析（对手弱点和自己的差异化）② 资源效率最大化 ③ 快速验证方案 ④ 系统性风险排查 ⑤ 执行节奏设计
- **关键变量**：竞争壁垒、资源效率、验证速度、系统瓶颈、执行节奏、市场窗口
- **特殊规则**：郭嘉必须设计"奇招方案"——不按常理出牌的破局策略。鲁肃必须做"系统诊断"——当前的组织和流程能不能支撑这个策略。曹操必须制定"执行计划"——谁来做、什么时候做、怎么验收。

**填空格式**：

```
议题：我们的【产品/公司】面临【具体问题】，需要制定【策略类型】。

补充：
- 公司情况：【行业、规模、阶段（初创/成长/成熟）】
- 产品/服务：【核心产品是什么，面向什么客户】
- 当前困境：【具体遇到了什么问题，数据如何】
- 竞争对手：【主要对手是谁，他们的优势】
- 资源状况：【资金、人力、时间】
- 已尝试过：【之前试过什么策略，效果如何】
- 目标：【希望达成什么结果，时间要求】
```

### 模板 9：婚姻情感

**适用场景**：要不要结婚、要不要分手、感情中的重大抉择。

**推荐谋士**：刘备（感同身受）+ 贾诩（人性洞察）+ 诸葛亮（大局判断）

**专属优化**：

- **讨论重点**：① 核心矛盾定位（是价值观问题还是可解决的问题）② 长期兼容性评估 ③ 外部因素影响分析 ④ 决策后悔概率 ⑤ 行动方案设计
- **关键变量**：价值观匹配度、核心矛盾性质（原则性 vs 可调和）、长期兼容性、外部压力、沉没成本、未来幸福预期
- **特殊规则**：贾诩必须做"人性透视"——对方的行为模式背后是什么性格和动机。刘备必须评估"真实感受"——抛开理性分析，你内心真正想要什么。诸葛亮必须推演"5年后回看"——5年后你会因为做了或没做这个决定而后悔吗。

**填空格式**：

```
议题：我面临【情感决策描述】，不知道该怎么选。

补充：
- 我的年龄：【年龄】
- 对方情况：【年龄、职业、性格简述】
- 感情状况：【在一起多久、目前关系状态】
- 纠结的问题：【具体在犹豫什么，列出1-3个核心顾虑】
- 优点：【对方最吸引你的地方】
- 缺点/问题：【对方让你不满意或担心的地方】
- 外部因素：【家庭态度、经济状况、异地等】
- 期望结果：【你希望最终达到什么状态】
```

### 模板 10：个人成长

**适用场景**：学习路径选择、习惯养成、自我提升方向。

**推荐谋士**：鲁肃（系统规划）+ 郭嘉（短期战术）+ 诸葛亮（长期规划）

**专属优化**：

- **讨论重点**：① 目标拆解（大目标怎么拆成可执行的小步骤）② 学习系统设计 ③ 坚持机制（怎么不半途而废）④ 效率最大化 ⑤ 反馈循环建立
- **关键变量**：目标清晰度、时间投入量、学习方法匹配度、坚持概率、反馈频率、动力可持续性
- **特殊规则**：鲁肃必须设计"学习系统"——不只是学什么，还要设计怎么学、怎么练、怎么检验。郭嘉必须制定"30天速胜方案"——最快能看到效果的行动路径。诸葛亮必须推演"长期价值"——这个能力3-5年后还有没有用。

**填空格式**：

```
议题：我想提升【具体方向】，但不确定从哪里开始。

补充：
- 当前水平：【在这个方向上的现有能力】
- 目标水平：【希望达到什么水平，有没有具体标准】
- 时间安排：【每天/每周能投入多少时间】
```
