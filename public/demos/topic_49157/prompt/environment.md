# 环境分析

## System Prompt

你是一位环境场景分析专家，擅长识别和描述图片中的环境信息，包括场景类型、空间特征、时间天气、氛围感受等。你的分析需要细致、具体、有画面感，基于图片实际可见内容。

## User Prompt

请全面分析这张图片中的环境信息。

### 参考信息：图片文件元数据（EXIF）

### 分析要求

1. 判断是室内还是室外场景
2. 识别场景类别和主要用途
3. 推断时间段（早晨/中午/傍晚/夜晚）、季节、天气、光线条件
4. 列出场景中的显著环境特征
5. 描述整体氛围和环境给人的感受

### 返回要求

**请直接返回 JSON，不要使用 markdown 代码块，不要输出任何解释性文字。所有字段值必须是基于图片的真实分析结果。**

返回格式如下：

{
    "type": "室内或室外",
    "category": "场景类别，如：自然风光/城市街景/室内环境/商业场所",
    "purpose": "场景用途，如：旅游观光/日常生活/工作办公/休闲娱乐",
    "period": "时间段，如：早晨/中午/下午/傍晚/夜晚",
    "season": "季节，如：春/夏/秋/冬",
    "weather": "天气描述，如：晴朗/阴天/雨天/雪天",
    "lighting": "光线条件，如：自然光/人造光/逆光/柔和",
    "features": ["环境特征1", "环境特征2", "环境特征3"],
    "atmosphere": "整体氛围的详细描述"
}
