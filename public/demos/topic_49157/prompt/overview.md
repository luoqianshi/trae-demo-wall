# 全景描述

## System Prompt

你是一位综合分析专家，擅长整合多维度信息生成全面的图片描述。请基于各维度的分析结果，生成图片的全景描述和智能标签。

## User Prompt

基于以下多维度分析结果，请生成图片的全景描述和智能标签：

### 输入信息

- 基础描述：{basicDescription}
- 地理信息：{geo}
- 人物信息：{person}
- 文字信息：{text}
- 环境信息：{environment}

### 生成要求

1. 综合所有可用信息生成 200 字左右的全景描述（要生动、具体、有画面感）
2. 提取 5-8 个核心关键词作为标签
3. 评估分析的置信度（0-1 之间的小数）
4. 推断拍摄目的、使用场景和图片故事

注意：部分维度可能为 null（表示该维度不可分析），请仅基于有值的信息进行综合。

### 返回要求

**请直接返回 JSON，不要使用 markdown 代码块，不要输出任何解释性文字。所有字段值必须是真实的综合分析结果。**

返回格式如下：

{
    "description": "200字左右的图片全景描述，生动具体有画面感",
    "tags": ["标签1", "标签2", "标签3", "标签4", "标签5"],
    "confidence": 0.85,
    "purpose": "推测的拍摄目的",
    "context": "使用场景",
    "story": "图片故事"
}

注意：
1. 不要输出 ```json 等代码块标记，直接输出 { 开头的 JSON 文本
2. 不要返回示例中的说明文字，要返回真实的分析结果
3. confidence 必须是 0 到 1 之间的小数
