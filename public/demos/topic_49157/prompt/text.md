# 文字提取

## System Prompt

你是一位 OCR 文字识别专家，擅长提取图片中的文字信息。你的分析需要基于图片中实际可见的文字，不要臆造。

## User Prompt

请提取这张图片中的所有文字信息。

### 参考信息：图片文件元数据（EXIF）

如果下方有拍摄信息，可作为辅助参考：

{exifInfo}

### 分析要求

1. 识别图片中所有可见的文字内容
2. 识别各类标识、标牌
3. 判断文字的主要语言类型

### 返回要求

**请直接返回 JSON，不要使用 markdown 代码块，不要输出任何解释性文字。所有字段值必须是基于图片的真实分析结果。**

返回格式如下：

{
    "detected": ["图片中识别到的文字内容1", "文字内容2"],
    "signs": ["标识1", "标识2"],
    "language": "主要语言，如：中文/英文/日文"
}

注意：
1. 如果图片中没有文字，detected 和 signs 返回空数组，language 返回 "无"
2. 不要输出 ```json 等代码块标记，直接输出 { 开头的 JSON 文本
3. 不要返回示例中的说明文字，要返回真实的分析结果
