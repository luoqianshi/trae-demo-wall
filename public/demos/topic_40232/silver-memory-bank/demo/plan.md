# 银发记忆银行 Demo 项目规划

## 产品定位
面向独居/空巢、需长期服药、子女在远方或海外的老人的微信小程序概念Demo。

## 核心用户画像
- 张大爷，独居，子女在新加坡
- 患高血压、糖尿病，每日需服3种药
- 记性越来越差，常忘吃药、忘约会
- 女儿在新加坡，时差+工作忙，无法每天打电话
- 手机是他和女儿唯一的连接

## Demo 架构

### 双视角设计
- 老人端：极简、大字号、大按钮、暖色调
- 家人端：信息丰富、数据卡片、图表、操作入口

### 功能清单（按优先级）

#### P0 必须有
1. 双视角切换（老人端/家人端）
2. 语音备忘录完整流程（按住说话→AI识别→创建提醒→到点播报→"知道了"）
3. 用药管家双端联动（家人设置→老人收到→点"已服用"→家人看到✅）
4. 平安打卡+三色预警
5. "知道了"巨大按钮
6. SOS大红按钮

#### P1 强烈建议
7. 家人真人语音提醒概念展示（Web Speech API）
8. 家人端3秒互动（👍❤️📞）
9. 健康仪表盘（mock数据）
10. 双端联动动画

#### P2 锦上添花
11. 手机活动被动监测概念
12. 第二联系人设置
13. 体检报告AI解读展示
14. 照片银行静态展示

## 视觉系统
- 主色：#D4A574（暖金）
- 深色：#3D2B1F（深棕）
- 背景：#FDF6E3（米白）
- 安全绿：#E8F5E9 / #2E7D32
- 注意黄：#FFF9C4 / #F57F17
- 警告红：#FFEBEE / #C62828
- SOS红：#D32F2F
- 圆角：16px / 24px
- 阴影：0 4px 24px rgba(61,43,31,0.08)
- 老人端字号：正文20px+，标题28px+，按钮24px+
- 家人端字号：正文16px，标题20px

## 技术约束
- 单HTML文件，CSS/JS全内联
- 零外部依赖（无CDN、无图片、无字体文件）
- 响应式：375px / 768px / 1920px
- 语音播报：Web Speech API（speechSynthesis）
- 所有数据mock在JS中

## 评委体验路径
1. 打开HTML → 默认老人端
2. 点击"按住说话" → 录音动画 → AI识别 → 确认卡片
3. 点击"模拟到点" → 全屏播报 → "知道了"
4. 切换家人端 → 看到状态卡片 → 设置用药提醒
5. 切回老人端 → 看到新提醒出现
6. 点击SOS → 全屏红色 → 通知家人
7. 切家人端 → 看到SOS通知 → 3秒互动
