⚡ L0 — 原子操作

⏱ 秒~5分钟 🎯 单次查询/操作 🛡 无需认知循环防护
  • 查分子属性、查一个数据点
  • 翻译一句话、检查语法
  • 简单计算、单位换算
  • 一次交互完成，无循环风险

🚀 L1 — 快速任务

⏱ 几分钟~2小时 🎯 单次明确产出 🛡 max_steps=10
  • 搜一批文献、下载论文 PDF
  • 生成研究想法、头脑风暴
  • 画数据图、画框架图/流程图
  • 润色段落、修改语法
  • 快速制作 PPT（论文答辩/组会汇报）
  • 模式光谱：Fidelity（模板驱动，可预测输出）

🎯 L2 — 专题研究

⏱ 几小时~2天 🎯 自包含专题 🛡 max_steps=20
  • 🧬 自我博弈课题发现 — 3层博弈筛选最佳课题
  • 🔬 核心假设验证 — 最小实验快速验证方向
  • 🎭 多角色对抗评审 — R1-R5 五角色并行评审
  • 🎯 瓶颈驱动迭代 — 评分最低维度定点突破
  • 📚 系统性文献综述 — 6阶段搜索→筛选→综合→报告

🔬 L3 — 深度研究

⏱ 几天~几周 🎯 完整科研项目 🛡 18阶段 + 认知循环检测
  • scout 文献调研 → baseline 基线建立
  • idea 想法生成（含自我博弈 + 假设验证）
  • experiment 实验执行 → analysis-campaign 数据分析
  • objective-evolution 目标进化 → decision 决策
  • paper-outline → paper-plot → write 论文写作
  • review 评审（含对抗评审 + 瓶颈驱动迭代）
  • figure-polish → integrity-check → reproducibility-check → finalize

📄 L4 — 发表级准备

⏱ 几天~几周 🎯 投稿级质量控制 🛡 G1-G9 九道质量门
  • 🎨 图表质量升级 — AutoFigure 发表级 SVG 插图
  • 📝 全文润色 — Nature 级写作质量检查
  • 📊 数据声明与引用核验 — 可重复性保证
  • 💬 审稿回复 — 结构化 rebuttal 生成
  • 📋 组会PPT — 答辩/汇报 PPT 自动生成
  • ✅ 预审检查 — 综合质量审查，确保投稿无忧
