# 《AI进家-让爱到家》· 初赛 Demo

> 一个让独居老人和留守儿童"一个人在家，也不害怕"的 APP。AI 不替代家人，但在家人来之前的那段空白里，做一个守在家里的硅基伙伴。

## 一、产品定位

**不是"教老人用 AI"，而是"为独居者准备的家"**。
- 摔倒有人救，菜有人买，电费有人交，空调不会调有人管
- 孩子在家的危险有人第一时间知道
- 子女在千里之外也能第一时间了解
- **AI 永远不会代替他们做决定**

## 二、三端架构

| 端 | 形态 | 核心职责 |
|---|---|---|
| **用户端** | 老人手机 / 儿童手表 | SOS 紧急、语音叫 AI、智能家电、今日提醒 |
| **家人端** | 子女微信小程序 | 接收推送、看一日简报、远程视频指导 |
| **社区端** | 网格员 / 志愿者 | 接 SOS 上门、接单买菜/买药/缴费、陪同就医 |

## 三、五大功能模块

| # | 模块 | 关键演示 |
|---|---|---|
| 1 | **SOS 紧急 ⭐ 核心** | 长按 3 秒 → 30s 反馈窗 → AI 音频 + 短信双通道拨 120 → 三方联动 |

## 四、SOS 30 秒反馈机制（核心差异化）

```
[触发] 长按 3 秒 / 设备剧烈撞击 / 语音"救命" / 儿童手表被强拆
   ↓
[30s 反馈窗] AI 语音问"您摔了吗？我能帮您做什么？"
   ↓
   ├── 老人清醒回应"我没事" → 取消，不打扰
   ├── 识别"救命/痛/难受" → 立即升级（跳过 30s）
   └── 30s 无回应 → 默认严重，自动升级
         ↓
[升级 1] AI 模拟音频拨 120（自动播报地址+情况+联系人）
[升级 2] 结构化短信推送 120 调度平台
[升级 3] 同步呼叫家属 + 社区网格员
[陪护]   AI 不挂断，持续陪老人/孩子说话
[回看]   家属收到完整时间线
```

**儿童场景额外规则**：
- 6-12 岁儿童无判断力 → 倒计时直接缩到 10s
- 儿童 SOS 必联系监护人（爸爸、奶奶、外公）
- 联动社区志愿者，3 分钟内上门（比 120 快）

**伦理立场（图灵一派）**：AI 永远等老人说"是"再做事。**但当老人无法判断时（昏迷），AI 必须替他喊救命**——这不是"替代判断"，是"人已经判断不了时让机器兜底"。

## 五、智能家居生态 - 9 大场景联动

> **核心升级**：智能家居不是"教老人按遥控器"，而是"让家里所有设备听懂一句话"。空调只是 6 大设备之一，老人说一句"看电视剧"，6 个设备同时响应。

**6 大设备**：❄ 空调 / 🧺 洗衣机 / 🤖 扫地机 / 📺 电视盒子 / 💡 智能灯 / 🪟 窗帘

**9 大场景联动**：

| 老人说 | AI 联动设备 | 设计意图 |
|---|---|---|
| 📺 "看电视剧" | 电视盒子 + 灯 + 窗帘 + 空调 | 4 设备联动：氛围感 |
| 🛏 "睡觉了" | 灯 + 空调 + 窗帘 + 扫地机 | 5 设备：全屋睡眠模式 |
| 🌙 "起夜" | 灯（人体感应）| 2 分钟无动作自动关 |
| 🏠 "回家了" | 灯 + 空调 + 窗帘 + 扫地机 | 4 设备：开门即享 |
| 🪟 "有点闷" | 窗帘 + 空调 | CO₂ 检测自动通风 |
| 🤖 "扫地" | 扫地机 | 电量低自动回充 |
| 🧋 "想喝奶茶" | 跨页跳转 → 社区帮忙 | 老人也有馋嘴的权利 |
| 💰 "心疼电费" | 空调 + 灯 + 窗帘 + 扫地机 | 4 设备 + 1 级能效科普 |
| 🧒 "小孩在家" | 空调 + 灯 + 窗帘 + 洗衣机 | 4 设备：儿童安全模式 |

**健康档案联动**：
- 风湿/老寒腿 → 空调 27°C + 无风 + 关抽湿
- 哮喘/慢阻肺 → 25°C + 微风 + 禁冷风
- 高血压 → 室内外温差 ≤ 8°C
- 节能模式 → 28°C + 智能定时 + 睡眠曲线

**三种控制方式**：
1. **红外直控**（手机自带红外）— 旧空调也能用
2. **智能家居联动**（米家/美的美居/海尔智家/小度/天猫精灵）
3. **语音优先**（"看电视剧"就够了）

**跨页联动**：智能家居点"想喝奶茶" → localStorage 写入订单 → 跳转社区帮忙页自动打开外卖下单面板

## 八、3 分钟演示动线

1. **首页（30 秒）**：看到三端架构 + 老人问候 + 今日提醒 + SOS 大按钮
2. **SOS（60 秒核心）**：
   - 切换场景：👴 老人 / 🧒 儿童
   - 演示 ① 摔倒昏迷：30s 反馈窗 → 无回应 → 三方呼叫 + AI 音频 + 短信

   - 演示 ③ 误触：30s 内取消，不打扰
3. **智能家居生态（60 秒）**：
   - 9 大场景卡：看剧/睡觉/起夜/回家/通风/扫地/奶茶/节能/儿童
   - 切换"风湿"档案 → 演示"看电视剧" → 4 设备联动日志滚动
   - 演示"想喝奶茶" → 跨页跳转到社区帮忙自动打开外卖下单
4. **伦理边界（30 秒）**：4 个 Q + 图灵派底线
