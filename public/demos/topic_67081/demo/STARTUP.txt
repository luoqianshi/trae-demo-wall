=======================================
  易愈（YìYù）— 本地启动说明
=======================================

【一、这是什么】
一个以《易经》为骨架的情绪疏导公益游戏 demo。
包含 LLM 模式（接入大模型对话）和规则模式（离线兜底）。

【二、运行环境】
- Node.js v18 或以上
  下载地址：https://nodejs.org/
  安装后在命令行执行 `node -v` 出现版本号即装好

【三、首次启动步骤】

1) 解压本压缩包到任意目录，应能看到以下文件：
     demo/
     ├── .env.example      配置模板
     ├── archetypes.js     角色原型库
     ├── index.html        前端页面
     ├── server.js         后端服务（零依赖，无需 npm install）
     └── STARTUP.txt       本说明

2) 在 demo/ 目录下复制配置模板：
     - Windows 资源管理器：复制 .env.example，重命名为 .env
     - 或命令行：
         cd demo
         copy .env.example .env

3) 用记事本（或任何文本编辑器）打开 .env，按下方说明二选一填入 API Key。

   .env.example 已给出两套模板，复制其中一套的三行到 .env 即可：

   ---------- 方案 A：DeepSeek（推荐，便宜稳定）----------
   注册地址：https://platform.deepseek.com/
     ARK_API_KEY=sk-你的deepseek_key
     ARK_MODEL=deepseek-v4-pro
     ARK_BASE_URL=https://api.deepseek.com

   ---------- 方案 B：火山引擎方舟 ----------
   注册地址：https://console.volcengine.com/ark
     ARK_API_KEY=你的火山引擎_key
     ARK_MODEL=doubao-1.5-pro-32k
     ARK_BASE_URL=https://ark.cn-beijing.volces.com/api/v3

   说明：
   - 变量名虽为 ARK_*，但兼容任何 OpenAI 兼容协议的 Provider
   - 留空 ARK_API_KEY 也能启动，但会自动降级为规则模式，
     无法体验真正的 LLM 对话

4) 在 demo/ 目录下启动服务：

     node server.js

   看到类似 "服务器运行在 http://localhost:3000" 即成功。

5) 浏览器打开：

     http://localhost:3000

6) 页面右上角徽章应显示「LLM 模式」。
   按 UI 提示描述一个人的生活方式，系统将为你构建一位"卦中人"，
   然后开始陪伴对话。

【四、常见问题】

Q: 徽章显示「规则模式」怎么办？
A: 三种可能：
   1) .env 里 ARK_API_KEY 没填或填错
   2) .env 文件不在 demo/ 目录下（要和 server.js 同级）
   3) 火山引擎账号未开通方舟服务或 Key 无效
   解决后重启 node server.js。

Q: 启动报错 "Cannot find module"？
A: 本项目零依赖，不需要 npm install。
   确认你是在 demo/ 目录下执行的 node server.js，
   且 Node.js 版本 ≥ v18。

Q: 想换端口？
A: 启动前设置环境变量 PORT，例如：
     PowerShell:  $env:PORT=8080; node server.js
     CMD:         set PORT=8080 && node server.js
     macOS/Linux: PORT=8080 node server.js

Q: 对话很慢？
A: LLM 推理本身需要时间，角色构建阶段尤甚（多步串行），
   请耐心等待 30-90 秒。如长期无响应可刷新页面重试。

【五、隐私与安全】
- 你的对话内容仅在本地 Node 服务与火山引擎 API 之间流转，
  不会上传到任何其他第三方。
- .env 文件包含你的 API Key，请勿截图分享或上传到公开仓库。
- 输入内容不与身份绑定，不做医疗诊断。
- 若对话中出现自伤/自杀倾向，游戏会跳出叙事并提示真实求助资源。

【六、如不需要 LLM】
直接双击 index.html 用浏览器打开即可体验规则模式，
但功能受限（无角色构建、无联网画像推断）。
