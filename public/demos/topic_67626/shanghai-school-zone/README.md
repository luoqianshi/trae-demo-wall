# 上海学区查询系统

## 项目简介

这是一个基于Web的上海市学区查询系统，用户可以通过输入小区名称，在地图上查看小区位置，并获取该小区对口的幼儿园、小学、中学和高中等学区信息。

## 功能特性

- 🗺️ **地图定位**：集成高德地图，精确定位小区位置
- 🏫 **学区查询**：查询小区对口的幼儿园、小学、中学、高中
- 📊 **学校等级**：显示学校等级（市重点/区重点/普通）
- 🔍 **模糊搜索**：支持小区名称模糊匹配
- 📱 **响应式设计**：支持桌面端和移动端访问
- ⚡ **快速查询**：提供快捷小区标签，一键查询

## 技术栈

- HTML5 + CSS3 + JavaScript
- 高德地图 JavaScript API

## 使用方法

### 1. 运行项目

由于浏览器的安全策略（CORS），建议通过本地服务器运行：

#### 使用Python（推荐）

```bash
# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080
```

#### 使用Node.js

```bash
npx serve .
```

#### 使用VS Code Live Server插件

在VS Code中安装Live Server插件，右键点击`index.html`选择"Open with Live Server"。

### 2. 访问网站

打开浏览器，访问 `http://localhost:8080` 即可使用。
