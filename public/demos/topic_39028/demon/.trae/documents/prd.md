# Moment PRD

## 1. Product Overview
Moment is an AI-powered travel memory organizer that automatically arranges and presents travel memories.

## 2. Core Features
- Home: Travel list with cards
- Detail: AI status, timeline, map, hotel/flights, photo wall, AI diary, expenses, memory cards

## 3. Design Style
- Colors: Warm orange (#FF6B35), mint green (#4ECDC4)
- Cards: Large rounded corners, soft shadows
