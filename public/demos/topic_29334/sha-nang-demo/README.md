# 沙囊AI - 企业决策沙盘推演系统

## 项目简介

沙囊AI是一个面向小微企业主的AI决策辅助工具。它不会替你做决定，而是像一位可塑的、有想象力的参谋，帮你从乐观、中性、悲观三个情景推演企业决策，发现认知盲区，降低决策风险。

**核心理念**："AI不是决策者，也不是预言家。它是一个可塑的、有想象力的参谋。"

## 功能亮点

### 1. 三情景推演
- 每个决策从乐观、中性、悲观三个情景进行推演
- 直观展示各情景的概率分布
- 体现"不追求唯一正确答案"的产品哲学

### 2. 认知盲区发现
- AI帮你发现"你不知道你不知道"的因素
- 每个盲区可展开查看详细解释
- 盲区前置，强调"先发现问题再行动"

### 3. AI诚实提示
- 主动告知推演置信度和局限性
- 列出影响准确性的因素
- 体现"不隐瞒AI局限性"的诚实态度

### 4. 参谋对话模式
- 推演后可追问AI参谋
- 预设3个场景相关问题
- 支持自由输入问题，AI基于关键词智能回复

### 5. 可视化沙盘
- 五维雷达图展示企业能力评估
- 分数条颜色分段（低/中/高）
- 交互式数据点，hover显示详情

### 6. 引导式首次体验
- 5步引导，帮助用户快速上手
- 本地存储记忆，避免重复引导
- 体现产品"降低门槛"的设计理念

### 7. 场景对比
- 一键对比三个场景的核心结论
- 展示置信度、最可能情景、最大风险、首要行动

## 技术实现

- **纯前端架构**：HTML + CSS + JavaScript，无需后端
- **模块化设计**：数据、雷达图、引导、对话、对比等模块分离
- **响应式适配**：支持桌面端、平板、移动端
- **动画体系**：fadeInUp、scaleIn、countUp、typewriter等
- **无障碍支持**：prefers-reduced-motion媒体查询

## 使用说明

1. 打开 `index.html` 文件（可直接双击打开）
2. 跟随引导了解产品（首次打开）
3. 选择感兴趣的决策场景
4. 点击"构建沙盘"，AI分析企业情况
5. 点击"开始推演"，查看三情景结论
6. 查看认知盲区、行动建议、知识库引用
7. 与AI参谋对话，追问关心的问题

5. **新增功能模块**：
   - Onboarding引导系统

   - 分数条颜色分段和计数动画
