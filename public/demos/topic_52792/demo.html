从七个维度，带你全面了解小蓝格如何解决个人物品管理难题

解决思路

不同于简单的备忘录或 Excel 表格，小蓝格以「物品生命周期」为核心抽象，将静态记录升级为动态状态流转。每件物品从录入、服役、用完到退役/卖出，全程自动追踪日均成本与价值变化，让消费决策有据可依。

突破点：状态机驱动的物品生命周期管理

解决效果

自动计算日均使用成本，直观量化每笔消费的实际价值。过期提醒避免食品药品浪费，资产总览让家庭财务一目了然。告别「买了就忘」和「翻箱倒柜」，显著提升居家管理效率。

效率提升：日均成本自动计算 + 过期智能预警

核心链路

核心操作链路完整跑通：添加物品（表单+拍照+AI识别）→ 首页浏览（筛选+搜索）→ 详情查看（成本+状态）→ 状态切换（用完/退役/卖出）→ 统计分析（饼图+柱状图）。每一步都有清晰的视觉引导和操作反馈。

链路验证：录入 → 浏览 → 详情 → 状态流转 → 统计分析

价值传递

首页资产总览卡片直接展示总资产、日均成本和状态分布，用户打开即见核心价值。物品详情页的「日均使用成本」让每件物品的性价比一目了然，目标成本对比功能帮助用户做出更理性的消费决策。

价值感知：资产总览 + 日均成本 + 目标成本对比

视觉设计

天蓝色品牌主色搭配卡片化布局，大圆角+柔和阴影营造层次感。Light/Dark双主题自适应，全局矢量图标保持视觉一致性。骨架屏加载、入场动画、微交互（按压缩放、卡片悬停）等细节打磨到位。

设计规范：双主题 + 设计令牌系统 + 动画体系

交互友好度

左滑露出操作按钮、下拉刷新、底部悬浮导航栏——交互模式符合移动端用户习惯。首次启动引导页降低学习成本，危险操作二次确认防止误操作，Toast四类提示即时反馈操作结果。

交互亮点：左滑操作 + 下拉刷新 + 引导页 + 二次确认

场景洞察

精准识别四大真实场景需求：居家收纳（记录位置分类，告别翻箱倒柜）、资产管理（追踪价格与日均成本，量化消费价值）、效期管理（食品药品化妆品过期监控与提醒）、断舍离助手（物品状态流转管理生命周期）。目标用户为注重生活品质、希望科学管理家庭物品的群体。

目标用户：注重生活品质、追求数字化家庭管理的用户群体

数据安全

全链路加密 · 原生安全存储 · 完善的异常处理与数据恢复机制

加密体系

备份文件 AES-256 加密（用户密码派生），自动备份使用设备派生密钥。设置文件、分析数据、AI配置均加密存储。

原生安全存储

Android 使用 AndroidKeyStore 派生密钥，Windows 使用 DPAPI。密钥与设备/操作系统用户绑定，跨设备不可移植。

回收站与恢复

软删除机制，删除物品先进入回收站，30天内可恢复。批量操作使用事务保证原子性，操作日志记录便于数据恢复。
