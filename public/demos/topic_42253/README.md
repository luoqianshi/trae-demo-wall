# PyCoach 静态评审 Demo

这是 PyCoach 主系统真实运行后的静态回放包，用于比赛社区上传和评审快速体验。

## 使用方式

1. 解压 `PyCoach-static-demo.zip`。
2. 双击打开 `pycoach-static-review-demo.html`。
3. 按页面底部的「下一步」查看完整学习闭环。

## 说明

- 本 Demo 不需要后端服务、Python 环境、Node 环境或模型 API。
- 页面展示的是真实运行快照，不会再次请求模型。
- 真实系统体验需要运行 PyCoach 主项目，并在设置中导入可用模型配置。
- 包内不包含密钥、原始中转站配置、环境变量密钥文件或本机绝对路径。
