# EQagent - 人际关系修炼助手

基于熊太行《识人攻略》《关系攻略》《掌控关系》提炼的人际关系处理技能，覆盖**职场、家庭、校园、体制内**等全场景，帮助你识人心、辨真伪、处关系。

## 适用人群

- **职场人士**：应对同事、上下级、客户关系
- **大学生**：处理校园人际、宿舍关系、师生关系
- **公务员/体制内**：适应机关文化、官场规则
- **家庭场景**：婆媳关系、夫妻关系、兄弟姐妹关系

## 核心理念

### 1. 识人先于应对
- 先识别对方是什么类型的人，再决定应对策略
- 不要用同一种方式对待所有人

### 2. 利益分析法
- 理解"利益接合部"概念：双方利益一致处越小，冲突越大
- 公与私的判断标准：看对方从谁的利益出发做决定

### 3. 行为胜于言语
- 关注对方**怎么做**，而非**怎么说**
- 言行一致者可信，言行不一者需警惕

### 4. 放下情绪，聚焦行动
- 情绪是工具，不是反应
- 在冲突中保持冷静，把精力放在解决问题上

### 5. 鹰鸽互补
- 优秀的鹰派会生出悲悯，优秀的鸽派会长出骨头
- 根据不同场景切换鹰鸽策略

## 应对策略速查

| 类型 | 识别特征 | 核心策略 |
|------|----------|----------|
| 推脱者 | 回避改进建议 | 绕道求助上级或换路径 |
| 伪君子 | 言行不一 | 避免把柄，不与其争论 |
| 标榜者 | 空谈道德无行动 | 不迁就、不激惹 |
| 畏祸者 | 关键时刻退缩 | 稳住情绪，准备备选 |
| 逐利者 | 待价而沽 | 满足利益诉求 |
| 投机者 | 顺风倒 | 保持距离 |
