# 如何向领导谈钱

## 核心原则

### 1. 谈钱不伤感情
- **谈钱不伤感情，谈感情才伤感情，谈感情还会伤钱**
- 谈利益并非不道德
- 别人跟你说他自己重情义，多半是来伤钱的！

### 2. 利益接合部是下手的地方
- 利益接合部就是一个人本人和他的职业角色之间的利益缝隙
- 级别越低的人，他和公司的利益接合部就会越远，也就越容易突破
- 跟别人谈钱如果有困难，一定要先从帮私不帮公开始
- 如果对方不是公司的老板，一定不要为对方的公司白干活儿

### 3. 领导不差钱，差的是情感需求
- 领导不差钱、领导差佩服、差尊重、差孝顺、差关爱
- 不能解决全部四点，至少给他一点亮，一点暖
- 让他知道还有人惦记着他

### 4. 谈加薪的时机
- 周一开会的时段，周五是人们准备松懈的时段，这个时候把球踢给老板不妥
- 最好的办法是周三或者周四，老板可能在本周内解决你的请求
- 大多数的老板都会对下班前的那一小段时间的低效睁一只眼闭一只眼
- 用这段时间去见见老板，谈论一下薪水，并不算失礼

## 常见问题及应对策略

### Q1：谈加薪之前要做什么？
**应对策略**：
1. **布局人脉**：进到一个公司里，要尽可能和行政、人力、财务的同事交朋友，跟前台的关系也要搞好
2. **了解规定**：公司提薪有关规定，表现特别勤快突出的可以申报，但需要一个过程
3. **做好准备**：精心准备观点抛出来，和临时定题目、谈论一件事是完全不同的难度
4. **认清价值**：不要动用私人交情去压价，很便宜的价格请来自己的朋友们，领导也会觉得理所当然，你反而没有太多功劳了

### Q2：谈加薪时的几个不要？
**应对策略**：
1. **不要周一、周五谈**：周一是开会的时段，周五是人们准备松懈的时段
2. **不要威胁**：不要以离职来威胁谈加薪
3. **不要攀比**：不要拿同事的工资来攀比
4. **不要情绪化**：不要带着情绪去谈

### Q3：女同学压价请我讲课，如何谈钱？
**应对策略**：
- 游说办法：突破她的利益接合部
- 作为公司的雇员，她最好是对你压价，但对她个人来说，最好的办法是保证你的价格，让公司稍微多花一点钱
- 建议她去为你申请更多的预算，她的领导会为她和更有价值的讲课者有交情而对她评价更高
- 如果她动用私人交情去压价，领导也会觉得理所当然，她反而没有太多功劳了

### Q4：公司降薪2000，如何取舍？
**应对策略**：
- 背景：部门的小领导，手下已经精简到只有1个手下，业务不冲突且不可或缺，公司业务确实不饱和，几乎从不加班，和大老板关系还算融洽
- 需要权衡：留下还是离开
- 如果留下，要考虑是否有晋升机会
- 如果离开，要提前布局人脉，寻找新的出路

### Q5：老好人的工资问题？
**应对策略**：
- 老好人容易被压价，不敢谈钱
- 健康的认知模式会给自己尽量减小压力
- "谈钱不伤感情"：明白谈利益是正常的、合理的
- 不要用"重情义"来压自己，别人说重情义多半是来伤钱的

## 实用建议

### 谈加薪的策略
1. **帮私不帮公**：跟别人谈钱如果有困难，一定要先从帮私不帮公开始，如果对方不是公司的老板，一定不要为对方的公司白干活儿
2. **布局安全关系网**：到一家新公司，除了推荐人，最先见到的就是前台，要首先跟她混熟，她很重要，她不是一颗闲棋冷子
3. **选择时机**：周三或周四，下班前的那一小段时间
4. **展现价值**：不要动用私人交情去压价，保证你的价格，让对方去申请更多预算

### 面对降薪
- 如果公司业务不饱和、几乎从不加班 → 权衡留下还是离开
- 考虑是否有晋升机会
- 如果介意自我，干脆离开这个企业才是王道

### 基层干部面对处分扣钱
- 认真跟领导谈一下，算一下账单
- 如果大家不一起负担此事，会一直申诉下去
- 争取补偿，但要明白代价：以后别指望提升了，要明白这个四万元的代价
