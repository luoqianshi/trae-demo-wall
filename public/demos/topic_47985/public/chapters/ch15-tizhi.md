# 体制内生存法则

## 核心原则

### 1. 不当"老好人"
- **真性老好人**：温和谦逊、业务能力强，不因模式感到折磨，不需要改变
- **假性老好人**：因阴差阳错变成老好人，需要转变
- **体制内的老好人**：大部分被当成窝囊废，真正成功的少
- 有个性的人机会更多，强行改变容易变成"抖机灵"

### 2. 烟文化不必学
- 成年后没学会抽烟，就再也别学
- 靠抽烟能换来的消息，基本都是底层信息
- 口气、黄牙会让你失去别的机会
- 这不是什么文化，是不值得碰的东西

### 3. 内心自洽优先
- 做事不要怕套路，套路一点都不俗
- 不爱惜自己、伤害内心自洽的事一概拒绝
- 尊重自己，比迎合他人更重要

### 4. 利益接合部意识
- 行事要考虑对方个人的利益，可能比公司利益更重要
- 级别越低的人，和公司的利益接合部越远，越容易突破
- 要明白领导的利益接合部：领导代表公司，又不是公司

## 常见问题及应对策略

### Q1：工作很闲，周围的人都很忙，会被讨厌吗？
**应对策略**：
- 少开玩笑，少说废话，多观察
- 不要被同事牵着鼻子走，同事再好也是同事，领导再坏也是领导
- 体制内部门人少时（3-4人），没有负责人安排自己主动做事，确实显得奇怪
- 观察组织里大家的角色，找到利益接合部

### Q2：被大机关借用，想留下来但工作越来越多怎么办？
**应对策略**：
- 几乎天天加班到凌晨、请病假还被骂 → 与处长哭一次
- 有些死硬的家伙吃软，哭着表达想要为他工作但压力太大才病倒
- 眼泪有时是策略，有时是武器
- 就俩人看见，他孤家寡人又没朋友，也没地儿说去

### Q3：处处受欺负、排挤怎么办？
**应对策略**：
- 送礼物给他们，没有回报好意，人家也看不上 → 做再多事也不会得到认可
- 不要指望用礼物改善关系，提升自己的实力才是根本
- 真正的成功者是有个性的人，不是一味迎合的人
- 考虑提升自己的能力，或者寻找新的出路

### Q4：被人评价为老好人，如何摆脱标签？
**应对策略**：
- 假性老好人的解释模式：
  - "太好了，正好大家都不担心我，我可以安心观察这个组织里大家的角色了"
  - "这帮人的利益接合部被我找到了，我知道该如何入手了"
- 健康的认知模式会给自己尽量减小压力
- 大力不能出奇迹，大的压力只会毁灭任何奇迹

## 实用建议

### 日常行为准则
1. **收敛自我**：性格上比较自我的人，混国企/机关要收敛，衣服颜色要素，说话要少而克制
2. **工作用微信**：申请一个新微信工作用，和以往的旧时代分开
3. **不随便给建议**：除非对方花钱请你给建议，不然落怨恨
4. **不主动提免费建议**：不要主动向别人提免费的建议

### 应对加班文化
- 午休在工位吃饭时，领导让做不重要不紧急的工作 → 做完下午在领导面前补吃这个饭，说中午加班了
- 晒加班 → 老板点赞，其实老板不保护这位童鞋，类似酒桌的感觉
- 公司业务不饱和、几乎从不加班 → 面对降薪，要权衡留下还是离开

### 面对处分/问责
- 不知情的情况下被处分 → 认真跟领导谈一下，算一下账单
- 如果大家不一起负担此事，会一直申诉下去
- 要争取补偿，但要明白代价（以后别指望提升了）

**体制内生存核心**：
- **不当老好人**：有个性的人机会更多

- **利益接合部**：考虑对方个人利益，可能比公司利益更重要
- **收敛自我**：说话少而克制，申请工作用微信

**行动指南**：
1. 观察组织里大家的角色，找到利益接合部
2. 不随便给建议，不主动提免费建议
