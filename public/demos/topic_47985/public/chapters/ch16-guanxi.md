# 体制内上下级关系

## 核心原则

### 1. 领导再坏也是领导
- 不要被同事牵着鼻子走，同事再好也是同事，领导再坏也是领导
- 直属领导，这么对抗没有好处
- 你家什么背景都没有，其他同事未必

### 2. 领导不差钱，差的是情感需求
- **领导不差钱**
- **领导差佩服**
- **领导差尊重**
- **领导差孝顺**
- **领导差关爱**
- 不能解决全部四点，至少给他一点亮，一点暖，让他知道还有人惦记着他

### 3. 汇报工作的智慧
- 想通过在大领导面前的一次汇报就给对方留下深刻印象从而平步青云是基本上不可能的
- 对大领导有这样效果的可能只有他的秘书
- 自己的前途和命运基本上是掌握在直接领导手中的
- 汇报工作要以直接领导的利益和角度为中心

### 4. 面对两个领导的不同意见
- 当一个工作，一个领导让你做，一个让你不做，一个让你做成a，一个让你做成b
- 说明他们的利益接合部不一样
- 听直管领导的，大分管领导越级给你下大任务，这事儿不妥

## 常见问题及应对策略

### Q1：直属领导能力差、情商不高、管理混乱怎么办？
**应对策略**：
- 不要记录领导的缺点，回去看看领导的优点
- 不要跟同事怼他，至少不得罪他
- "你喜欢记录下这个人的所有缺点" → 熊十力大喝："像你这样，读多少书都没用！"
- 直属领导这么对抗没有好处，你家什么背景都没有，其他同事未必

### Q2：向领导汇报时，夜班组长轻蔑地笑怎么办？
**应对策略**：
- 直接质问他："笑什么"，请他谈谈对昨晚夜班工作的评价，叫他笑
- 或者换个地方站，让他冲你笑不着
- 直接请他谈谈评价是最好的方式

### Q3：正副两位领导不和，汇报工作怎么办？
**应对策略**：
- 专栏内搜"两个领导"有专门课程
- 分管我们部门的是副总，正总以前抛出过橄榄枝，这个也被副总看在眼里
- 汇报工作是向副总汇报还是向两位同时汇报 → 看利益接合部

### Q4：体制内，原来的直属领导升职做了副总（但不分管我们部门），怎么继续保持关系？
**应对策略**：
- 仍然经常请教，点赞她的朋友圈，热情评论
- 男孩哄姐姐开心不难，永远不在她面前膨胀
- 虽然汇报对象发生变化，不像以前那样经常汇报，但可以继续请教、交流
- 准备小礼物方面，保持适当的关系维护

### Q5：面临调动或升迁的领导发起谈心活动，是什么意思？
**应对策略**：
- 此时的谈心不再是广撒网，而是在已经确立君臣或师徒关系的下属中进行
- 一般是三个意思：
  1. **挖角**：希望你跟着他走
  2. **考察**：看你是否忠诚、可用
  3. **布局**：为新岗位做准备

## 实用建议

### 汇报工作要点
1. **准备工作**：对于汇报内容要反复思考和推敲，尽量把问题考虑全面，尤其是细节一定要心中有数
2. **认清角色**：小角色被带到大领导面前就是为了解释细节，补充要到位和及时
3. **尊重直接领导**：汇报前除了内容外还应该就内容向直接领导汇报和沟通，必须理性地知道自己的前途和命运基本掌握在直接领导手中
4. **紧张不等于混乱**：紧张只是一种局促和不安，不等于语无伦次和逻辑混乱

### 面对领导情绪不好
- 老板情绪不好时，助理就发挥作用
- 跟老板信任的助理关系好点，好处多多，至少不会在老板面前说你坏话

### 向领导请示
- 复杂的局面请示主管领导，请他来定夺
- 请示领导还有一点，就是你会更安全
- 不论是学生工作还是其他工作，都要请教领导，他们的经验也很丰富

### 避免越级汇报
- 如果一个工作，分管领导不在，另一个领导就开始按自己的想法乱搞一通
- 等分管领导回来时，汇报工作，领导一听就觉得是你趁他不在时就瞎搞
- 体体制就是这么的操蛋，说明自己工作压力过大的苦衷是最适用的方法

## 总结要点

**上下级关系核心**：
- **领导再坏也是领导**：不要被同事牵着鼻子走
- **领导不差钱**：差的是佩服、尊重、孝顺、关爱
- **汇报智慧**：前途和命运基本掌握在直接领导手中
- **面对分歧**：听直管领导的，看利益接合部

**行动指南**：
1. 看领导优点，不记录缺点，不跟同事怼领导
2. 汇报工作以直接领导的利益和角度为中心
3. 准备工作要反复思考推敲，细节心中有数
4. 复杂局面请示主管领导，你会更安全
