# 如何向领导提升职

## 核心原则

### 1. 文章写得好容易被提拔
- 文章写得好，证明你的听说读写四大技能点都比别人高
- 用智商测试不够礼貌，用文章可以判断别人的见识和受教育程度
- 领导要提拔的人，需要各方面能力都比较强

### 2. 领导需要的是忠诚和能力
- 领导往往会有目标感，有能力解决问题的人中，选择对自己忠心的人培养
- 领导/老板喜欢迎合自己的人，这是误会
- 领导会选择有能力解决问题的人中，对自己忠心的人培养

### 3. 提升前的准备
- 继续忍，等你被提升了才有机会
- 当然要考虑的是你有没有机会晋升了
- 如果很介意自我，干脆就离开这个企业才是王道

- 格局放大一点，组员以后可能都是你手底下的人

## 常见问题及应对策略

### Q1：如何开口向领导提升职？
**应对策略**：
1. **做好业绩**：业绩完成率和增长率在十个人里排第一，承担了可沉淀的工作，比如产品资料汇总、建立系统查询产品提高效率、带新员工、坚持学习
2. **认清形势**：部门人员超编，年中会议上一把手说过要输出人才；直属领导比较软弱，不争利益
3. **提前布局**：如果付出却得不到升职或加薪继续留下来也怕自己没有了斗志，调岗到其他部门是更好的选择
4. **直接沟通**：向领导表达自己的想法和诉求，但要方式得体

### Q2：做公务员被提拔后，如何处理同事关系？
**应对策略**：
- 跟同事相处时间比家人都长，而且超级稳定
- 随着学习和分析，加上被提拔后能看到更多，渐渐意识到很有问题
- 现在怎么做：仍然友好，心里要明白
- 以前当朋友（现在的平级），以前当朋友的（下级）应该怎样继续相处 → 仍然友好，心里要明白

### Q3：没有明确的领导职务，但起到了领导作用，名字却在后面怎么办？
**应对策略**：
- 银行研究部门工作，成立小组当组长，业务上有所收获，得到领导认可
- 报告署名：大领导第一，自己第二，两位同事在后
- 突然领导决定每个报告主笔人在前 → 领导都要提拔你了，你还担心他不知道你的贡献？
- 格局放大一点，组员以后可能都是你手底下的人，看大领导为什么不纠结自己名字在后面，主笔在前面？

### Q4：领导曾经找我谈话说可能提拔我，怎么避免犯错？
**应对策略**：
- 不要觉得小气没格局
- 继续领导整个小组的工作，不要因为名字顺序就放弃领导工作
- 格局放大一点，组员以后可能都是你手底下的人
- 大领导为什么不纠结自己名字在后面，主笔在前面？好好想想

### Q5：教会徒弟害死师傅怎么办？
**应对策略**：
- 公司做项目同事以前不是做这一行的什么都不会，总是过来问
- 实话说不想教他们，感觉威胁到自己的晋升、平级，存在竞争关系
- 她不能提供什么给我，上次她发现她会在上司哪里邀功
- 如果邀的是你的功的话，别教了
- 养小弟是可以的，前提是小弟比你低很多，教手下的手下是师徒，教同级或者略低的，那就是孙膑庞涓故事

## 实用建议

### 提升职前的准备
1. **业绩突出**：业绩完成率和增长率排第一，承担可沉淀的工作
2. **带新员工**：坚持学习，希望带动其他人让部门有活力
3. **认清形势**：了解部门人员超编情况，直属领导是否争利益
4. **提前布局**：如果得不到升职或加薪，考虑调岗到其他部门

### 提升职的方式
1. **直接沟通**：向领导表达自己的想法和诉求，方式得体
2. **展现能力**：听说读写四大技能点都比别人高
3. **展现忠诚**：在有能力解决问题的人中，展现对领导的忠心
4. **格局放大**：不要纠结细节，组员以后可能都是你手底下的人

### 面对提拔后的关系
1. **仍然友好**：心里要明白，以前当朋友（现在的平级），以前当朋友的（下级）
2. **认清角色**：跟同事相处时间比家人都长，而且超级稳定，但心里要明白
3. **不膨胀**：永远不在前领导面前膨胀，点赞她的朋友圈，热情评论
4. **继续请教**：虽然汇报对象发生变化，可以继续请教、交流

### 面对竞争
1. **不教同级**：如果邀的是你的功的话，别教了
2. **养小弟**：前提是小弟比你低很多，教手下的手下是师徒
3. **孙膑庞涓**：教同级或者略低的，那就是孙膑庞涓故事
4. **保持距离**：保持积极的态度，但对竞争者要警惕

- **格局放大**：不要纠结细节，组员以后可能都是你手底下的人
