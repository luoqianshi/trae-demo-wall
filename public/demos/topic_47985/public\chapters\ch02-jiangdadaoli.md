# 识别爱讲大道理的人

## 三种类型

### 1. 方正君子
- 言行一致
- 自己遵守自己讲的道理
- **可以好好相处**

### 2. 伪君子（乡愿）
- 用道德苛求别人，自己无原则
- 道德绑架是惯用伎俩
- 孔子说："乡愿，德之贼也"

### 3. 自保者
- 处境不佳缺乏盟友
- 大道理是最后的"铠甲"
- 不擅处理人际关系

### 4. 控制狂
- 用大道理输出价值观
- 从无关紧要小事入手批评
- 目的是让你不敢违逆他

## 识别方法

**关键：言行是否一致**

- 言行一致 → 方正君子
- 言行不一 → 伪君子/控制狂

## 应对策略

| 类型 | 策略 |
|------|------|
| 伪君子 | 避免被抓住把柄 |
| 自保者 | 避免成为其同盟 |
| 方正君子 | 宽容友好相处 |
| 控制狂 | 不被驯从，保持独立 |

## 核心原则

1. **不要争论** - 跟他在嘴上分胜负没有意义
2. **看行为** - 对方怎么做比怎么说重要
3. **做实事** - 在事上见真章

## 总结

- 爱讲大道理不一定有坏心
- 从言行一致判断好坏
- 重点提防伪君子和控制狂
