# 人际关系速查决策树

## 职场识人

### 遇到"公事公办"的人？
```
是公事公办的人吗？
├─ 求助测试：愿意帮助改进 → 忠于职守，正常合作
└─ 只说不知道 → 推脱者
      │
      └─ 原因分析
         ├─ 怕担责 → 完善方案，请上级协调
         ├─ 要利益 → 分享收益
         ├─ 刷存在感 → 戴高帽子
         └─ 给下马威 → 工作外给甜头，工作内不让步
```

### 遇到"爱讲大道理"的人？
```
言行一致吗？
├─ 是 → 方正君子，可以好好相处
└─ 否
   ├─ 处境不好，用道理自保 → 不当同盟
   ├─ 伪君子戏精 → 避免把柄
   └─ 控制狂 → 不被驯从，保持独立
```

### 遇到"道德洁癖"的人？
```
是谁？
├─ 标榜者（要控制你）→ 不迁就、不激惹
│   ├─ 点到为止
│   ├─ 不顺着说
│   └─ 被点名时表明态度
└─ 有认知谬误 → 可以宽容相处
```

### 遇到"神色为难"的人？
```
关键测试：为什么拒绝帮你？
├─ 怕得罪强者 → 畏祸者
│   ├─ 稳住情绪
│   ├─ 强硬交涉
│   └─ 准备备选方案
└─ 想索取高价 → 逐利者
    ├─ 满足利益诉求
    └─ 事后观察处置
```

### 遇到"宣誓效忠"的人？
```
观察什么？
├─ 你失势时他还在吗？
├─ 愿不愿承担困难任务？
└─ 变了还跟不跟你？

→ 顺风倒 → 保持距离，不给实权
→ 不变 → 可考虑培养
```

## 人格与情绪

### 判断鹰派还是鸽派？
```
看行为，不是看怎么说
├─ 咄咄逼人、相信实力 → 鹰派倾向
└─ 注重合作、相信真情 → 鸽派倾向

最佳状态：
└─ 鸽四鹰六 → 最容易出狠角色
```

### 鹰派需要注意什么？
```
1. 谜之自信 → 需要鸽派副手
2. 冲动是魔鬼 → 克制攻击性
3. 学习悲悯 → 鹰派应该是骑士，不是魔头
4. 避免逆境崩盘 → 从别人那里体验挫折
5. 不做十分鹰 → 十分鹰有十分敌人
```

### 鸽派需要做什么？
```
长出骨头 → 学会拒绝、建立边界、必要时出手
```

### 嫉妒怎么办？
```
自查清单：
1. 我在嫉妒吗？（承认它）
2. 我嫉妒谁？（识别对象）
3. 为什么嫉妒？（分析原因）
4. 我能从他身上学到什么？（转化为动力）
5. 我需要改变什么习惯？（行动）

关键：
└─ 增强实力是解决嫉妒的大道
```

## 家庭关系

### 婆媳冲突怎么办？
```
丈夫的处理方式（吴聘式）：
1. 表明态度
2. 承诺共进退
3. 态度柔和打动媳妇

核心认知：
└─ 父母绝对不会把儿子赶出家门
   这是丈夫最大的筹码
```

### 夫妻关系核心原则？
```
夫妻关系是家庭中第一位的

维护策略：
├─ 纪念日仪式感
├─ 共同学习成长
├─ 家里设"会议桌"沟通
└─ 发现出轨时冷静处理
```

### 兄弟姐妹借钱？
```
亲兄弟明算账

处理流程：
1. 问用途 → 合理用途才考虑
2. 问还款计划 → 没计划不借
3. 写借条 → 金额、还款日期、用途
4. 设还款提醒 → 到期前提醒
5. 到期不还 → 每月提醒一次，不催

关键：
└─ 疏不间亲——不介入对方的夫妻关系
```

## 校园关系

### 遇到校园欺凌？
```
不做"包子"
├─ 拒绝从众压力
├─ 克制表达，不当众反抗
├─ 鹰鸽兼备
└─ 实力结盟

遇到欺凌时：
1. 保护自己安全
2. 记录证据
3. 告诉信任的人
4. 寻求学校帮助
```

### 宿舍矛盾？
```
核心原则：
├─ 理解鹰鸽差异
├─ 建立边界
├─ 克制表达
└─ 避免情绪传染

冲突处理步骤：
1. 冷静分析
2. 直接沟通（私下）
3. 提出解决方案
4. 寻求第三方调解
```

### 与导师相处？
```
核心原则：
├─ 尊重导师
├─ 理解师生关系本质（和上下级相通）
├─ 反求诸己
└─ 避免关系破裂

建立良好关系：
├─ 认真完成任务
├─ 及时汇报进展
├─ 主动请教问题
└─ 表达感激
```

## 体制内生存

### 体制内核心法则？
```
1. 不当老好人 → 有个性的人机会更多
2. 烟文化不必学 → 不必强融
3. 内心自洽优先 → 不被同事牵着走
4. 利益接合部意识 → 找到下手的地方
```

### 体制内上下级关系？
```
核心原则：
├─ 领导再坏也是领导
├─ 领导不差钱，差情感需求
├─ 汇报工作要智慧
└─ 两个领导分歧时听直属的

汇报技巧：
├─ 定期汇报
├─ 重大决策请示
├─ 方案准备充分
└─ 表达忠诚和能力
```

### 向领导谈钱？
```
时机选择：
├─ 任务完成后
├─ 年度考核前
├─ 获得重大成果后
└─ 避免部门困难时

准备工作：
1. 成绩清单
2. 市场行情调研
3. 提出具体数字
4. 准备应对压价
```

### 向领导提升职？
```
核心认知：
├─ 文章写得好容易被提拔
├─ 领导需要忠诚和能力
└─ 提升前要有准备

方式选择：
├─ 正式谈话（准备充分）
├─ 日常暗示（持续输出）
└─ 寻找机会（领导心情好时）
```

```
想修复关系吗？
├─ 是
│   ├─ 对方是好人 → 主动示好
│   ├─ 对方是坏人 → 评估风险后再决定
│   └─ 对方是中间人 → 可以修复
└─ 否 → 渐行渐远，不撕破脸
```
