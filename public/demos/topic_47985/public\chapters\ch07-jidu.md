# 嫉妒：如何把毒药变成春药

## 什么是嫉妒？

**定义**：看见别人取得成就或收获感情后感受到屈辱，并对对方抱有敌意和厌恶的感情。

## 嫉妒 vs 怨恨

| 嫉妒 | 怨恨 |
|------|------|
| 刃口对内的刀子 | 刃口对外的刀子 |
| 折磨自己 | 折磨对方 |
| 通过羞耻感伤害人 | 想毁掉得不到的东西 |
| 嫉妒者比怨恨者怂 | 怨恨者攻击性强 |

**心理学家最感兴趣的两种情感就是"嫉妒"和"害羞"。**

## 两种嫉妒

### 1. 成就嫉妒
- 紧扣"别人家的孩子"
- 适度的成就嫉妒是有益的

### 2. 性嫉妒
- 关于"自家的老公"
- 伴侣之间的不良情绪
- 低自尊者常见

## 成就嫉妒：从毒药到春药

**适度的成就嫉妒能激发进步动力**

### 中国父母的教育方式

传统中国父母不习惯称赞式教育，而是用称赞别的孩子来激起嫉妒：
- 提升孩子自尊程度
- 引导孩子服从
- 让孩子更认真学习

孔子在校园里也这样：颜回就是"别人家的孩子"，让子贡惭愧。

### "好的嫉妒"如何发挥作用

**三个关键点：**

1. **称赞榜样时坚持具体化**
   - 不要说"他处处比你强"
   - 要说"他每天做完作业再玩"
   - 描绘的是学习对象，不是决斗对象

2. **称赞榜样，不挖苦贬低面前的人**
   - 对自尊水平高的人，只需指出榜样即可

3. **对"奋起直追"的行为提供帮助**
   - 不能只批评不给支持
   - 父母批评孩子时自己却打麻将，孩子成绩堪忧

### 成年人如何用嫉妒驱策自己

```
分析嫉妒对象的具体做法 → 学习 → 告诉自己也很棒 → 改变不良习惯 → 提高效率
```

## 我们如何挑选嫉妒对象

正常人不会随便嫉妒人，被嫉妒者往往符合以下条件：

### 1. 和嫉妒者很像
- 身份、地位、相貌、年龄越接近，嫉妒可能性越高
- 作家嫉妒作家，不嫉妒医生

### 2. 他的优点我很在乎
- 《天龙八部》：慕容复嫉妒萧峰（武林地位朋友多），段誉嫉妒慕容复（王语嫣喜欢他）

### 3. 认为自己更厉害
- 在对方成功的领域，觉得自己应该更强

### 4. 认为对方运气好
- 把对方成功归结为运气

### 5. 对方不是最好的朋友
- 嫉妒产生在熟人到一般朋友之间
- 对好朋友有"嫉妒豁免权"——"他是我兄弟，替他高兴"

**关羽的嫉妒豁免名单：**
- 张飞：弟弟
- 赵云：跟了哥哥很久，也是弟弟
- 马超：名门之后
- 黄忠：算老几？（关羽嫉妒的对象）

## 恶性嫉妒：危机重重

### 恶性嫉妒的触发条件

1. **极端行为出现**
   - 孙膑被庞涓嫉妒，骗去魏国剜去膝盖骨

2. **因为嫉妒不肯原谅自己**
   - 内心自洽严重破坏

### 如何克制恶性嫉妒

**三个步骤：**

1. **提醒自己"你在嫉妒了"**
   - 自己承认，感受立马好很多
   - 嫉妒对你的压迫到此为止

2. **开出嫉妒豁免名单**
   - 把好友放进去
   - 像关羽那样

3. **谈谈你的感受**
   - 找信任你的人倾诉
   - 或找心理咨询师
   - **不要找认识被嫉妒者的人聊**

## 性嫉妒：看好自己家的老公

### 什么是性嫉妒？

- 对配偶性魅力过于迷信
- 对配偶性道德过分不信任
- 低自尊者常见

### 区分性嫉妒 vs 正常担忧

| 正常担忧 | 性嫉妒 |
|----------|--------|
| 指出老公和女同事半夜聊 | 认为伴侣没有任何自主能力 |
| 大胆假设、合理怀疑 | 怀疑一切身边的人 |
| | 有跟踪调查追问行为 |
| | 使用暴力 |

### 性嫉妒的特征

- 不是天生的，逐渐变成这样
- 性嫉妒者和性嫉妒者组合不罕见
- 性嫉妒可能传染

### 性嫉妒的"好处"（其实是代价）

历史上：
- 男性：保证孩子血统纯正
- 女性：避免婚内财产损失

今天：
- **好处根本无法和损失相比**

性嫉妒者会沉迷此道，输出价值观，在朋友圈替人妻出头，获得尊重和权力。

## 解决之道

### 成就嫉妒的解决方案

**增强自己的实力是解决成就嫉妒和性嫉妒共同的大道。**

用好了，嫉妒是强者的春药。
用不好，嫉妒是弱者的毒药。

### 性嫉妒的解决方案

- 增强自信
- 不沉迷猜疑
- 把精力放在提升自己上

## 金句

> "用好了，嫉妒是强者的春药；用不好，嫉妒是弱者的毒药。"
