硬件接入操作指南

适用 已有支持 MQTT 的 WiFi 温湿度设备
1
确认设备支持 MQTT

查看设备说明书，确认支持通过 MQTT 协议上报数据。常见支持型号：精创 WiFi 版、部分 IoT 温湿度传感器。

2
配置 MQTT 连接参数

在设备管理后台或配套 App 中填入以下信息：

Broker 地址: mqtt://<服务器IP>:1883
主题格式: sensor/{设备编号}/data
上报间隔: 建议 5~30 秒
3
设置数据格式

确保设备上报的 JSON 格式如下：

{
"temperature": 25.6,
"humidity": 58.2,
"battery": 85
}
4
在本系统中添加设备

进入「设备管理」>「添加设备」，填写设备名称、编号（需与 MQTT 主题中的设备编号一致）、选择传感器型号和连接方式。

适用 使用 LoRa 传感器的大面积实验室/仓库
1
准备 LoRa 网关

购买支持 MQTT 转发的 LoRa 网关（如 Dragino LG01、瑞兴恒方等），或使用树莓派 + LoRa 模块自建网关。

2
部署 LoRa 传感器

将 LoRa 温湿度传感器（如华汉维 T40L-E、康耐德 LoRa 系列）安装在监测点位，记录每个传感器的设备编号。

3
配置网关 MQTT 转发

在网关管理页面设置 MQTT Broker 地址为本系统服务器 IP:1883，配置数据转发规则，将传感器数据映射到主题 sensor/{设备编号}/data。

4
验证数据上行

观察网关日志确认数据正常转发，然后在本系统中添加对应设备即可看到实时温度。

适用 已有 RS485/Modbus 温控仪表的实验室
1
准备协议转换器

需要一台支持 RS485 转 MQTT 的边缘设备，推荐方案：

  • 方案 A：购买商用协议转换网关（约 ¥200-500）
  • 方案 B：树莓派/工控机 + USB 转 RS485 模块 + Node-RED
  • 方案 C：ESP32 + RS485 模块（成本低，适合少量设备）
2
接线与配置

将温控仪表（虹润 NHR-5100、宇电等）的 RS485 A/B 线接入转换器，确认波特率、数据位、校验位一致（通常为 9600/8/N/1）。

3
编写/配置转发脚本

使用 Node-RED 或 Python 脚本定时读取 Modbus 寄存器，将温度值封装为 JSON 通过 MQTT 上报。

# Python 示例 (pymodbus + paho-mqtt)
from pymodbus.client import ModbusSerialClient
import paho.mqtt.client as mqtt
import json, time

client = ModbusSerialClient(port='/dev/ttyUSB0', baudrate=9600)
client.connect()
mqttc = mqtt.Client()
mqttc.connect("服务器IP", 1883)

while True:
rr = client.read_holding_registers(0, 1, slave=1)
temp = rr.registers[0] / 10.0
mqttc.publish("sensor/A-101/data",
json.dumps({"temperature": temp}))
time.sleep(10)
4
系统中添加设备

在「设备管理」中新增设备，设备编号与 MQTT 主题中的编号保持一致，选择对应的传感器型号。

适用 有一定开发能力，希望低成本快速接入
1

  • ESP32 或 ESP8266 开发板（¥15-30）
  • 温湿度传感器：DHT22（¥15）或 SHT30（¥25）
  • 杜邦线若干、USB 数据线

3
烧录 Arduino 代码

使用 Arduino IDE 上传以下代码（需安装 PubSubClient 和 DHT 库）：

#include <WiFi.h>
#include <PubSubClient.h>
#include <DHT.h>

#define DHTPIN 4
#define DHTTYPE DHT22
DHT dht(DHTPIN, DHTTYPE);

const char* ssid = "你的WiFi";
const char* password = "WiFi密码";
const char* mqtt_server = "服务器IP";
const char* device_id = "A-101";

WiFiClient espClient;
PubSubClient client(espClient);

void setup() {
Serial.begin(115200);
dht.begin();
WiFi.begin(ssid, password);
while (WiFi.status() != WL_CONNECTED) delay(500);
client.setServer(mqtt_server, 1883);
}

void loop() {
if (!client.connected()) {
client.connect("esp32-" + String(device_id));
}
client.loop();
float t = dht.readTemperature();
float h = dht.readHumidity();
String payload = "{\"temperature\":" + String(t)
+ ",\"humidity\":" + String(h) + "}";
String topic = "sensor/" + String(device_id) + "/data";
client.publish(topic.c_str(), payload.c_str());
delay(10000);
}
4
系统中添加设备

在「设备管理」>「添加设备」中填入设备编号（与代码中的 device_id 一致），选择对应传感器型号（DHT22），连接方式选 WiFi，保存后即可看到实时数据。
