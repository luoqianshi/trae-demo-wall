13. 总结

经过整个系列的学习，我们了解了 SelectionDAG 的完整流程。

13.1 SelectionDAG 工作流程回顾

   |

   |

   |

   |

   |

   |
MachineIR / MachineInstr

13.2 核心概念

  • MVT/EVT：SelectionDAG 的类型系统
  • SDNode：DAG 的基本节点
  • SDValue：节点的输出
  • SDUse：节点的输入引用
  • Chain：非数据依赖，用于调度和副作用

13.3 合法化策略

  • Type Legalization：把非法类型转成合法类型（Promote/Expand/Split/Widen）
  • Operation Legalization：把非法操作转成合法操作（Legal/Promote/Expand/Custom）

13.4 解决问题的方法论

当你遇到后端问题时：

  1. 先用 -view-dag-raw-dag 看看 DAG 长什么样
  2. 检查类型是否 Legal——类型不合法会触发类型合法化
  3. 检查操作是否 Legal——操作不合法会触发操作合法化
  4. 看 DAG Combine 是否做了你想做的优化
  5. 检查 TableGen pattern 是否匹配
