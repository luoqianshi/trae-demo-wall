仿生五大脑区架构

对标人类大脑功能分区，每个脑区承担独立职责，协同构成完整记忆中枢。

🧠
海马体
记忆巩固中枢
将短期工作记忆转化为长期经验沉淀。每次 Bug 修复、设计决策都会被打包成"记忆链"持久化存储。
技术实现：向量嵌入 + 时序衰减
已沉淀：538 条记忆 · 239 条知识链
🎯
前额叶
模式匹配引擎
从海量记忆中识别跨项目重复出现的模式。当相似问题命中 N 次以上，自动提炼为"通用模式"并标记。
技术实现：语义聚类 + 频次统计
已识别：17 个跨游戏模式
⚠️
杏仁核
风险预警雷达
基于历史踩坑记录，在新任务启动时主动预警潜在风险。让"踩过的坑"变成"避坑指南"。
技术实现：规则引擎 + 关键词触发
已预警：42 次 · 命中率 89%
⚙️
小脑
自动化协调器
负责静默调度与自动检索。在 Agent 编辑代码、做决策的瞬间，后台静默触发记忆检索，命中才出声。
技术实现：Hook 钩子 + 异步检索
日均调度：120+ 次静默检索
💡
联想大脑
跨域推理网络

技术实现：知识图谱 + 图遍历
