## 文件说明

### 1. index.html（网页版 - 核心功能演示）
**可直接用浏览器打开运行，无需服务器！**

功能特点：
- 从发票PDF文件提取信息（发票号码、金额、日期、购买方/销售方）
- 高铁票、机票均可识别（车次、航班号、出发地/目的地）
- 自动汇总发票金额，多维度统计
- 导出Excel报表（发票明细、员工统计、购买方汇总）
- 重复发票检测与预警
- 100%前端处理，数据不上传云端，保护隐私

使用方法：
1. 用浏览器直接打开 `index.html`
2. 点击"上传发票PDF"按钮
3. 选择PDF发票文件（支持批量上传）
4. 系统自动提取信息并显示结果
5. 点击"导出Excel"下载汇总报表

### 2. proposal.html（参赛介绍页面）
作品展示页面，包含创意介绍、功能特性、目标用户、价值意义等内容。

## 技术栈

- **前端**：HTML5 + Tailwind CSS + Vanilla JavaScript
- **PDF解析**：PDF.js 3.x（浏览器端直接解析）
- **Excel导出**：SheetJS（前端生成Excel文件）
- **AI辅助开发**：TRAE Work / TRAE IDE

## 核心亮点

1. **AI智能识别**：自动判断发票类型（普通发票/高铁票/机票）
2. **XBRL/XML精准提取**：解析PDF内嵌的XML数据
3. **多维度查重**：发票号码+代码+日期+金额综合检测
4. **本地隐私保护**：所有处理在浏览器完成，零数据上传
5. **批量处理**：支持一次性导入整个文件夹的发票
