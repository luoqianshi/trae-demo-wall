📋 项目概述

渔迹（YuJi） 是一款面向钓鱼爱好者的一站式微信小程序，提供完整的钓鱼生态服务。

API 开发规范

⚠️ 注意事项
  1. 所有数据库调用集中在 src/db/api.ts
  2. 错误处理：统一 console.error 输出 + 返回 null / [] / false
  3. 联表查询：使用 profiles!user_id(*) 或 profiles!column_name_fkey(...) 格式
  4. 批量查询：优先使用 in() 而非循环单查
  5. 新增 API：按功能区域追加到 api.ts 对应区块，保持字母排序

🧩 核心组件

RouteGuard（路由守卫）

文件：src/components/RouteGuard.tsx

功能：页面级认证保护

规则：

  • 未登录用户访问非公开页 → 跳转登录页
  • 登录后昵称/头像不完整 → 跳转完善信息页
  • TabBar 页用 navigateTo，非 TabBar 用 redirectTo

用法：export default withRouteGuard(PageComponent)

RLS 安全策略模式

-- 读取：所有人/登录用户
CREATE POLICY "xxx_select" ON xxx FOR SELECT USING (true);

-- 写入：仅限本人
CREATE POLICY "xxx_insert" ON xxx FOR INSERT WITH CHECK (auth.uid() = user_id);

-- 更新：本人 或 admin
CREATE POLICY "xxx_update" ON xxx FOR UPDATE USING (
  auth.uid() = user_id OR is_admin_user()
);

-- 删除：本人
CREATE POLICY "xxx_delete" ON xxx FOR DELETE USING (auth.uid() = user_id);

微信登录流程

用户点击微信登录
    ↓
Taro.login() 获取 code
    ↓
Taro.getUserProfile() 获取昵称/头像（可选）
    ↓
supabase.functions.invoke('wechat_miniapp_login', { code, nickname, avatar_url })
    ↓
后端调用微信 auth.code2Session 获取 openid
    ↓
生成/查找 auth.users 记录，创建/更新 profiles
    ↓
返回 magiclink token
    ↓
前端 verifyOtp(token_hash, 'magiclink') → 完成登录

📊 数据流与状态管理

架构模式

采用 「React Context + API 层 + 本地状态」 的轻量架构：

页面组件 (useState, useEffect)
    ↕
API 层 (src/db/api.ts) —— 所有 Supabase 调用
    ↕
Supabase Client (src/client/supabase.ts)
    ↕
PostgreSQL Database

全局状态

状态 位置 说明
用户信息 AuthContext user + profile
模块列表 api.ts getAppModules() 实时获取
本地缓存 Taro.setStorageSync 登录重定向路径等

页面数据获取模式

const [data, setData] = useState<Data[]>([])
const [loading, setLoading] = useState(false)

const fetchData = async () => {
  setLoading(true)
  const result = await getData()
  setData(result)
  setLoading(false)
}

useEffect(() => { fetchData() }, [])
usePullDownRefresh(() => { fetchData(); Taro.stopPullDownRefresh() })
useDidShow(() => { fetchData() })  // 返回刷新

新增页面步骤

  1. 创建 src/pages/<page-name>/index.tsx + index.config.ts
  2. 在 src/app.config.ts 的 pages 数组中添加路径
  3. 如需 TabBar，添加到 tabBar.list
  4. 执行 npm run lint

新增数据库表步骤

  1. 创建迁移文件：supabase/migrations/000XX_xxx.sql
  2. 编写 CREATE TABLE + RLS POLICY
  3. 在 src/db/types.ts 添加 interface
  4. 在 src/db/api.ts 添加 CRUD 函数
  5. 执行 npm run lint
