# 📥 收集箱说明

## 用途

这里是所有想法、碎片信息、临时需求的暂存区。

**想到什么就先丢进来，不要让它从脑子里溜走。**

## 使用规则

1. **随时记录**：有想法立刻新建一条笔记，不用管格式
2. **每日整理**：每天花 5-10 分钟清空收集箱
3. **归类处理**：
   - 属于某个项目的 → 移到对应项目文件夹
   - 需要立项的 → 在 10_项目Projects 新建项目
   - 参考资料 → 移到 20_资源Resources/参考文档
   - 没用的 → 直接删除

## 快速记录模板

每条笔记可以这样写：

```
# [简短标题]

- 来源：哪里想到的
- 内容：具体是什么
- 下一步：要不要处理，怎么处理
```

## 为什么要有收集箱

> 大脑是用来思考的，不是用来记东西的。

把所有杂事都丢进收集箱，大脑才能专注于真正重要的事。
