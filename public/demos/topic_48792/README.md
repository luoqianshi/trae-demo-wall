# 青芽 —— 试管群体的情绪陪伴小站

> TRAE AI创造力大赛参赛作品 | 社会服务赛道

## 项目简介

青芽是一款面向试管婴儿（IVF）群体的情绪陪伴应用。在中国，每年有超过100万对夫妇接受辅助生殖治疗，她们在长达数月甚至数年的治疗周期中，承受着巨大的身体痛苦、心理压力和经济负担。青芽用AI技术和温暖的社区，为这群"最勇敢的女性"提供全程情绪陪伴。

## 核心创意：我懂你

"我懂你"不是一句口号，而是贯穿整个产品的设计理念：

- **懂你的心情** — 根据心情+阶段，给出温暖的个性化建议（非医学）
- **懂你的记录** — 不仅记录数据，更分析情绪趋势，给你鼓励
- **懂你的孤独** — 树洞社区，匿名倾诉，互相拥抱
- **懂你的坚持** — 花园种子随你的记录生长，见证每一步

## 功能亮点

### 1. 今天 — 每日温暖陪伴
- 智能问候语（根据时段、心情、阶段动态变化）
- 试管之路倒计时提醒
- 今日待办（检查、用药等）

### 2. 记录 — 我懂你
- 心情记录（8种情绪标签）
- 检查报告、用药记录、验孕记录
- **AI智能建议**：根据心情+阶段+内容关键词，给出温暖建议
- 关键词匹配：失眠、肚子疼、老公不理解、家人压力、经济困难等

### 3. 花园 — 情绪可视化
- 种子随记录生长，从萌芽到开花
- 天气反映整体心情趋势
- 养成式陪伴，让坚持有迹可循

### 4. 树洞 — 匿名互助社区
- 大树场景，倾诉卡片飘浮
- 匿名发布，互相拥抱鼓励
- 历史记录查看

### 5. 我的 — 个人时光
- 试管之路时间线可视化
- 月度心情统计 + 鼓励语
- 内容汇总（记录/树洞/花园）

## 技术栈

- 纯前端：HTML5 + CSS3 + Vanilla JavaScript
- 无后端依赖：localStorage存储用户数据
- 响应式设计：适配手机端
- AI逻辑：规则引擎 + 关键词匹配（无外部API依赖）
