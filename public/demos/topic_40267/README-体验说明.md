# TPMS结构参数探索器体验说明

## 打开方式

解压后使用最新版 Chrome 或 Edge 打开 `index.html` 即可体验。

## 注意事项

- 首次打开需要联网加载 Three.js 模块。
- 页面支持拖拽旋转、滚轮缩放、右键平移。
- 顶部工具按钮支持重置视角、自动旋转、复制当前参数链接、保存 PNG、导出 STL 和全屏浏览。
- 如果浏览器拦截本地文件下载，请允许页面下载 STL/PNG 文件。

## 推荐体验流程

1. 切换 Gyroid、Diamond、Schwarz P 三种曲面。
2. 调节孔隙率、单元密度和壁厚系数，观察结构实时变化。
3. 点击仿生骨支架、轻量化零件、散热结构三个预设。
4. 复制当前参数链接，或使用 PNG / STL 导出当前结果。
