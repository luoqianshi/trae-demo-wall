# 思板 - 轻量实时协作白板

一款面向中文用户的轻量实时协作白板，让团队在浏览器中快速开展头脑风暴、方案讨论和协作绘图。

## 功能特性

- ✅ **实时协作**：多人同时编辑，毫秒级同步
- ✅ **丰富工具**：便利贴、文字、画笔、箭头、矩形、橡皮擦
- ✅ **中文友好**：完美支持中文输入和显示
- ✅ **一键导出**：导出为 PNG 图片
- ✅ **在线人数**：实时显示房间在线人数
- ✅ **房间过期**：免费版房间有效期 1 小时

## 快速开始

### 环境要求

- Node.js >= 16.0.0

### 安装运行

```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 访问地址
http://localhost:3000
```

### 协作测试

1. 打开第一个浏览器窗口：`http://localhost:3000` → 进入白板
2. 打开第二个浏览器窗口：`http://localhost:3000` → 输入相同房间号进入
3. 两个人同时操作，即可看到实时同步效果

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML5 + JavaScript + CSS3 | 纯原生实现 |
| 实时通信 | WebSocket | 低延迟实时传输 |
| 后端 | Node.js + Express | 轻量服务器 |
