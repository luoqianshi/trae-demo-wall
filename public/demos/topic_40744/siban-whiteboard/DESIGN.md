# 思板 - 轻量实时协作白板 设计说明

## 一、产品概述

### 1.1 产品名称
思板（SiBan）

### 1.2 产品定位
面向中文用户的轻量实时协作白板，让团队在浏览器中快速开展头脑风暴、方案讨论和协作绘图。

### 1.3 核心价值
- **轻量易用**：打开浏览器即可使用，无需安装
- **实时同步**：多人同时编辑，毫秒级同步
- **中文友好**：完美支持中文输入和显示
- **数据安全**：数据存储在国内服务器，符合企业合规要求

### 1.4 目标用户
- 远程/混合办公的产品经理、设计师、研发团队
- 在线课程中引导学生互动的教师群体
- 需要快速协作的中小团队

### 1.5 应用场景
1. **产品迭代复盘会**：团队成员把想法快速可视化到画布上
2. **设计评审**：设计师和需求方在同一图上标注反馈
3. **线上课堂**：老师布置小组讨论任务，学生同步填写共享画布
4. **远程头脑风暴**：团队成员共同发散思维，收集创意

## 二、功能特性

### 2.1 核心工具
| 工具 | 功能说明 | 使用场景 |
|------|----------|----------|
| **选择** | 选中画布上的元素，查看选中状态 | 查看、编辑已有的元素 |

| **文字** | 添加文本，支持双击编辑 | 标注说明、标题 |
| **画笔** | 自由绘画 | 手绘草图、标注 |
| **箭头** | 绘制带箭头的连接线 | 展示流程、关系 |
| **矩形** | 绘制矩形框 | 框选区域、流程图元素 |
| **橡皮擦** | 删除元素 | 移除不需要的内容 |

### 2.2 辅助功能
| 功能 | 说明 |
|------|------|
| **放大/缩小** | 通过按钮或滚轮调整画布缩放比例（0.25x - 3x） |
| **重置视图** | 恢复画布到默认缩放和位置 |
| **导出图片** | 将画布导出为 PNG 图片保存 |

### 2.3 协作功能
| 功能 | 说明 |
|------|------|
| **房间管理** | 创建/加入房间，房间号唯一标识 |
| **实时同步** | 所有操作实时同步到同房间所有用户 |
| **在线人数** | 实时显示当前房间在线人数 |
| **房间过期** | 免费版房间有效期 1 小时 |

## 三、技术架构

### 3.1 架构模式
采用 B/S（浏览器/服务器）架构，前端纯静态页面，后端提供 WebSocket 实时通信服务。

### 3.2 技术栈
| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | HTML5 + JavaScript + CSS3 | 纯原生实现，无需构建工具 |
| **实时通信** | WebSocket | 浏览器原生支持，低延迟实时传输 |
| **后端** | Node.js + Express | 轻量服务器，处理 HTTP 和 WebSocket |
| **样式** | Tailwind CSS | 快速构建美观的 UI |
| **图标** | SVG | 内联 SVG 图标，无需外部依赖 |

### 3.3 数据同步流程
```
用户A操作 → WebSocket发送 → 服务器接收 → 广播给同房间其他用户 → 更新画布
```

### 3.4 房间管理机制
- **房间创建**：用户输入房间号或自动生成随机房间号
- **房间存储**：服务器内存存储，包含元素列表、客户端集合、过期时间
- **自动清理**：房间过期后数据清空，无活跃客户端 5 分钟后自动删除房间

## 五、使用方法

### 5.1 快速开始

#### 第一步：进入应用
在浏览器中打开思板应用：
```
http://localhost:3000
```

#### 第二步：创建或加入房间
- **创建新房间**：点击「新建」按钮，系统自动生成房间号
- **加入现有房间**：输入房间号，点击「进入白板」

#### 第三步：开始协作
- 在工具栏选择工具
- 在画布上添加元素、绘画、编辑
- 同房间的其他用户会实时看到你的操作

### 5.2 工具使用指南

#### 便利贴
1. 点击工具栏中的「便利贴」工具
2. 在画布上点击任意位置添加便利贴
3. 双击便利贴可以编辑内容
4. 点击空白处或按 Ctrl+Enter 保存

#### 文字
1. 点击工具栏中的「文字」工具
2. 在画布上点击位置添加文字
3. 双击文字可以编辑内容

#### 画笔
1. 点击工具栏中的「画笔」工具
2. 按住鼠标左键在画布上绘制
3. 松开鼠标完成绘制

#### 橡皮擦
1. 点击工具栏中的「橡皮擦」工具
2. 鼠标划过的位置会删除附近的元素

#### 拖拽画布
1. 点击工具栏中的「拖拽」工具
2. 按住鼠标左键拖动画布

### 5.3 快捷键
| 快捷键 | 功能 |
|--------|------|
| 鼠标滚轮 | 缩放画布 |
| Ctrl+Enter | 保存编辑内容 |
| Delete/Backspace | 删除选中的元素 |

## 六、开发与部署

### 6.1 项目结构
```
d:\trae/
├── server.js          # 后端服务器代码
├── public/
│   └── index.html     # 前端页面（包含所有逻辑）
├── src/               # Vue 源码（保留，可用于后续扩展）
└── DESIGN.md          # 设计说明文档
```

### 6.2 启动方式

#### 开发环境
```bash
# 安装依赖
npm install express ws

# 启动服务器
node server.js
```

#### 生产环境
```bash
# 构建静态资源（如使用构建工具）
npm run build

# 启动服务器
NODE_ENV=production node server.js
```

### 6.3 服务器配置
| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 端口 | 3000 | HTTP 和 WebSocket 服务端口 |
| 房间有效期 | 60 分钟 | 免费版房间过期时间 |
| 自动清理间隔 | 60 秒 | 检查过期房间的时间间隔 |
| 无活跃清理延迟 | 5 分钟 | 无活跃客户端后删除房间的延迟 |

### 6.4 部署建议
- **本地演示**：直接运行 `node server.js`，局域网内设备通过 IP 访问
- **云服务器部署**：部署到阿里云、腾讯云等云服务器，配置域名和 HTTPS
- **容器化部署**：使用 Docker 容器化，便于横向扩展

## 八、附录

### 8.1 设计理念
- **极简主义**：只提供必要的功能，避免功能冗余
- **中文优先**：完美支持中文输入和显示
- **实时协作**：毫秒级同步，让协作更流畅
- **数据安全**：尊重用户数据隐私，提供私有化部署选项

### 8.2 竞品分析
| 竞品 | 优势 | 劣势 |
|------|------|------|
| Miro | 功能丰富，生态成熟 | 价格高，加载慢，中文支持差 |
| FigJam | 设计精美，协作体验好 | 价格高，网络延迟明显 |
| 腾讯文档 | 免费，国内访问快 | 功能有限，非专业白板工具 |
| 思板 | 轻量、免费、中文友好 | 功能相对基础（初期） |

### 8.3 差异化优势
1. **本地化定价**：相比 Miro/FigJam 的美元定价，更适合国内用户
2. **数据合规**：数据存储在国内服务器，符合《网络安全法》要求
3. **中文体验**：专门针对中文用户优化，输入流畅
4. **轻量高效**：打开即用，无需安装，加载速度快
