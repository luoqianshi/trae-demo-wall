GuardianAngel - Parental Control Software
==============================================

How to Run:
1. Double-click index.html to open in browser
2. Double-click "start.bat" to auto-open in browser  
3. Run "python start_server.py" to start local server

Default Password: 123456

Features:
- Parent Panel (Agreement Management, App Management, Time Quotas, Content Protection, Settings)
- Child Space (Agreements, Achievements, Rewards)
- Basic Info Page (Password Protected)
- AI Engine Keyword Filter
- App Search and Local Scan

Notes:
- Data is stored in browser memory only
- All settings reset to default on page refresh
