一个Skill的进化之旅

从单一执行到多智能体协同进化，见证AI系统如何通过自检、辩论与协作不断成长

演化历史与来源

理论来源：基于 Self-Evolving Agents 综述中的 Multi-agent Co-evolution 机制。核心洞察：单 Agent 自我反思存在视角局限，而多 Agent 的协同、竞争与辩论能触发更深层的进化。

协作进化
一个 Agent 失败，另一个教它
竞争进化
多个 Agent 比赛，赢家教输家
辩论进化
Agent 争论，暴露知识漏洞

工程转化：将理论框架转化为可执行的四角色协作架构——执行员、安全员、质检员、优化员，实现 Skill 的持续自我优化。
