# 知行板 LearnFlow - 初赛交互 Demo

## 使用方式
1. 解压 `learnflow_initial_demo.zip`。
2. 双击打开 `index.html`，即可在浏览器本地体验。
3. Demo 不依赖服务器、不依赖外部图片或字体，适合直接打包上传社区。

## 已实现核心功能
- 目标拆解器：根据学习/工作目标、截止日期、投入时间、当前基础和卡点生成阶段计划。
- 执行看板：自动生成今天、本周、复盘三类任务，并用 localStorage 保存勾选状态。
- 复盘助手：根据当日完成情况、卡点和状态生成明日调整建议。
- 投稿摘要：一键生成可复制到社区帖的 Demo 简介。

## 建议提交方式
在 TRAE 官方社区初赛专区发帖时，Demo 体验地址可选择“交互式可体验的 HTML 格式文件，使用 Zip 打包上传到社区”。直接上传本 zip 即可。

## 重要提醒
官方要求初赛作品必须基于 TRAE IDE 或 TRAE Work 完成，并提供不少于 3 张开发关键步骤截图和不少于 3 个关键任务 Session ID。请把本 Demo 的需求、代码和优化过程在 TRAE 中重新跑一遍或继续迭代，并用真实 TRAE 会话截图与 Session ID 替换帖子模板中的占位内容。
