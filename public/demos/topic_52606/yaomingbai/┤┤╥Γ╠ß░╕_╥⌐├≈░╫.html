💡 创意介绍

"药明白"是一款专为老年人设计的药品说明书AI解读助手。老年人只需用手机拍下药品说明书，AI就能自动识别文字、提取关键信息，用大字体重新呈现用法用量、禁忌、注意事项等核心内容，并支持语音播报。

🎯 想解决什么问题

中国60岁以上老年人超过2.8亿，其中绝大多数需要长期服药。但药品说明书存在三大痛点：

  • 字太小看不清：说明书字号通常只有8-9pt，老年人难以阅读
  • 内容太专业看不懂：满篇医学术语，老年人理解困难
  • 关键信息找不到：用法用量、禁忌散落在长篇大论中，难以快速定位

❤️ 为什么会想到做这个

我的奶奶今年78岁，患有高血压和糖尿病，每天需要服用多种药物。她经常因为看不清说明书而吃错药量，有一次甚至把外用药当成了口服药。我意识到，这不是个例——全国数千万老年人都面临着同样的困境。借助TRAE的AI能力，我想为爷爷奶奶们做一件真正有用的事。

💻 产品形态

一款基于Web的移动端应用（PWA），无需下载安装，扫码即用。核心功能包括：拍照OCR识别、AI智能解读、大字体展示、语音播报、药品收藏、用药提醒。

👤 目标用户及痛点

🏠 核心用户

60岁以上需要长期服药的老年人，尤其是患有慢性病、需同时服用多种药物的群体。

📍 使用场景

医院取药后查看新药说明、在家服药前确认用量、整理家庭药箱时了解药品信息。

💔 当前痛点

说明书字号小、术语多、结构乱；老年人要么看不清、要么看不懂、要么找不到关键信息。

🔥 如果没有"药明白"，用户现在会遇到什么不方便：

  • 需要戴老花镜+放大镜才能勉强看清说明书
  • 看不懂"禁忌症""不良反应"等医学术语
  • 找不到用法用量，凭感觉吃药，容易过量或不足
  • 多种药品混在一起，分不清哪个是哪个
  • 忘记按时服药，影响治疗效果

⚙️ 核心功能设计

  • 拍照识别：上传或拍摄药品说明书，OCR自动识别文字
  • AI智能解读：自动提取药品名称、用法用量、禁忌、注意事项、不良反应
  • 大字体展示：一键切换大字体模式，高对比度配色，适合老年人视力
  • 语音播报：一键朗读解读结果，语速放慢，方便听力下降的老年人
  • 药品收藏：收藏常用药品，建立个人药品库，随时查看
  • 用药提醒：设置服药时间和剂量提醒，避免漏服误服

🔄 用户使用流程

1

拍照上传

打开"药明白"，点击拍照按钮，对准药品说明书拍照或从相册选择。

2

AI识别分析

系统自动识别图片中的文字，并智能分析提取关键信息。

3

查看解读结果

大字体清晰展示药品名称、用法用量、禁忌、注意事项，一目了然。

4

语音播报 / 收藏 / 设提醒

点击语音按钮听解读，收藏药品方便下次查看，设置提醒按时服药。

🔧 技术方案

本项目完全使用TRAE IDE开发，纯前端实现，技术栈如下：

HTML5 CSS3 JavaScript Tailwind CSS Tesseract.js Web Speech API LocalStorage

🔎 OCR识别

使用Tesseract.js在浏览器端完成文字识别，无需上传图片到服务器，保护用户隐私。

🤖 AI分析

基于规则+关键词的本地智能分析引擎，从OCR文本中提取药品关键信息。

🔊 语音播报

使用Web Speech API实现浏览器原生语音合成，语速放慢至0.85倍，适合老年人收听。

🎨 界面设计

界面采用"适老化设计"原则：大字体、大按钮、高对比度、简洁布局、温暖配色。

🏠 首页

功能入口清晰，大按钮引导操作，底部导航方便切换。

📷 拍照页

四步流程引导：上传 -> 确认 -> 识别中 -> 结果展示，进度圆点指示当前步骤。

📄 结果页

分块展示药品名称、用法用量、注意事项、禁忌、不良反应，配合AI智能总结。

📖 药品库

收藏的药品列表，支持查看详情和删除管理。
