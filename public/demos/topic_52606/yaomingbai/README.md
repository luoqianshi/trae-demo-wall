# 药明白 - 老年人药品说明书AI解读助手

## 项目简介

"药明白"是一款专为老年人设计的药品说明书AI解读助手，参加 **TRAE AI创造力大赛** 的参赛作品。

## 赛道选择

- **通用赛道**：社会服务（造种新体验）
- **附加赛题**：社会公益 - 智慧助老

## 核心功能

1. **拍照识别**：上传或拍摄药品说明书照片，自动识别文字
2. **AI智能解读**：自动提取药品名称、用法用量、注意事项、禁忌、不良反应等关键信息
3. **大字体展示**：界面采用大字体、高对比度设计，方便老年人阅读
4. **语音播报**：一键语音播报解读结果，解决老年人视力不佳的问题
5. **药品收藏**：收藏的药品可随时查看，建立个人药品库
6. **用药提醒**：设置定时用药提醒，避免漏服、误服

## 技术栈

- TRAE IDE / TRAE Work
- HTML5 + CSS3 + JavaScript（纯前端）
- Tesseract.js（OCR文字识别）
- Web Speech API（语音合成）
- LocalStorage（本地数据存储）
- Tailwind CSS（UI框架）

## 项目特点

- **0门槛使用**：无需注册登录，打开即用
- **隐私安全**：所有数据存储在本地，不上传服务器
- **适老化设计**：大字体、大按钮、高对比度、简洁界面
- **离线可用**：纯前端实现，无需后端服务器
