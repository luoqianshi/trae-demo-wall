# 全家适配智能菜单管家

> 兼顾老幼营养与忌口的冰箱食材搭配工具 · TRAE AI 创造力大赛 · 生活娱乐赛道

## 简介

一款面向多代同堂家庭的智能菜单生成工具。录入家庭成员的健康档案与冰箱食材后，一键生成兼顾老人慢病忌口、儿童防呛噎与年轻人口味的全家适配菜单，并附分人群烹饪指引和智能采购清单。

## 快速开始

### 方式一：双击打开（推荐）

1. 下载 `smart-menu-proposal.zip` 并解压
2. 双击 `index.html` 即可在浏览器中打开

> 支持所有现代浏览器（Chrome / Edge / Safari / Firefox），无需安装任何环境。

### 方式二：本地服务器

```bash
cd smart-menu-proposal
python3 -m http.server 8090
# 浏览器访问 http://localhost:8090/index.html
```

## 体验流程

1. 打开页面，点击 **"立即体验"** 进入应用
2. 在"家庭成员"页，点击 **"导入示例家庭"** 一键加载三代同堂数据
3. 依次点击 **"下一步"** 完成冰箱食材 → 菜单生成 → 烹饪指引 → 采购清单
4. 在菜单页点击 **"生成今日菜单"**，再点击菜品卡片查看分人群做法
5. 点击右上角 **"适老化"** 开关体验大字模式

## 功能模块

| 模块 | 说明 |
|------|------|
| 家庭成员档案 | 录入年龄/慢病/咀嚼能力/口味/忌口，自动生成饮食标签 |
| 冰箱食材管理 | 文字/拍照/语音三种录入方式，自动追踪保质期 |
| 全龄适配菜单 | 规则引擎生成三餐菜单，每道菜标注三种适配变体 |
| 分人群烹饪指引 | 标准做法 + 老人调整（减盐/软嫩）+ 儿童调整（防呛噎）|
| 智能采购清单 | 按食材缺口生成分类清单，标注用量与预算 |
| 适老化展示模式 | 一键大字高对比度 + 语音播报 |

## 技术架构

- **纯前端零依赖**：HTML5 + CSS3 + 原生 JS（ES5 兼容）
- **无需构建工具**：双击 HTML 即可运行
- **数据持久化**：localStorage 自动保存家庭成员与冰箱数据
- **菜单引擎**：规则评分算法（硬约束过滤 + 软评分排序）
- **适老化**：CSS 变量实现无刷新模式切换
- **语音播报**：Web Speech API（speechSynthesis）

## 示例数据

Demo 内置一个三代同堂示例家庭，包含精心设计的饮食冲突：

| 成员 | 年龄 | 慢病 | 忌口 | 特殊需求 |
|------|------|------|------|---------|
| 老张（爷爷） | 65 | 高血压+糖尿病 | 海鲜 | 咀嚼弱、小食量 |
| 老李（奶奶） | 63 | 缺钙 | - | 咀嚼弱 |
| 小王（儿媳） | 30 | - | 香菜 | 偏辣 |
| 小宝（孙子） | 6 | - | 胡萝卜 | 防呛噎、偏甜 |

示例冰箱含 10 种食材（番茄、鸡蛋、瘦猪肉、豆腐、青菜、土豆、胡萝卜、小米、葱、姜）。
