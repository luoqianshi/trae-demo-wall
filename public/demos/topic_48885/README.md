# 暖心陪伴 - 智慧助老陪伴助手

## 🚀 快速体验

**方法一：直接双击打开**
- 双击 `index.html` 文件
- 用浏览器（推荐 Chrome、Edge、Safari）打开即可使用

**方法二：拖入浏览器**
- 将 `index.html` 文件直接拖入浏览器窗口

---

## 📱 功能介绍

### 💬 聊天陪伴
- 与AI暖心小助手对话聊天
- 点击快捷话题快速开始（讲故事、聊天气、讲笑话等）
- 支持语音输入（点击🎤按钮）

### 👨‍👩‍👧 家人连接
- 家人通讯录，一键联系
- 快速操作：拨打电话、视频通话、发送位置
- **🎙️ 录制家人声音**：录下家人的声音，之后用熟悉的声音播报

### 📖 读文档
- 把文字粘贴进来，AI帮您朗读
- 简化成大白话，方便理解
- 常用内容模板（养生、医保、用药说明）

### 🎨 小活动
- 每日暖心名言
- 小故事（点击🔊可以朗读）
- 记忆配对游戏（锻炼大脑）
- 附近社区活动

### ❤️ 健康管理
- 用药提醒
- 健康数据记录（血压、血糖）
- 在线问诊

---

## ⭐ 特色功能

### 🎙️ 录制家人声音
1. 进入"家人"页面
2. 在"录制家人声音"区域输入名称（如"儿子的声音"）
3. 点击"开始录制"，让家人对着设备说话
4. 录制完成后，点击"设为默认"
5. 之后所有语音播报都会用家人熟悉的声音！

### 🔊 语音播报
- 所有内容都支持语音朗读
- 声音温柔、语速适中，适合老年人
- 可以调整字体大小（小/中/大）

## 🛠️ 部署说明

这是一个纯静态HTML文件，可以部署到任何静态托管服务：

### GitHub Pages
1. 创建GitHub仓库
2. 上传 `index.html` 文件
3. 在仓库Settings → Pages中选择main分支
4. 等待几分钟即可获得公开访问链接

### Netlify / Vercel
1. 注册账户
2. 创建新项目
3. 上传 `index.html` 文件
4. 自动生成公开访问链接

### 阿里云OSS / 腾讯云COS
1. 创建存储桶
2. 上传 `index.html` 文件
3. 配置公开访问
4. 获取访问地址
