Agent 1: 访谈提取智能体 (Interviewer Agent)

使用苏格拉底提问法与老专家对话。不只记录，而是主动发现逻辑断层。 找出“现象”到“行动”之间的隐藏判断条件，如果发现逻辑跳跃，必须向用户追问，直到形成闭环的因果链。

Agent 2: 知识编译智能体 (Graph Compiler Agent)

后台数据结构化引擎。监听 Agent 1 收集到的完整上下文，将其解析为 JSON 格式的知识图谱节点和边。 输出包含 nodes 和 edges 的标准结构，用于前端动态渲染。

Agent 3: 场景匹配智能体 (Resolution Agent)

青年工作台辅助 AI。接收青年求助，将问题向量化，在数据库中检索图谱节点，并生成“现象-根因-排查步骤”的结构化指导方案。

核心壁垒： 这不是普通的对话机器人。模型会在后台验证逻辑闭环，如果发现“故障现象”与“解决手段”之间缺少关键排查步骤， 系统会主动向老专家发问补全逻辑链路，最终生成的是可被代码执行的动态决策树。
