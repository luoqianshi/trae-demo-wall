# 夸夸社区App 使用说明

## 快速启动（Mac）
双击 `start.command` 文件，会自动打开浏览器并启动本地服务。

## 手动启动
1. 打开终端（Terminal）
2. cd 到本文件夹
3. 运行命令：`python3 -m http.server 8080`
4. 浏览器打开：`http://localhost:8080/index.html`

## 停止服务
在终端按 `Ctrl + C`

## 页面列表
- index.html - 首页
- search.html - 搜索页
- mood.html - 心情首页
- mood-daily.html - 每日一语
- mood-calendar.html - 心情日历
- mood-blindbox.html - 心情盲盒
- profile.html - 我的
- edit-profile.html - 编辑资料
- followers.html - 粉丝/关注
- publish.html - 发布
- message.html - 消息
- comment.html - 夸夸详情
- comment-input.html - 写夸夸
