## 部署方式

### 方式1: 本地直接打开
直接用浏览器打开 `今晚吃啥_精简版.html` 即可使用

### 方式2: GitHub Pages部署（推荐）
1. 创建GitHub仓库
2. 上传所有文件到仓库
3. 在仓库Settings → Pages中启用GitHub Pages
4. 选择分支（main）和目录（/root）
5. 等待几分钟后访问: `https://你的用户名.github.io/仓库名/今晚吃啥_精简版.html`

### 方式3: Vercel部署
1. 登录 https://vercel.com
2. 点击 "New Project"
3. 上传整个文件夹
4. 等待部署完成后获得访问链接

### 方式4: Netlify部署
1. 登录 https://netlify.com
2. 点击 "Add new site" → "Deploy manually"
3. 拖拽整个文件夹到上传区域
4. 等待部署完成后获得访问链接

## PWA功能使用
部署后，在手机浏览器中访问：
1. 点击浏览器菜单中的"添加到主屏幕"
2. 应用将作为独立APP安装到手机
3. 支持离线使用（需首次访问时加载完成）
