# 通用用户需求驱动分析模型 (User Need Model)

本模型用于指导 AI 智能体在处理任何商品素材时，从产品属性导向升级为**用户需求导向**。其核心理念是：**商品本身不是终点，它只是解决用户在特定处境下所遭遇的痛点和情绪冲突的解决方案。**

---

## 一、 用户需求四层抽象模型

对任何产品进行洞察时，必须将其从物理属性向上抽象为以下四个层级的用户需求：

```text
┌────────────────────────────────────────────────────────┐
│             4. 身份认同需求 (Identity)                   │
│             - 这款产品代表我是谁？我的审美态度是什么？     │
├────────────────────────────────────────────────────────┤
│             3. 社交需求 (Social)                       │
│             - 在社交场景中，它带给我什么眼光或社交资本？   │
├────────────────────────────────────────────────────────┤
│             2. 情绪需求 (Emotional)                    │
│             - 它解决了我的什么焦虑、心理负担或情绪冲突？   │
├────────────────────────────────────────────────────────┤
│             1. 功能需求 (Functional)                   │
│             - 它解决了什么物理层面的不适、累赘或便利性？   │
└────────────────────────────────────────────────────────┘
```

1. **功能需求 (Functional)**：用户在日常使用中面临的实际痛点（如：保暖度不足、版型显得臃肿、材质粗糙磨损皮肤、收纳携带不便等）。
2. **情绪需求 (Emotional)**：伴随物理痛点产生的心理负荷（如：职场穿搭的不自信感、身材微瑕产生的自我怀疑、追求高品质却预算有限的焦虑感等）。
3. **社交需求 (Social)**：用户在与他人交往中所期望呈现的社会形象（如：希望在职场显得利落专业、在休闲场合显得随性优雅、在户外露营中展现品味等）。
4. **身份认同需求 (Identity)**：用户自我审美的投射和生活态度的具象化（如：崇尚简约低调的静奢美学、支持环保可持续的生活态度、推崇自由无束的穿衣哲学等）。

## 二、 变量化推导链 (Variables Derivation Chain)

AI 在分析素材并生成内容策略简报时，必须严格按照以下递进式变量链条进行逻辑推导：

```text
{目标人群} 
   ↓ (在什么时间、地点或处境下)
{用户处境} 
   ↓ (遭遇了什么冲突，导致未满足什么渴望)
{核心痛点} 与 {情绪期待}
   ↓ (商品所具备的哪项物理属性可以介入并改变处境)
{产品解决方案} 
   ↓ (如何向用户证明这套解决方案不是凭空捏造的)
{证据来源} (Evidence Refs)
```

### 1. 变量项详细定义

*   **`{目标人群}`**：该品类的典型消费者画像（结合性别、年龄段、核心风格偏好、消费水平）。
*   **`{用户处境}`**：具体的、可感知的日常场景或状态（例如：急促转场的早通勤、长时间的办公久坐、夜晚冷热交替的户外散步）。
*   **`{核心痛点}`**：在该处境下最容易产生的不适感或糟糕体验（例如：版型紧绷限制活动、劣质面料起毛刺痒、视觉上显得沉闷无精神）。
*   **`{情绪期待}`**：用户在此处境中极其渴望得到的心理宽慰或状态（例如：轻松体面、无感拘束、获得自信与专注）。
*   **`{产品解决方案}`**：针对该核心痛点，商品所提供的直接物理改善措施（例如：利用微弹材料释放腰腹压力、平滑顺滑织物减少肌肤摩擦、立体微廓形修饰腰部曲线）。
*   **`{证据来源}`**：支撑上述解决方案的客观商品事实（例如：检测报告中的高弹力数据、设计师说明中的编织工艺、实拍图中的光泽质感与版型特写）。

## 三、 用户需求识别与推导方法论

### 步骤一：文本与图像物料粗筛
1. **寻找痛点信号**：从设计师文档和素材文本中抓取负面或改善型词汇，反向推导用户可能存在的生理与心理痛点。
2. **寻找场景信号**：从外景和搭配图片目录中，识别拍摄场景与搭配氛围，定位用户穿戴该产品的黄金生存空间。
3. **寻找事实信号**：在 Excel 尺码表、PDF/图像检测报告及洗水唛中提取纤维成分、弹力参数、工艺名称等，作为最底层的事实基石。

### 步骤二：用户视角的情绪冲突翻译
禁止直接输出“该面料含有拉架，因此很有弹性”。
必须将其翻译为：**“「内心独白：一整天坐着肚子好勒，站起来裤头还会勒红印，真的太局促了。」 → 解决方案：利用高比例弹力纤维配合高密编织，提供无痕物理拉伸，让身心重获体面与舒展。”**

### 步骤三：可信度（Confidence）分级
评估 `{证据来源}` 的扎实程度：
*   **High**：来源于经官方质检报告、国家标准、Excel 尺码表或清晰洗水唛合格证中的具体纤维含量/物理力学指标。
*   **Medium**：来源于设计师卖点原文、拍摄搭配示意或图片库中可直接肉眼清晰佐证的细节。
*   **Low**：来源于无直接证据、仅能通过通用常识进行的经验性合理推断。
