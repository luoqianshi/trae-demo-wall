# 天猫与淘宝平台通用运营 Playbook (Tmall & Taobao Playbook)

天猫和淘宝是**“决策漏斗完整、对比理性、重在细节事实、强调权威资质、促成即时转化”**的搜索型电商平台。本指南用于规范天猫/淘宝主图与详情页内容资产的生成。

---

## 一、 主图 5 张黄金视觉策略 (Main Image 5-Page Strategy)

主图是激发点击与完成快速决策的第一触点。AI 必须为商品规划出结构严密的 5 张主图卖点架构：

```text
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│     第 1 张     │ │     第 2 张     │ │     第 3 张     │ │     第 4 张     │ │     第 5 张     │
│  核心视觉大片   │ │  多色及静态展示 │ │  版型与细节放大 │ │  核心事实与证书 │ │  成套场景联想   │
│  (风格吸睛首选) │ │  (展示物理选择) │ │  (解答版型疑虑) │ │  (硬核权威背书) │ │  (拓宽使用边界) │
└─────────────────┘ └─────────────────┘ └─────────────────┘ └─────────────────┘ └─────────────────┘
```

1.  **第一张：核心视觉大片 (风格吸睛)**
    *   **选图要求**：最具视觉张力的棚拍全身照（5.2）或高奢外景氛围图（5.3），模特处于黄金分割位，整体色调优雅统一。
    *   **打标文字**：字数控制在 8 字以内，直击核心痛点的终极解决方案（如“无感自舒展，体面自利落”）。
2.  **第二张：多色及静态展示 (展示选择)**
    *   **选图要求**：多颜色平铺展示图（5.1）或全套正面模特图。
    *   **打标文字**：标明颜色命名，强调多元美学选择（如“一色一格，多元美学态度”）。
3.  **第三张：版型与细节放大 (科学解答)**
    *   **选图要求**：腰腹、膝盖、肩部或领口的细节拉扯/微距特写图。
    *   **打标文字**：指出版型设计亮点，解答微瑕身形焦虑（如“微廓无束，恰到好处的修饰”）。
4.  **第四张：硬核事实与权威证书 (消除疑虑)**
    *   **选图要求**：面料平铺质感大图（8.1）、国家标准质检报告截图或环保认证等（8.2）。
    *   **打标文字**：列出纤维含量、弹力参数或硬核测试成绩（如“高密精织，硬核质检无忧”）。
5.  **第五张：成套场景联想 (场景拓宽)**
    *   **选图要求**：全套穿搭效果图（5.4）或模特在外景活动中的自然捕捉图（5.3）。
    *   **打标文字**：拓宽使用场景，勾勒精致生活场景（如“从容通勤到周末咖啡，场景自由转场”）。

## 二、 详情页五重体验叙事结构 (Detailed Page Structure)

详情页必须构建成一条**“视觉吸引 → 情绪痛点共鸣 → 物理方案拆解 → 细节背书 → 尺码指导”**的理性说服链路：

1.  **第一重：核心视觉海报**：采用宽幅高奢大图，确立整件商品的静奢/高规格美学调性。
2.  **第二重：场景痛点揭示**：以消费者的日常尴尬处境（久坐、转场、紧绷）作为切入点，引发痛点共鸣。
3.  **第三重：硬核解决方案拆解**：针对痛点，逐一拆解产品的工艺细节（如弹力纤维、立体廓形、无痕接缝），展示物理解决方案。
4.  **第四重：细节实拍与资质背书**：展示细节图片（面料、洗水唛）及质检合格表格，用无可辩驳的数据做信任背书。
5.  **第五重：尺码与试穿选码卡片**：提供详细的尺寸图表与不同身形模特的真实试穿建议，打消最终购买疑虑。
