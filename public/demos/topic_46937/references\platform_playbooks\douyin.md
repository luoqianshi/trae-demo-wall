# 抖音短视频通用营销 Playbook (Douyin Playbook)

抖音（Douyin）是极度依赖**“强黄金3秒视觉开场、强情绪冲突切入、快节奏信息轰炸、高密度硬核事实、直接转化逼单”**的高频推流算法平台。本指南用于规范抖音平台衍生短视频脚本的黄金节奏与话术。

---

## 一、 抖音黄金 38 秒视频节拍控制

抖音短视频资产包的生成必须严格遵循以下时间节拍框架，严禁拖沓冗长：

```text
  0s ─── 3s ────── 8s ────────────── 16s ────────────── 26s ─────── 31s ── 36s ── 38s
  ┌──────────┬──────────┬──────────────────┬──────────────────┬──────────┬──────────┬─────┐
  │  Hook    │  痛点    │     面料特写     │     版型效果     │  多色切  │  场景穿  │ CTA │
  │ (吸睛)   │  (共鸣)  │     (硬事实)     │     (显微镜)     │  (选择)  │  (联想)  │ (逼)│
  └──────────┴──────────┴──────────────────┴──────────────────┴──────────┴──────────┴─────┘
```

1.  **0-3s：Hook 吸睛开场**：用视觉张力、物理拉伸动作或情绪质问瞬间抓住眼球。
2.  **3-8s：痛点精准切入**：展示糟糕穿搭的尴尬瞬间，用真实痛点引发心理共鸣。
3.  **8-16s：面料细节放大**：特写面料纹理、回弹拉伸或质检结果，输出硬核科学事实。
4.  **16-26s：版型上身上镜**：展示模特上身细节（如腰部、裤腿、领口），强调拉长与修饰效果。
5.  **26-31s：多色系拼贴快切**：快速滑过不同颜色，展示多样选择性。
6.  **31-36s：生活场景搭配切**：搭配不同单品展示 2-3 套成套场景（如通勤、咖啡、散步）。
7.  **36-38s：直接转化 CTA**：给出购买路径、库存告急或尺码引导，完成商业闭环。

## 二、 黄金 3 秒视频钩子库 (Hook Library)

在生成脚本时，AI 必须根据物料特征，从以下通用 Hook 库中选择并适配生成黄金开场语：

*   **视觉拉扯型 (针对高弹或韧性材料)**：
    *   > “【镜头大特写，主播用力拉伸面料并弹回】别眨眼！你看这惊人的回弹力。如果你穿普通{品类}稍微下蹲就勒腰，赶紧看过来！”
*   **痛点揭短型 (针对版型修饰类品类)**：
    *   > “【模特侧身展示微瑕身形，再快速套上本款】还在为{核心痛点}头疼？答应我，把衣柜里那些让你看起来{糟糕处境}的衣服扔掉！”
*   **质感打假型 (针对优质面料/工艺类)**：
    *   > “【微距镜头展示织物表面与质检报告数据】别再花冤枉钱买大牌了！睁大眼睛看这个高密编织纹理，真正的奢华品质不需要溢价。”

## 三、 多镜头转场策略 (Transitional Strategies)

*   **切屏转场**：用“手遮镜头快切”或“跳跃换装”，在 1 秒内将场景从影棚自然切换到真实生活场景。
*   **微距拉近**：从模特的全身动作瞬间拉近至面料纱线的微距特写（与 8.1 面料大图完美映射）。
*   **动作配合**：视频画面必须伴随高密度的音效或重低音拍点，镜头在 1.5-2 秒内必须发生视角切换。

## 四、 商业购买转化口径 (CTA Templates)

*   **痛点逼单式**：
    *   > “再过5秒这个专属链接就失效了，如果你也讨厌{核心痛点}的局促，马上下单体验极致的{情绪期待}！”
*   **库存紧迫式**：
    *   > “【主播贴近屏幕】这批高定制{材质}由于工艺复杂全国仅限{变量}件，点击下方小黄车，尺码拍一个少一个！”
