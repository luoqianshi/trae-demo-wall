# 齐步走APP - HTML原型

这是一个移动端友好的HTML原型，展示"齐步走"徒步APP的主要界面和交互功能。

## 📱 功能特点

### 1. 路线库页面
- 精选路线列表展示（5条模拟路线）
- 按难度分级筛选（亲子级/入门级/进阶级）
- 亲子友好路线专门筛选
- 路线搜索功能
- 路线卡片展示（距离、爬升、用时、特色标签）

### 2. 路线详情页
- 路线详细信息展示
- 距离、爬升、用时等统计数据
- 路线特色标签
- 起终点信息
- 设施信息（厕所、补给点）
- 离线数据下载/删除功能
- 开始徒步按钮

### 3. 导航页面
- 实时GPS位置追踪（模拟）
- 路线状态指示（正常/偏离）
- 实时统计显示：
  - 已行进距离
  - 剩余距离
  - 用时计时器
  - 累计爬升
- 暂停/继续追踪
- 结束徒步功能
- 到达终点自动提示

### 4. 成就系统页面
三个标签页切换：

#### 概览标签
- 用户统计数据（完成路线、总里程、总爬升、连续天数）
- 成就进度展示
- 最近徒步记录

#### 成就标签
- 已解锁成就展示（带勋章）
- 待解锁成就展示（带进度条）

#### 记录标签
- 所有徒步记录详情
- 完成状态标识

## 🚀 使用方法

### 直接打开
1. 在浏览器中打开 `index.html` 文件
2. 即可体验完整的APP原型

### 本地服务器（推荐）
```bash
# 使用Python
python -m http.server 8080

# 或使用Node.js
npx http-server

# 或使用VS Code的Live Server插件
```

然后在浏览器中访问：
```
http://localhost:8080
```

## 🎯 核心交互

### 路线浏览
- 点击路线卡片查看详情
- 使用顶部筛选按钮过滤路线
- 使用搜索框搜索路线

### 路线详情
- 点击"下载离线数据"按钮切换下载状态
- 点击"开始徒步"进入导航模式

### 导航追踪
- 点击"暂停追踪"暂停计时和数据更新
- 点击"结束徒步"结束本次徒步
- 自动检测到达终点

### 成就系统
- 切换三个标签页查看不同内容
- 实时更新统计数据
- 进度条展示成就完成度

## 📱 响应式设计

原型针对375px宽度设计，完美适配：
- iPhone 6/7/8
- iPhone X/XS/11 Pro
- iPhone 12/13 mini

在桌面浏览器中查看时，会居中显示手机轮廓。

## ⚠️ 注意事项

1. **GPS功能**: 原型中的GPS位置是模拟数据，实际使用时需要集成真实地图SDK
2. **数据存储**: 所有数据存储在内存中，刷新页面会重置
3. **离线功能**: 下载按钮只是模拟，实际需要文件系统API支持
4. **计时器**: 导航页面的计时器和距离是模拟数据，实际需要GPS计算

## 🔧 自定义

### 修改路线数据
在 `script.js` 中修改 `routes` 数组：

```javascript
const routes = [
    {
        id: 'route_001',
        name: '新路线名称',
        description: '路线描述',
        difficulty: 'beginner',
        // ... 其他字段
    }
];
```

### 修改成就数据
在 `script.js` 中修改 `achievements` 数组：

```javascript
const achievements = [
    {
        id: 'achievement_001',
        name: '新成就名称',
        description: '成就描述',
        icon: '🏆',
        progress: 0,
        maxProgress: 10,
        isUnlocked: false,
        reward: '成就奖励'
    }
];
```

### 修改样式
在 `styles.css` 中修改CSS变量：

```css
:root {
    --primary-color: #4CAF50;
    --warning-color: #FF9800;
    --danger-color: #F44336;
    /* ... 其他变量 */
}
```
