# Social Decoder - HTML Demo

## 项目介绍

AI社恐翻译器（Social Decoder）是一款基于多 Agent 的社交对话分析系统，帮助用户读懂社交潜台词。

本 Demo 为**静态演示版本**，用于比赛展示。

## 如何运行

双击 `index.html` 即可在浏览器中打开，无需任何依赖。

> 提示：推荐使用 Chrome / Edge 浏览器。

## Demo 包含的案例

| 案例 | 场景 | 模式 |
|------|------|------|
| 👔 职场 | 上下级沟通分析 | 单句 |
| 🎯 面试 | 面试官潜台词识别 | 单句 |
| 🤝 客户 | 商务沟通分析 | 单句 |
| 💬 多轮聊天 | 完整对话分析 | 多轮 |

## 功能演示

- 🎯 **场景识别** - 自动识别当前沟通场景
- 😊 **情绪分析** - 分析说话者真实情绪
- 🧠 **潜台词推理** - 推理隐藏的真实意图
- 🛡️ **风险评估** - 评估沟通风险等级
- 💬 **回复生成** - 生成多种回复策略
- ✨ **最终报告** - 汇总完整分析结果

## 技术特点

- **Multi-Agent Workflow** - 6 个 Agent 协同分析
- **Streaming 展示** - 实时流式展示分析过程
- **Evidence Chain** - 完整的推理链展示

## 与正式项目的关系

本 Demo 是正式项目的**静态展示版本**：

| 特性 | 正式项目 | 本 Demo |
|------|---------|--------|
| 前端 | React | 纯 HTML + JS |
| 后端 | FastAPI | 无（纯静态） |
| AI 框架 | LangGraph | 无（Mock 数据） |
| LLM | DeepSeek API | 无（预置数据） |
| 实时分析 | ✅ 真实调用 | ❌ 模拟演示 |

正式项目源码位于项目根目录的 `frontend/` 和 `backend/` 目录。
