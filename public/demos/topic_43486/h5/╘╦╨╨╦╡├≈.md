# 茶愈应用运行说明

## 一、访问方式

### 本地服务器运行

#### 使用 Node.js
```bash
# 进入 h5 目录
cd h5

# 安装 serve（如未安装）
npm install -g serve

# 启动服务器
serve -s

# 浏览器访问 http://localhost:3000
```

#### 使用 Python
```bash
# 进入 h5 目录
cd h5

# Python 3
python -m http.server 8080

# 浏览器访问 http://localhost:8080
```

#### 使用 VS Code Live Server
1. 在 VS Code 中安装 Live Server 扩展
2. 右键点击 `index.html` → "Open with Live Server"

## 二、操作说明

### 应用入口
打开浏览器访问后，首先进入**引导页**，展示应用简介和免责声明。

### 主要功能流程

#### 1. 拍舌苔辨证
- 首页点击「拍舌苔辨证」入口
- 进入拍摄页面，查看拍摄引导说明
- 点击取景框或底部「开始拍摄」按钮
- 选择拍摄舌部照片或从相册选取
- 系统自动进行舌诊辨证分析
- 显示辨证结果（体质、证候、调理建议）

#### 2. 症状自测
- 首页点击「症状自查」入口
- 选择当前身体不适症状（可多选，建议1-3项）
- 为每个症状设置强度（轻度/中度/重度）
- 点击底部「获取推荐」按钮

#### 3. 茶饮推荐
- 根据舌诊辨证 + 症状自测结果，智能推荐茶饮
- 按品类分类展示：传统养生茶、外卖花茶、健康奶茶
- 每个茶饮显示匹配度、推荐理由、糖含量等信息
- 点击茶饮卡片查看详情

#### 4. 茶饮详情
- 展示茶饮完整信息：功效、禁忌、冲泡指南
- 提供购买入口：京东、天猫、美团、饿了么等
- 显示禁忌提醒（如孕期、哺乳期慎用等）

### 其他页面

| 页面 | 说明 |
|------|------|
| 历史记录 | 查看过往辨证和推荐记录 |
| 个人中心 | 用户偏好设置、特殊状态标记 |
| 设置 | 应用配置、免责声明查看 |

## 三、注意事项

1. **DEMO版本**：当前为演示版本，舌诊辨证使用模拟数据，不调用真实AI服务
2. **免责声明**：应用推荐仅供参考，不能替代专业医疗诊断
3. **浏览器兼容**：推荐使用 Chrome、Edge、Firefox 等现代浏览器
4. **移动端适配**：页面已适配移动端屏幕，可在手机浏览器中访问
