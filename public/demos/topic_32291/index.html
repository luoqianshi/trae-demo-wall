自闭症双系统双向沟通翻译工具 Dual-System Bidirectional Communication · Web Demo

核心范式：双系统不融合 · 低层感知协议 · 双向翻译

本 Demo 完全依据论文《Dual-System Communication Model: A New Bidirectional Paradigm for Autism Spectrum Disorder Based on a Shared Low-Level Perceptual Protocol》构建：将自闭症心智（系统 A）与神经典型心智（系统 B）视为两套独立、平等但底层逻辑不同的信息处理系统，不要求任意一方改造，而是借助 颜色 + 基础几何图形 + 纯器乐音乐 三类跨人群共享的低层感知信号，作为中立的中间通用协议（physical-layer consensus），完成 A↔B 双向情绪—意图翻译。

演示链路

  • 儿童端：点击 红 + 三角 + 快鼓 → 家长端弹出「濒临崩溃」预警 + 干预建议。
  • 家长端：点击「等待 5 分钟」 → 儿童端全屏变蓝 + 沙漏图形 + 舒缓钢琴音乐。
  • 所有交互写入情绪日志，可一键导出 CSV，用于家庭 / 学校 / 临床复盘。

技术与价值

  • 无需昂贵脑机接口或可穿戴硬件，普通手机 / 浏览器即可运行。
  • 规则驱动确定性映射（无 AI 黑箱），完全可解释、可审计。
  • 无胁迫、无惩罚、无错点击，降低自闭症儿童感官过载与情绪崩溃风险。
  • 可扩展至 ADHD、失语症、ICU 患者、老年认知障碍等沟通受限人群。

点击右上角切换至「儿童端」或「家长 / 教师端」，亦可在两个标签页 / 两个浏览器窗口同时打开本页面，模拟双端实时互译。本 Demo 将所有数据包广播至 BroadcastChannel("dual_system_demo") 实现端到端通信，无需服务器。
