# 心声树洞 —— 产品需求文档（PRD）

## 1. 产品概述

心声树洞 是一款面向 14–28 岁青年群体的「匿名情绪树洞 + AI 情感陪伴 + 轻量心理自助」Web 应用。用户可以在其中匿名倾诉、与 AI 情感陪伴师对话、使用呼吸练习/白噪音/过来人故事等情绪急救包功能，缓解情绪内耗。

- 核心价值：给"想说又不敢说"的年轻人一个安全、匿名、随时在线的情绪出口
- 社会价值：以 AI 作为心理服务的普惠桥梁，弥补学校/家庭心理支持的不足

## 2. 核心功能

### 2.1 功能模块

1. 登录/首页：欢迎语、今日情绪打卡入口、快捷进入各功能模块
2. 心声树洞：匿名发布心事、浏览他人树洞、表情回应、标签筛选
3. AI 情感陪伴：与 AI 心理陪伴师对话，温柔回应、情绪识别、建议卡
4. 情绪急救包：呼吸练习动画、白噪音播放、过来人故事卡、心理小贴士
5. 我的：个人主页、情绪日记、每周情绪总结、成就徽章、树洞收藏

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 登录页 | 匿名登录 | 输入昵称/随机生成 ID，无真实身份绑定 |
| 首页 | 情绪打卡 | 选择当日心情（5 档），配引导语 |
| 首页 | 快捷卡片 | 一键进入树洞 / AI / 急救包 |
| 心声树洞 | 发布树洞 | 匿名发布心事，支持 5 种情绪标签 |
| 心声树洞 | 信息流 | 卡片展示，支持点赞抱抱 |
| 心声树洞 | 标签筛选 | 按情绪标签过滤 |
| AI 陪伴 | 对话窗口 | 对话气泡，支持情绪追问 |
| AI 陪伴 | 推荐练习 | AI 推荐的情绪调节练习卡 |
| 情绪急救包 | 呼吸引导 | 4-7-8 呼吸节奏动画指引 |
| 情绪急救包 | 白噪音 | 下雨/海浪/森林 三档音效 |
| 情绪急救包 | 故事卡 | 过来人分享的治愈故事 |
| 我的 | 情绪日记 | 每日心情与 AI 对话摘要 |
| 我的 | 周报 | 情绪分布饼图 + AI 文字总结 |
| 我的 | 成就徽章 | 解锁徽章 |

## 3. 核心流程

用户打开应用 -> 匿名登录 -> 情绪打卡 -> 按心情推荐进入树洞/AI/急救包 -> 每日情绪记录 -> 周末生成情绪周报。

## 4. UI 设计

### 4.1 设计风格

- 美学方向：深夜电台风 + 治愈插画风
- 配色：深夜蓝紫渐变 #1a1b4b -> #3b185f，珊瑚粉 #ff6b9d，薄荷绿 #6ee7b7
- 卡片：玻璃态 Glassmorphism
- 字体：中文圆体，英文 Quicksand
- 按钮：大圆角 24px，柔和阴影
- 动画：呼吸脉动、卡片漂浮、星星闪烁

### 4.2 响应式

- 桌面端：三列布局
- 平板：两列布局
- 移动端：单列 + 底部 Tab Bar

## 5. 技术选型

- 前端：Vue 3 + TypeScript + Vite + TailwindCSS 3
- 路由：Vue Router 4
- 状态管理：Zustand
- 数据持久化：localStorage
- 图标：Lucide Vue
- 图表：Chart.js
- 白噪音：Web Audio API
