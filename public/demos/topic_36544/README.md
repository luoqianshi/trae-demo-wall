# 忆光 MemoryLight 初赛 Demo

## 作品简介

忆光 MemoryLight 是一个面向 60+ 长辈和异地子女的 AI 记忆伴侣 Demo。它把“老照片”作为入口，让 AI 主动提出三个问题，引导长辈讲出照片背后的时间、人物和故事，再自动整理成一页可以给家人查看、打印或保存的“家族记忆纪念页”。

## 如何体验

1. 解压 `memorylight-demo.zip`
2. 双击打开 `index.html`
3. 选择顶部任一试用案例，或上传本地照片
4. 查看照片线索和 3 个 AI 追问
5. 修改回答后点击“生成记忆家书”
6. 查看最终生成的家族记忆纪念页
7. 可复制家书、导出文本，或打印/保存 PDF

## 隐私说明

本 Demo 不会把照片上传到任何服务器。用户上传的照片只在本地浏览器中预览，刷新页面后即清空。正式版会继续强化“默认私密、授权可见、一键删除、分享前确认”的原则。
