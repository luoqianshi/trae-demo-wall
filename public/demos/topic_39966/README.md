# 非遗共创展台 Demo

TRAE AI 创造力大赛参赛 Demo。目标是在一个前端原型里展示：输入非遗项目和目标观众后，AI 帮传承人、志愿者或校园社团生成可讲解、可互动、可传播的微展内容。

## 一句话

把一个非遗项目变成可体验、可教学、可传播的互动微展。

## Demo 范围

- 内置「蓝印花布」「皮影戏」「漆扇」三个样例，打开即可切换展示。
- 左侧是创作输入：项目名称、地域、材料、步骤、观众。
- 中间是生成结果：三分钟讲解词、儿童版解释、互动问答、文创灵感、短视频脚本。
- 右侧是微展预览：故事卡、工艺步骤、小测验、传播海报文案。
- 支持复制单张内容卡、复制全部内容、复制展卡文本。
- 支持演示模式，适合录制参赛视频时逐项高亮内容包。
- 当前版本使用本地模板模拟 AI 生成，便于参赛视频和现场演示稳定运行。

## 运行方式

直接打开 `index.html`，或用任意静态服务器打开本目录。
