v1.0

认证方式

API 支持可选的 Bearer Token 认证。Token 通过环境变量 AGENT_ACCESS_TOKEN 设置。

请求时可通过以下两种方式传递：

Header 方式： Authorization: Bearer <token>

Query 参数方式： ?token=<token>

🤖Agent 管理

查询系统中可用的 AI Agent 列表及其状态信息。

GET /api/agents

描述

获取所有可用 Agent 的列表，包括 Claude、Codex、Mock 等。

请求参数

无

响应示例 (200 OK)

{
  "agents": [
    {
      "name": "claude",
      "displayName": "Claude",
      "description": "Anthropic Claude AI 助手",
      "available": true,
      "version": "1.0"
    },
    {
      "name": "codex",
      "displayName": "Codex",
      "description": "OpenAI Codex 编程助手",
      "available": true,
      "version": "1.0"
    },
    {
      "name": "mock",
      "displayName": "Mock",
      "description": "模拟 Agent，用于测试",
      "available": true,
      "version": "1.0"
    }
  ]
}

💬会话 (旧版)

管理旧版聊天会话，包括创建、列表查询和详情获取。

GET /api/sessions

描述

获取所有会话列表。

请求参数

无

响应示例 (200 OK)

{
  "sessions": [
    {
      "id": "sess_abc123",
      "agent": "claude",
      "title": "代码审查会话",
      "cwd": "/home/user/projects",
      "createdAt": "2026-06-19T10:00:00Z"
    }
  ]
}
POST /api/sessions

描述

创建一个新的聊天会话。

请求参数

参数 类型 必填 说明
agent string 是 Agent 名称，如 claude、codex、mock
cwd string 是 工作目录路径
title string 否 会话标题

请求示例

{
  "agent": "claude",
  "cwd": "/home/user/projects",
  "title": "新会话"
}

响应示例 (201 Created)

{
  "id": "sess_xyz789",
  "agent": "claude",
  "title": "新会话",
  "cwd": "/home/user/projects",
  "createdAt": "2026-06-19T12:00:00Z"
}

{

  "cwd": "/home/user/projects",
  "messages": [

      "role": "assistant",

      "timestamp": "2026-06-19T10:05:30Z"
    }

🔀Git

执行 Git 相关操作，包括状态查询、提交历史和差异比较。

GET /api/git/status?cwd={path}

描述

获取指定目录的 Git 仓库状态。

查询参数

参数 类型 必填 说明
cwd string 是 Git 仓库所在目录

响应示例 (200 OK)

{
  "branch": "main",
  "isRepo": true,
  "files": [
    {
      "path": "src/main.py",
      "status": "modified",
      "staged": false
    },
    {
      "path": "README.md",
      "status": "added",
      "staged": true
    }
  ]
}
GET /api/git/log?cwd={path}&limit={n}

描述

获取 Git 提交历史记录。

查询参数

参数 类型 必填 说明
cwd string 是 Git 仓库所在目录
limit integer 否 返回的最大提交数量，默认 20

响应示例 (200 OK)

{
  "commits": [
    {
      "hash": "a1b2c3d4e5f6...",
      "shortHash": "a1b2c3d",
      "message": "修复登录 bug",
      "author": "张三 <zhangsan@example.com>",
      "date": "2026-06-18T14:30:00Z"
    },
    {
      "hash": "b2c3d4e5f6a7...",
      "shortHash": "b2c3d4e",
      "message": "添加用户认证功能",
      "author": "李四 <lisi@example.com>",
      "date": "2026-06-17T09:15:00Z"
    }
  ]
}
GET /api/git/diff?cwd={path}&target={file}

描述

获取工作区与暂存区或指定提交的差异。

查询参数

参数 类型 必填 说明
cwd string 是 Git 仓库所在目录
target string 否 目标文件路径或提交引用

响应示例 (200 OK)

{
  "diff": "diff --git a/src/main.py b/src/main.py\n...",
  "files": ["src/main.py"]
}
GET /api/git/diff/staged?cwd={path}

描述

获取暂存区与最新提交的差异（已暂存的更改）。

查询参数

参数 类型 必填 说明
cwd string 是 Git 仓库所在目录

响应示例 (200 OK)

{
  "diff": "diff --git a/README.md b/README.md\n..."
}

WS /ws/{session_id}

描述

旧版聊天 WebSocket 连接，用于发送执行命令和停止操作。

路径参数

参数 类型 必填 说明
session_id string 是 会话唯一标识

客户端发送消息

// 执行命令
{
  "type": "execute",
  "command": "请帮我优化这段代码"
}

// 停止当前操作
{
  "type": "stop"
}

服务端推送消息

// 文本输出
{
  "type": "output",
  "content": "正在分析代码..."
}

// 错误信息
{
  "type": "error",
  "message": "请求超时"
}
WS /ws/terminal/{session_id}

描述

交互式 PTY 终端 WebSocket 连接，提供完整的终端仿真体验。

路径参数

参数 类型 必填 说明
session_id string 是 终端会话唯一标识

连接初始化 (客户端发送)

{
  "type": "config",
  "agent": "claude",
  "cwd": "/home/user/projects",
  "cols": 120,
  "rows": 30
}

服务端就绪响应

{
  "type": "ready",
  "sessionId": "term_abc123",
  "agent": "claude",
  "cwd": "/home/user/projects"
}

终端输入 (客户端发送)

{
  "type": "input",
  "data": "ls -la\n"
}

终端输出 (服务端推送)

服务端以二进制帧 (bytes) 形式推送终端输出数据。

终端尺寸调整 (客户端发送)

{
  "type": "resize",
  "cols": 160,
  "rows": 40
}

📄文件 CRUD

创建、读取、更新、删除和搜索文件与目录。

POST /api/files?path={path}&is_directory={bool}

描述

创建新文件或目录。

查询参数

参数 类型 必填 说明
path string 是 要创建的文件或目录路径
is_directory boolean 否 是否创建目录，默认 false（创建文件）

响应示例 (201 Created)

{
  "success": true,
  "path": "/home/user/projects/new_file.txt",
  "type": "file"
}
POST /api/files/rename?old_path={}&new_path={}

描述

重命名或移动文件/目录。

查询参数

参数 类型 必填 说明
old_path string 是 原文件或目录路径
new_path string 是 新文件或目录路径

响应示例 (200 OK)

{
  "success": true,
  "old_path": "/home/user/projects/old_name.txt",
  "new_path": "/home/user/projects/new_name.txt"
}
PUT /api/files?path={path}

描述

更新文件内容。

查询参数

参数 类型 必填 说明
path string 是 文件路径

请求体

{
  "content": "新的文件内容..."
}

响应示例 (200 OK)

{
  "success": true,
  "path": "/home/user/projects/README.md",
  "size": 2048
}
DELETE /api/files?path={path}

描述

删除文件或目录。

查询参数

参数 类型 必填 说明
path string 是 要删除的文件或目录路径

响应示例 (200 OK)

{
  "success": true,
  "path": "/home/user/projects/temp.txt"
}
GET /api/files/search?cwd={}&query={}&max_results={}

描述

按文件名搜索文件。

查询参数

参数 类型 必填 说明
cwd string 是 搜索起始目录
query string 是 搜索关键词（支持模糊匹配）
max_results integer 否 最大返回结果数，默认 50

响应示例 (200 OK)

{
  "results": [
    {
      "name": "main.py",
      "path": "/home/user/projects/src/main.py",
      "type": "file"
    },
    {
      "name": "utils.py",
      "path": "/home/user/projects/src/utils.py",
      "type": "file"
    }
  ],
  "total": 2
}

🔀Git 操作

暂存、取消暂存、提交、切换分支以及列出所有分支。

POST /api/git/add?cwd={}&file={}

描述

将文件添加到暂存区（stage）。

查询参数

参数 类型 必填 说明
cwd string 是 Git 仓库所在目录
file string 是 要暂存的文件路径

响应示例 (200 OK)

{
  "success": true,
  "file": "src/main.py"
}
POST /api/git/reset?cwd={}&file={}

描述

将文件从暂存区移除（unstage）。

查询参数

参数 类型 必填 说明
cwd string 是 Git 仓库所在目录
file string 是 要取消暂存的文件路径

响应示例 (200 OK)

{
  "success": true,
  "file": "src/main.py"
}
POST /api/git/commit?cwd={}&message={}

描述

提交暂存区的更改。

查询参数

参数 类型 必填 说明
cwd string 是 Git 仓库所在目录
message string 是 提交信息

响应示例 (200 OK)

{
  "success": true,
  "hash": "a1b2c3d4e5f6...",
  "message": "修复登录 bug"
}
POST /api/git/checkout?cwd={}&target={}&create_branch={bool}

描述

切换分支或恢复文件。

查询参数

参数 类型 必填 说明
cwd string 是 Git 仓库所在目录
target string 是 分支名、提交哈希或文件路径
create_branch boolean 否 是否创建新分支，默认 false

响应示例 (200 OK)

{
  "success": true,
  "branch": "feature/login",
  "created": true
}

描述

列出仓库中的所有分支。

查询参数

      "name": "main",

⚙️后台任务

提交异步后台任务、查询任务状态和取消任务。

POST /api/tasks/run?agent={}&cwd={}&prompt={}

描述

提交一个后台异步任务。

查询参数

参数 类型 必填 说明
agent string 是 执行任务的 Agent 名称
cwd string 是 任务工作目录
prompt string 是 任务指令/提示词

响应示例 (202 Accepted)

{
  "taskId": "task_bg_001",
  "status": "pending",
  "agent": "claude",
  "createdAt": "2026-06-19T12:00:00Z"
}
GET /api/tasks/{task_id}

描述

获取指定后台任务的状态和结果。

路径参数

参数 类型 必填 说明
task_id string 是 任务唯一标识

响应示例 - 进行中 (200 OK)

{
  "taskId": "task_bg_001",
  "status": "running",
  "agent": "claude",
  "progress": 45,
  "startedAt": "2026-06-19T12:00:05Z"
}

响应示例 - 已完成 (200 OK)

{
  "taskId": "task_bg_001",
  "status": "completed",
  "agent": "claude",
  "result": "任务执行结果摘要...",
  "startedAt": "2026-06-19T12:00:05Z",
  "completedAt": "2026-06-19T12:02:30Z"
}
DELETE /api/tasks/{task_id}

描述

取消指定的后台任务。

路径参数

参数 类型 必填 说明
task_id string 是 任务唯一标识

响应示例 (200 OK)

{
  "success": true,
  "taskId": "task_bg_001",
  "status": "cancelled"
}

⚠️错误码

API 可能返回的 HTTP 状态码及含义。

200 请求成功
201 资源创建成功
400 请求参数错误
401 未授权，Token 无效或缺失
403 禁止访问，权限不足
404 资源不存在
500 服务器内部错误
