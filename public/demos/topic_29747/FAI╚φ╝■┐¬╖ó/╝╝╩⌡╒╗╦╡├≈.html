1. 产品设计定义

1.1 产品愿景

DIGIT FAI 是一款面向制造业的样品全尺寸报告软件（FAI 首件检验报告软件）， 旨在通过 OCR 技术自动识别工程图纸上的尺寸标注，生成标准化的首件检验报告（First Article Inspection Report）， 支持 AS9102、PPAP、ISO 13485、FDA 21 CFR Part 820 等国际标准，帮助用户节省 50-80% 的报告生成时间。

1.2 核心需求定义

文档输入
  • 可搜索文本 PDF（矢量 PDF）尺寸提取
  • 位图 PDF 尺寸提取
  • 不可搜索 PDF（纯图像 PDF）OCR 提取
  • 图像格式支持：TIF、JPEG、PNG、BMP
  • CAD 格式支持：DWG / DXF（规划中）
智能识别
  • 自动捕获尺寸值、上下偏差
  • GD&T 形位公差符号提取
  • 自动添加气泡标注（Bubble Chart）
  • 多语言 OCR 支持（中/英/德/日等）
  • 多方向旋转识别（0°/90°/180°/270°）
报告输出
  • 输出尺寸列表到 Excel / CSV 模板
  • 用户自定义 Excel 全尺寸报告模板
  • 输出气泡图 PDF / TIFF / JPG / BMP
  • 气泡颜色、大小、编号可自定义
  • 检验报告自动判定（OK / NG）
扩展能力
  • CMM 三坐标测量数据导入（即将支持）
  • 项目保存与加载（localStorage）
  • 拖放文件快速加载
  • 多页文档管理与切换
  • 画布缩放、平移、旋转

1.3 目标用户与场景

用户角色使用场景核心价值
质量检验员首件检验、过程检验、来料检验自动识别尺寸，减少手工录入
工艺工程师图纸数字化、尺寸管理、公差分析快速提取 GD&T 符号和公差信息
生产管理人员检验报告生成与归档一键生成标准化报告，节省 50-80% 时间
航空航天 / 汽车 / 医疗AS9102 / PPAP / ISO 13485 合规满足行业法规对全尺寸报告的要求

6. 架构设计优势

零后端依赖

纯前端单文件架构，无需服务器部署，数据不离开本地浏览器，保障企业图纸安全

离线可用

所有依赖库内联或本地化，断网环境下仍可完成全部 OCR 识别和报告生成

即开即用

单个 HTML 文件双击即可运行，无需安装、无需配置环境，降低部署成本

跨平台

基于浏览器运行，Windows / macOS / Linux 均可使用，无平台限制

渐进增强

从 v3.8 到 v5.5，每个版本在上一版本基础上增量改进，保持向后兼容

工业级精度

连通域检测 + 多维投票 + 边界精修，满足制造业对尺寸识别精度的严苛要求
