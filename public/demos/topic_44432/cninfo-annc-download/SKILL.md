# 巨潮资讯公告下载器

## 背景

行业分析师需要每日跟踪上市公司重要公告（融资、股权变动、高管变动等），手动查询效率低下。本技能自动化获取并筛选关键公告，生成结构化日报，提升工作效率。

## 功能概述

该技能从巨潮资讯网站获取上市公司公告，支持按公司、行业、日期范围进行筛选，并输出结构化的 Markdown 报告。

## 适用场景

### 何时使用
- 用户请求下载指定公司的公告
- 用户查询特定行业的公告（如金融业、制造业）
- 用户需要指定日期区间的公告数据
- 用户需要生成公告日报或周报

### 何时不使用
- 用户请求实时行情数据
- 用户请求投资建议或分析
- 用户需要超过 500 页的历史数据（受 API 限制）

## 触发词
- 下载公司公告
- 查询公告
- 获取公告
- 下载XX公司公告
- 查询XX行业公告
- 获取XX日期的公告

## 输入参数

### 必需参数
- **日期范围**：查询日期，格式为 YYYY-MM-DD 或 YYYY-MM-DD~YYYY-MM-DD（默认：今日+明日）

### 可选参数
- **行业**：目标行业名称（默认：金融业）
- **公司**：指定公司代码或名称（可选，默认使用内置公司清单）
- **分类**：公告类别（可选，默认使用 categories.json 配置）

### 参数说明
| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| date | string | 今日+明日 | 日期范围，支持单个日期或范围 |
| trade | string | 金融业 | 行业名称 |
| companies | path | None | 公司清单 JSON 文件路径 |
| category | string | None | 公告类别名称，支持多个用逗号分隔 |
| max_pages | int | 500 | 最大翻页数 |

### 无效输入处理
- **日期格式错误**：提示"日期格式应为 YYYY-MM-DD 或 YYYY-MM-DD~YYYY-MM-DD"
- **行业不存在**：提示"行业名称无效，请使用：金融业、制造业、信息技术服务业..."
- **文件不存在**：提示"公司清单文件不存在，请检查路径"

## 执行流程

1. 解析用户输入参数（日期范围、行业、公司、分类）
2. 调用 scripts/query_cni_annc.py 发起 HTTP 请求获取公告列表
3. 按公司清单和关键词筛选公告
4. 去重处理避免重复公告
5. 生成结构化 Markdown 报告
6. 保存至 output 目录

## 输出格式

输出为 Markdown 表格，包含以下字段：

| 字段 | 说明 |
|------|------|
| Sec Code | 证券代码 |
| Sec Name | 公司名称 |
| Announcement Title | 公告标题 |
| Announcement Date | 公告日期 |
| Download URL | 下载链接 |
| Status | 状态（include/exclude 及原因） |

### 示例输出

| Sec Code | Sec Name | Announcement Title | Announcement Date | Download URL | Status |
|----------|----------|-------------------|------------------|--------------|--------|
| 002966 | 苏州银行 | 2025年年度权益分派实施公告 | 2026-06-16 | https://static.cninfo.com.cn/finalpage/2026-06-16/... | include |
| 600030 | 中信证券 | 2026年第一季度报告 | 2026-06-15 | https://static.cninfo.com.cn/finalpage/2026-06-15/... | exclude: routine periodic report |

**Status 字段说明**：
- `include`：符合筛选规则，已包含
- `exclude + 原因`：不符合筛选规则，已排除（如：routine periodic report）

## 筛选规则

### 包含关键词（INCLUDE_KEYWORDS）
- 融资类：定增、配股、发债、债券、融资、增资、发行、回购
- 股权变动：减持、增持、质押、股权变动、股权转让
- 高管变动：高管变动、董事长、总经理、副总
- 组织架构：组织架构、营业部、子公司
- 资质获批：资质、牌照、批复

### 排除关键词（EXCLUDE_KEYWORDS）
- 定期报告类：定期报告、年报、季报、半年报
- 董监高：独立董事、监事、董事
- 风险提示：风险提示、更正公告、补充公告
- 会议材料：例行会议、临时股东大会、制度
- 评级报告：跟踪评级、信用评级
- 分红派息：付息、派息、股息

## 异常处理

### 网络超时
- 重试 3 次，间隔递增（0.5s、1s、1.5s）
- 连续 3 次失败则停止并提示用户

### 空结果
- 输出「暂无相关公告」

### 文件不存在
- 检查公司清单文件和分类配置文件是否存在

## 安全边界

### 免责声明
- 本技能提供的资讯仅供参考，不构成投资建议
- 公告内容以巨潮资讯官方发布为准
- 本技能不对信息准确性承担责任
- 使用前请核实信息来源

## 使用示例

### 示例 1：下载今日金融业指定公司列表的公告
**输入**：下载今日公告
**执行命令**：
```bash
python scripts/query_cni_annc.py --trade 金融业 --companies companies.json
```

### 示例 2：下载指定日期范围的制造业指定公司列表的公告
**输入**：下载2026-06-01到2026-06-15的制造业指定公司列表的公告
**执行命令**：
```bash
python scripts/query_cni_annc.py --date 2026-06-01~2026-06-15 --trade 制造业 --companies companies.json
```

### 示例 3：下载单个行业公告（不使用公司筛选）
**输入**：下载2026-06-15的公告，不筛选公司
**执行命令**：
```bash
python scripts/query_cni_annc.py --date 2026-06-15 --trade 金融业 --no-keyword-filter
```

### 示例 4：输出到指定文件
**输入**：下载公告并保存到指定位置
**执行命令**：
```bash
python scripts/query_cni_annc.py --date 2026-06-15 --output output/my_report.md
```
