cninfo-annc-download

巨潮资讯公告下载器 - 从巨潮资讯获取上市公司公告，支持按公司、行业、日期范围筛选并生成结构化报告

功能特性

  • 自动从巨潮资讯获取指定日期范围的上市公司公告
  • 支持按公司清单筛选目标企业
  • 智能关键词过滤，提取融资、股权变动、高管变动等重要公告
  • 生成结构化 Markdown 日报输出
  • 支持今日/指定日期/日期范围查询

快速开始

环境要求

  • Python 3.8+
  • requests 库

安装依赖

pip install -r requirements.txt

基本使用

# 查询今日公告（默认今日+明日）
python scripts/query_cni_annc.py

# 查询指定日期
python scripts/query_cni_annc.py --date 2026-06-15

# 查询日期范围
python scripts/query_cni_annc.py --date 2026-06-01~2026-06-15

# 指定公司清单筛选
python scripts/query_cni_annc.py --companies companies.json

# 跳过关键词过滤
python scripts/query_cni_annc.py --no-keyword-filter

配置说明

categories.json

定义要查询的公告分类，设置 query: "Y" 启用对应分类：

{
  "version": "1.0",
  "update_time": "2026-06-15",
  "categories": [
    {"category_name": "年报", "category_code": "category_ndbg_szsh", "query": "Y"},
    {"category_name": "增发", "category_code": "category_zf_szsh", "query": "Y"}
  ]
}

companies.json（可选）

定义监控的公司清单：

{
  "companies": [
    {"code": "600030", "name": "中信证券"},
    {"code": "601377", "name": "兴业证券"}
  ]
}

公告筛选规则

包含关键词
融资类 定增、配股、发债、债券、融资、增资、发行、回购
股权变动 减持、增持、质押、股权变动、股权转让
高管变动 高管变动、董事长、总经理、副总
组织架构 组织架构、营业部、子公司
资质获批 资质、牌照、批复
排除关键词
定期报告 定期报告、年报、季报、半年报
高管相关 独立董事、监事、董事
公告类型 风险提示、更正公告、补充公告
其他 例行会议、临时股东大会、跟踪评级、信用评级、付息、派息、股息、制度

输出格式

生成的 Markdown 文件包含以下字段：

  • Sec Code: 证券代码
  • Sec Name: 公司名称
  • Announcement Title: 公告标题
  • Announcement Date: 公告时间
  • Download URL: 下载链接
  • Status: 状态（include/exclude 原因）

命令行参数

参数 说明 默认值
-o, --output 输出文件路径 自动生成
--timeout 请求超时时间（秒） 20
--date 查询日期范围 今日+明日
--companies 公司清单 JSON 文件 无
--categories-file 分类配置文件 categories.json
--trade 行业筛选 金融业
--no-keyword-filter 跳过关键词过滤 否
--category 覆盖分类筛选 无
--max-pages 最大翻页数 500
