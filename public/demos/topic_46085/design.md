# 心情便利店 - 学生心理健康产品设计文档

**文档日期：** 2026-06-26
**产品名称：** 心情便利店 (Mental Health Space)
**目标用户：** 中学生和大学生群体

## 1. 产品概述

### 1.1 产品定位

心情便利店是一个面向中学生和大学生群体的心理健康 Web 应用，通过娱乐化的情绪测试和情绪书写工具，帮助学生了解自己的情绪状态，记录心情变化，并通过数据分析洞察情绪规律，从而实现情绪管理和压力释放。

### 1.2 核心价值

- **自我认知：** 通过快速情绪测试了解当前心理状态
- **情绪记录：** 提供便捷的心情记录工具，释放压力
- **数据洞察：** 可视化情绪趋势，发现情绪规律
- **贴心陪伴：** 定时提醒关怀，帮助养成记录习惯

### 1.3 设计理念

- **活泼彩色：** 采用渐变色卡片设计，视觉活泼，符合年轻学生审美
- **娱乐化测评：** 轻松有趣的测试体验，降低心理负担
- **隐私至上：** 所有数据本地存储，不上传服务器，保护用户隐私
- **简单易用：** 无需注册登录，打开即用，降低使用门槛

## 2. 目标用户

### 2.1 用户画像

**主要用户群体：**

- 中学生（初中、高中）
- 大学生（本科、研究生）

**用户特征：**

- 年龄：13-25 岁
- 面临多重压力：学业、人际关系、家庭、情感等
- 心理健康意识逐渐增强
- 习惯使用手机和 Web 应用
- 对隐私保护有较高要求

### 2.2 用户需求

1. **快速了解情绪状态：** 通过简单测试快速评估当前心情
2. **记录和释放情绪：** 提供便捷的心情记录工具
3. **情绪变化趋势：** 可视化查看近期情绪波动
4. **养成记录习惯：** 通过提醒功能坚持记录
5. **隐私保护：** 不希望情绪数据上传到服务器

## 3. 功能模块

### 3.1 首页（卡片式导航）

**设计：** 活泼彩色卡片式布局
**功能：**

- 情绪小测试卡片（粉红渐变色）
- 情绪日记卡片（蓝色渐变色）
- 情绪分析卡片（绿色渐变色）
- 温馨提醒卡片（橙黄渐变色）

**特点：**

- 每个卡片显示实时状态（如"今日已记录"、"下次提醒"）
- emoji 图标增加亲和力
- 点击卡片进入对应功能模块

### 3.2 情绪测试模块

#### 3.2.1 测试页面

**问题形式：**

- 单选题，10-15 题
- 每题带有情绪 emoji，娱乐化表达
- 完成时间：2-3 分钟

**功能特点：**

- 顶部显示题号（第 X/Y 题）
- 底部进度条可视化
- 选项点击立即高亮，可修改
- 支持上一题/下一题导航

#### 3.2.2 测试结果页

**结果展示：**

- 情绪等级评分（心情指数 0-100）
- 情绪状态描述（轻松、一般、需要关注等）
- 个性化建议（如：建议进行放松练习）
- 快速入口按钮（跳转到情绪日记或放松工具）

#### 3.2.3 多问卷选择

**问卷类型：**

- 日常情绪测试（每日推荐）
- 考前焦虑测试
- 情绪低落专项测试
- 人际关系压力测试

### 3.3 情绪日记模块

#### 3.3.1 记录页面

**核心功能：**

- 快速情绪选择：emoji 直观表达（😊开心、😐一般、😔低落、😰焦虑、😤愤怒）
- 情绪标签：预设标签（学业压力、人际关系、家庭问题、恋爱、健康、其他）
- 自由书写：文字记录（500 字限制）
- 多媒体支持：照片（📷）、语音（🎤）

#### 3.3.2 历史记录

**查看功能：**

- 按日期筛选
- 按情绪类型筛选
- 按标签筛选
- 支持编辑和删除记录

### 3.4 情绪分析模块

#### 3.4.1 数据可视化

**图表展示：**

- 情绪趋势图：近期情绪指数变化曲线
- 情绪分布图：各情绪类型占比可视化
- 标签统计：主要情绪来源和频率

#### 3.4.2 统计概览

**数据指标：**

- 本月记录次数
- 平均情绪指数
- 测试次数统计
- 时间筛选：按日、周、月查看

### 3.5 提醒设置模块

#### 3.5.1 提醒配置

**提醒类型：**

- 记录心情提醒
- 情绪测试提醒
- 放松练习提醒（未来功能）

**时间设置：**

- 支持多个提醒时间点
- 可选择具体时间（如 08:00、21:00）
- 开关控制各提醒启用状态

#### 3.5.2 提醒方式

**通知形式：**

- 系统通知（使用浏览器 Notification API）
- 震动提醒（可选）
- 声音提醒（可选）

**技术限制：**

- 需用户授权浏览器通知权限
- 浏览器关闭或标签页隐藏时提醒可能不触发
- 建议用户将应用加入书签，经常打开

## 5. 设计要点

### 5.1 视觉设计

**配色方案：**

- 主色：活泼渐变色（粉红、蓝色、绿色、橙黄）
- 背景色：浅灰色（#f8f9fa）
- 卡片：白色 + 渐变色边框
- 文字：深灰（#333）、中灰（#666）、浅灰（#999）

**设计原则：**

- 移动优先：卡片式布局，适合手机滑动浏览
- 清晰层级：标题 + 描述 + 状态信息，层次分明
- 即时反馈：交互元素带有视觉反馈（高亮、阴影）
- emoji 图标：增加亲和力和视觉识别度

### 5.2 交互设计

**交互原则：**

- 单击选择：选项点击立即高亮
- 流畅过渡：页面切换平滑动画
- 状态可见：实时显示进度、状态信息
- 简洁操作：减少不必要的步骤

**关键交互：**

1. **情绪选择：** emoji 点击选中/取消，视觉反馈明显
2. **标签选择：** 多选标签，点击切换状态
3. **进度条：** 实时更新，平滑动画
4. **卡片点击：** 视觉缩放反馈，进入对应模块

### 5.3 响应式设计

**布局策略：**

- 手机端（< 768px）：单栏布局，卡片全宽
- 平板端（768px - 1024px）：两栏布局
- 电脑端（> 1024px）：三栏布局，侧边导航

**适配要点：**

- 字体大小：手机端略大，提高可读性
- 间距调整：手机端增加间距，便于触摸
- 按钮大小：手机端按钮更大，便于点击

## 6. 数据存储策略

### 6.1 本地存储方案

**存储位置：** LocalStorage
**存储容量：** 约 5MB（浏览器限制）
**数据类型：**

- `emotion_records`: 情绪记录列表
- `test_results`: 测试结果列表
- `reminder_settings`: 提醒设置
- `user_preferences`: 用户偏好设置

### 6.2 数据管理

**数据操作：**

- 添加：记录新数据
- 编辑：修改已有记录
- 删除：删除记录
- 导出：导出为 JSON 或文本格式
- 清空：清空所有数据（需二次确认）

**数据备份：**

- 建议用户定期导出数据
- 提示风险：浏览器清除缓存会丢失数据
- 未来考虑：IndexedDB 以支持更大容量

### 6.3 隐私保护

**保护措施：**

- 无服务器交互：数据不上传
- 无第三方追踪：不使用 analytics
- 无账号系统：无需注册登录
- 本地加密：考虑敏感数据加密（可选）

### 7.2 技术挑战与解决方案

#### 挑战 1: 浏览器通知限制

**问题：** 浏览器关闭时提醒不触发
**解决：**

- 提示用户授权通知权限
- 建议用户将应用加入书签，经常打开
- 显示明确的提醒状态提示

#### 挑战 2: LocalStorage 容量限制

**问题：** 5MB 容量可能不足以长期存储
**解决：**

- 提供数据导出功能
- 建议用户定期备份
- 未来可升级到 IndexedDB

#### 挑战 3: 跨设备数据同步

**问题：** 本地存储无法跨设备同步
**解决：**

- 明确告知用户此限制
- 提供数据导出导入功能（未来）
- 可考虑添加可选的云端同步（未来）

#### 挑战 4: 原生 JS 实现复杂交互

**问题：** 无框架可能增加开发复杂度
**解决：**

- 模块化代码组织
- 清晰的函数职责划分
- 使用现代 JS 特性（ES6+）

## 10. 风险与应对

### 10.1 产品风险

**风险 1: 用户认知不足**

- **描述：** 学生可能不理解心理健康的重要性
- **应对：** 提供心理健康教育内容，降低使用门槛

**风险 2: 竞品冲击**

- **描述：** 已有心理健康产品市场竞争
- **应对：** 强调隐私保护和本地存储的独特优势

### 10.2 技术风险

**风险 1: 浏览器兼容性问题**

- **描述：** 不同浏览器 API 支持差异
- **应对：** 充分测试，提供降级方案

**风险 2: 数据丢失风险**

- **描述：** 用户清除缓存导致数据丢失
- **应对：** 提醒用户定期导出，明确风险提示

### 10.3 法律与伦理风险

**风险 1: 心理诊断责任**

- **描述：** 测试结果可能被误解为专业诊断
- **应对：** 明确声明仅为娱乐化测评，不替代专业诊断

**风险 2: 未成年人保护**

- **描述：** 需考虑未成年人心理健康保护
- **应对：** 内容审核，避免不当引导，必要时建议专业帮助

## 11. 总结

心情便利店是一个面向中学生和大学生群体的心理健康 Web 应用，通过娱乐化的情绪测试和便捷的情绪记录工具，帮助学生了解和管理自己的情绪状态。产品采用活泼彩色的卡片式设计，视觉吸引力强，符合年轻用户的审美偏好。

核心技术采用原生 HTML/CSS/JavaScript，数据存储使用 LocalStorage，确保用户隐私安全。产品设计注重易用性和隐私保护，无需注册登录，打开即用。

通过情绪测试、情绪日记、数据分析和定时提醒四大核心功能模块，心情便利店为用户提供完整的情绪管理解决方案，帮助他们养成记录习惯，洞察情绪规律，实现自我认知和压力释放。
