📋 启动方式

请回到项目根目录，双击「启动网页.bat」文件即可：
① 自动检查 Python 环境
② 自动检测依赖库（缺少时会提示下载安装）
③ 自动启动后端服务
④ 自动打开浏览器访问平台

💡 首次运行提示

如果启动脚本提示缺少依赖库，请按 Y 自动下载安装（需联网），
或手动在命令行执行：
pip install -r backend\requirements.txt
