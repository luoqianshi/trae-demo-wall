============================================================
一、快速开始
============================================================
1. 双击 index.html，使用 Chrome / Edge 浏览器打开。
   （所有 JavaScript 已内联到 HTML，无需联网即可运行核心功能；
    仅 Tailwind 样式通过 CDN 加载，联网时视觉更佳。）
2. 在登录页输入下方任一测试账号即可进入对应角色工作台。
3. 推荐体验路径：管理员 → 企业专员 → 服刑人员，依次走通
    档案审核 → 岗位发布 → AI 人岗匹配 → 意向对接 → 刑释后追踪
    的完整闭环。

============================================================
五、技术与合规声明
============================================================
- 技术栈：Vite 5 + 原生 JavaScript（ES Modules）+ Tailwind CSS + Chart.js
- AI 能力：接入硅基流动 SiliconFlow 平台 Qwen/Qwen2.5-7B-Instruct 模型；
          未配置 API Key 时自动回退本地规则引擎，保证 Demo 可离线运行。
- 数据说明：所有服刑人员、企业、岗位数据均为演示用虚构数据，
            不涉及任何真实在押人员信息。
- 合规声明：Demo 已实现 RBAC 三级权限（菜单/操作/数据）、
            全量操作审计日志、屏幕水印防截屏、登录页版权页脚，
            符合司法行政数据安全规范要求。

============================================================
六、如遇问题
============================================================
- 白屏：请确认使用 Chrome / Edge 浏览器打开（非 IE）。
- 样式缺失：联网后 Tailwind 样式会自动加载。
- AI 无响应：未配置 API Key 时自动回退本地规则引擎，
            返回预设的政策问答与匹配结果，不影响演示。
