# 🚗 二手车购买完全手册 — 使用说明文档

## 一、项目概述

**二手车购买手册**是一个面向购车新手的 Web 应用，帮助用户系统性了解二手车购买全流程：
从预算规划、选车、现场看车、合同签订到买车后的整备保养。

### 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 | 语义化页面结构 |
| CSS3 | 自定义样式系统（CSS 变量、动画、响应式） |
| Bootstrap 5.3 CDN | 组件基础 + 网格系统 |
| JavaScript ES6+ | 核心逻辑（箭头函数、模板字符串、Proxy、class） |
| Canvas API | 计算器饼图可视化 |
| marked.js CDN | Markdown 解析渲染 |
| localStorage | 离线数据存储 |

### 5 种设计模式

| 模式 | 实现位置 | 应用场景 |
|------|---------|---------|
| **Singleton 单例** | `js/patterns.js` — `AppState` | 全局应用状态管理，确保状态唯一 |
| **Observer 观察者** | `js/patterns.js` — `EventBus` | 跨组件通信（路由变化、收藏变化） |
| **Factory 工厂** | `js/patterns.js` — `ComponentFactory` | 统一创建面包屑、目录、卡片等 UI 组件 |
| **Module 模块** | `js/patterns.js` — `RouterModule` | 路由模块封装，IIFE 模拟模块化 |
| **Strategy 策略** | `js/patterns.js` — `CalculatorContext` | 计算器根据燃油/新能源、不同城市切换策略 |

## 三、功能详解

### 3.1 首页（`#home`）

- **Hero 区域**：展示核心数据和导航入口
- **快速导航**：4 篇指南卡片（选车/看车/签约/整备），点击跳转
- **预算分类**：按 3-5万 / 5-8万 / 8-12万 / 12-18万 分类推荐车型
- **新手必看**：6 步购车流程引导

### 3.2 指南详情页（`#guide/{id}`）

4 篇深度指南，采用 Markdown 格式存储，通过 marked.js 实时解析：

| 路由 | 内容 |
|------|------|
| `#guide/select` | 选车篇 — 预算规划、车型选择、保值率分析 |
| `#guide/inspect` | 看车篇 — 事故车识别、发动机检查、路试指南 |
| `#guide/contract` | 签约篇 — 合同要点、过户流程、常见陷阱 |
| `#guide/prepare` | 整备篇 — 全车油液、安全检查、清洁保养 |

**每篇指南包含：**
- ✅ **面包屑导航**：首页 > 当前篇目
- ✅ **左侧目录**：自动生成 H2/H3 锚点导航，滚动高亮
- ✅ **右侧相关阅读**：链接到其他 3 篇指南
- ✅ **内容标签**：明确区分 📊客观数据 和 💡经验建议
- ✅ **数据来源标注**：每篇文章底部标注数据来源和更新时间
- ✅ **收藏按钮**：一键收藏/取消，localStorage 持久化

### 3.3 工具箱 — Checklist（`#toolbox/checklist`）

**6 大检查类别，共 46 项检查项：**

| 类别 | 项数 | 内容 |
|------|------|------|
| 外观检查 | 8 | 漆面、缝隙、玻璃、轮胎、大灯、铰链、保险杠、车顶 |
| 内饰检查 | 8 | 座椅、方向盘、安全带、仪表盘、空调、音响、天窗、备胎 |
| 发动机舱检查 | 8 | 漏油、机油、冷却液、刹车油、电瓶、螺丝、皮带、异味 |
| 底盘检查 | 6 | 锈蚀、减震器、排气管、半轴、碰撞痕迹、刹车片 |
| 试驾检查 | 8 | 冷启动、怠速、跑偏、加速、刹车、底盘异响、变速箱、转向 |
| 手续检查 | 8 | 登记证书、行驶证、保险、抵押、保养记录、出险记录、违章、排放 |

**特性：**
- ✅ 逐项勾选，进度条实时更新
- ✅ 数据自动保存到 localStorage，刷新不丢失
- ✅ 全部完成/重置全部 一键操作
- ✅ 导出 JSON 格式检查结果
- ✅ 每条检查项附带检查要点提示

### 3.4 工具箱 — 计算器（`#toolbox/calculator`）

**输入参数：**
- 车辆类型（燃油/新能源）
- 成交价（裸车价）
- 排量（1.0L及以下 ~ 2.5L以上）
- 所在城市（影响上牌费）
- 保险方案（基础/标准/全面）

**计算项目：**

| 费用项 | 计算方式 |
|--------|---------|
| 车辆裸车价 | 用户输入 |
| 购置税 | 按排量阶梯计算（新能源 = 0） |
| 交强险 | 950元（新能源免） |
| 商业险 | 车价 × 3.5% × 方案系数 |
| 车船税 | 360元/年（新能源免） |
| 上牌费 | 按城市 300-500元 |
| 检测费 | 300元 |
| 整备预估费 | 车价 × 1% |

**特性：**
- ✅ Canvas 饼图展示费用构成
- ✅ 实时计算（输入即算）
- ✅ 策略模式切换燃油/新能源计算逻辑
- ✅ 费用明细列表

## 四、如何使用

### 4.1 本地运行

```bash
# 1. 进入项目目录
cd used-car-buying-guide

# 2. 使用任意 HTTP 服务器启动（因为有 CDN 外链，直接打开也可用）
# 方式一：Python
python -m http.server 8080

# 方式二：Node.js
npx serve .

# 方式三：直接双击 index.html（部分功能可能受限）
```

### 4.2 部署到服务器

本项目为纯静态 HTML，可部署到任何静态托管服务：

- **GitHub Pages**：推送仓库后在 Settings > Pages 启用
- **Vercel / Netlify**：拖拽文件夹即可部署
- **Nginx**：将文件放入 `html` 目录
- **任何 CDN**：上传文件即可

### 4.3 内容更新

修改指南内容，编辑 `js/markdown.js` 中的 `guides` 对象：

```javascript
// 在 guides 对象中添加新指南
'new-guide': {
  title: '新指南标题',
  icon: '📖',
  category: 'guide',
  tags: ['标签1', '标签2'],
  updated: '2025-07-01',
  sources: ['数据来源名称'],
  content: `
## 第一章
正文内容（支持 Markdown 格式）...
  `
}
```

然后在 `js/main.js` 中注册路由：

```javascript
RouterModule.register('guide/new-guide', () => renderGuidePage('new-guide'));
```

## 十、常见问题

**Q: 为什么选择 SPA 而非多页面？**
A: SPA 路由切换更快，用户体验更流畅；同时所有页面共享导航栏和页脚，维护成本低。

**Q: Checklist 数据存在哪里？**
A: 存在浏览器 localStorage 中，不会上传到任何服务器，隐私安全。

**Q: 计算器的税费标准准确吗？**
A: 基于现行法规计算，仅供参考。实际税费可能因地区政策、车辆具体情况有所不同。

**Q: 如何添加新的指南文章？**
A: 在 `js/markdown.js` 的 `guides` 对象中添加新条目，在 `js/main.js` 中注册路由即可。

**Q: 支持离线使用吗？**
A: Checklist 和收藏功能支持离线使用。但首次加载需要 CDN 资源（Bootstrap、marked.js），建议部署后使用 Service Worker 实现完整离线。
