# CodeNotes - 程序员代码笔记智能整理工具

> TRAE AI 创造力大赛参赛作品 - 学习工作赛道

## 项目简介

CodeNotes 是一款面向程序员的代码笔记智能整理工具，帮助开发者高效管理代码片段、构建个人知识体系。

---

## 核心功能

### 基础功能（V1.0）

- ✅ **创建笔记**：表单提交新增数据，POST 请求存入数据库
- ✅ **查询笔记**：页面加载自动 GET 获取全部数据渲染列表
- ✅ **编辑笔记**：点击编辑弹窗回显原有数据，PUT 请求更新保存
- ✅ **删除笔记**：确认删除后从数据库移除
- ✅ **搜索过滤**：支持关键词搜索、标签筛选
- ✅ **表单校验**：必填项验证、错误提示
- ✅ **操作反馈**：成功/失败 Toast 提示

### 【新增】拓展功能（V2.0）

#### 一、数据统计面板

- **扩展统计卡片**：笔记总数、标签数量、今日新增、收藏笔记
- **语言分类统计**：自动计算各编程语言笔记数量，按数量排序展示
- **实时刷新**：增删操作后自动更新统计数据

#### 二、笔记收藏模块

- **收藏字段**：每条笔记新增 `isCollect` 布尔字段
- **收藏按钮**：笔记卡片新增收藏/取消收藏按钮（⭐图标）
- **收藏筛选**：顶部筛选栏新增「仅查看收藏」切换按钮
- **收藏样式**：已收藏按钮金黄色高亮显示

#### 三、代码增强功能

- **一键复制**：代码块右上角复制按钮，复制成功 Toast 提示
- **详情弹窗**：点击「查看完整」打开全屏弹窗，展示完整代码，支持滚动
- **语法高亮**：引入 Prism.js，按 language 自动语法高亮（支持 JavaScript、Python、TypeScript、Java、Bash、SQL、CSS、HTML 等）

#### 四、主题切换

- **深色/浅色模式**：右上角主题切换按钮（🌙/☀️）
- **本地存储**：localStorage 保存主题偏好，页面加载自动应用
- **完整适配**：两套配色方案，所有组件同步切换

#### 五、导出功能

- **一键导出**：点击导出按钮，将当前筛选后的所有笔记导出为纯文本
- **导出格式**：包含标题、语言、标签、完整代码片段、更新时间
- **文件下载**：自动生成 `.txt` 文件下载

#### 六、参赛体验优化

- **Loading 状态**：所有异步操作添加 loading 提示
- **成功/失败提示**：扩展 Toast 调用时机
- **中文注释**：新增逻辑代码添加详细中文注释
- **代码结构**：注释标注新增功能模块，方便阅读理解

## 快速启动

### 步骤 1：安装依赖

```bash
cd CodeNotes
npm install
```

### 步骤 2：启动 json-server 模拟后端

```bash
npm run start
```

或者直接运行：

```bash
json-server --watch db.json --port 3000
```

启动成功后会显示：

```
JSON Server is running on port 3000
http://localhost:3000/notes
```

### 步骤 3：打开页面

在浏览器中打开 `index.html` 文件，或使用 VS Code Live Server 插件。

**推荐方式**：使用 Live Server（VS Code 插件）自动打开页面

## 接口地址说明

| 接口 | 方法 | 地址 | 说明 |
|------|------|------|------|
| 获取全部笔记 | GET | `http://localhost:3000/notes` | 返回所有笔记列表 |
| 获取单个笔记 | GET | `http://localhost:3000/notes/:id` | 按 ID 获取笔记详情 |
| 创建笔记 | POST | `http://localhost:3000/notes` | 新增笔记数据 |
| 更新笔记 | PUT | `http://localhost:3000/notes/:id` | 按 ID 更新笔记 |
| 删除笔记 | DELETE | `http://localhost:3000/notes/:id` | 按 ID 删除笔记 |
| 搜索笔记 | GET | `http://localhost:3000/notes?q=关键词` | 按关键词搜索 |
| 标签筛选 | GET | `http://localhost:3000/notes?tags_like=标签名` | 按标签筛选 |
| 【新增】收藏筛选 | GET | `http://localhost:3000/notes?isCollect=true` | 获取收藏笔记 |

## V2.1 更新详情

### 一、UI/UX 优化

#### 1. 代码块遮罩修复
- **问题**：代码块内的渐变遮罩会随内容滚动而移动
- **解决**：将遮罩从 `.note-content::after` 移到 `.note-content-wrapper::after`，设置 `z-index: 5` 固定遮罩位置
- **优化**：调整遮罩对齐方式，添加底部圆角与代码块边框匹配

#### 2. 详情弹窗滚动条修复
- **问题**：详情弹窗中代码区域的滚动条遮住了圆角边框
- **解决**：调整 `.detail-code` 和 `.detail-actions` 的圆角样式，使用 `calc(var(--radius-md) - 1px)` 匹配边框内部圆角

#### 3. 弹窗布局修复
- **问题**：新建和编辑笔记弹窗的底部按钮被遮住，只能缩小到 75% 才能完整显示
- **解决**：使用 flex 布局替代固定高度计算：
  - `.modal` 添加 `display: flex; flex-direction: column`
  - `.modal-body` 设置 `flex: 1; min-height: 0` 自动填充剩余空间
  - `.modal-footer` 设置 `flex-shrink: 0` 防止被压缩

#### 4. 弹窗宽度统一
- **优化**：新建/编辑笔记弹窗宽度从 600px 调整为 800px，与详情弹窗保持一致

### 二、侧边栏交互优化

#### 1. 侧边栏滚动行为
- **改变**：侧边栏从 `position: sticky` 改为 `position: relative`，随页面滚动而滚动

### 三、筛选功能优化

#### 1. 语言分类交互
- **新增**：语言分类项可点击进行筛选
- **样式**：添加 hover 和 active 状态样式，视觉反馈清晰
- **交互**：点击可切换选中状态，支持与其他筛选条件叠加

#### 2. 收藏筛选逻辑优化
- **问题**：之前收藏筛选与其他筛选（标签、语言）互斥
- **解决**：改为叠加式筛选，收藏筛选可以与其他筛选条件同时生效
- **示例**：
  - 选择某个语言分类 → 点击"仅查看收藏" = 该语言下的收藏笔记
  - 选择某个标签 → 点击"仅查看收藏" = 该标签下的收藏笔记
  - 搜索关键词 → 选择语言 → 查看收藏 = 符合条件的收藏笔记

### 四、代码编辑器增强

#### 1. 实时语法高亮输入
- **新增**：新建和编辑笔记时，代码输入框支持实时语法高亮
- **实现**：使用 textarea + pre/code 叠加层
  - textarea 负责接收输入（透明文字，光标可见）
  - pre/code 层负责显示语法高亮
- **语言切换**：选择不同代码语言时，高亮语法自动更新
- **自动高度**：输入内容增多时，编辑器自动扩展高度

#### 2. 代码编辑器样式
- **背景**：使用 `--code-bg` 变量，与深色/浅色主题适配
- **边框**：聚焦时显示主题色边框和光晕效果
- **错误状态**：验证失败时显示红色边框和光晕

#### 3. 编辑器状态重置
- **优化**：关闭弹窗后重置编辑器高度，下次打开恢复默认
- **修复**：避免上一次打开的高度状态影响新的编辑会话
