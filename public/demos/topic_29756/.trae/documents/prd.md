# 记住Ta - 产品需求文档

## 1. 产品概述
"记住Ta"是一款基于大模型AI的移动端社交关系管理App，帮助用户记住重要的人和重要的日子。
- 核心价值：让"在乎"从想法变成行动，通过AI智能规划帮助用户在关键时刻表达关心
- 目标用户：注重人际关系、希望更好地关心身边人的年轻用户群体

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 本地注册 | 添加好友、管理关系、AI规划生成、分享卡片 |

### 2.2 功能模块
1. **注册页**：用户信息录入、兴趣标签选择
2. **首页**：今日提醒、快捷入口、底部导航
3. **添加好友页**：好友信息录入、关系类型设置
4. **好友列表页**：关系分类、好友卡片展示
5. **好友详情页**：好友信息展示、纪念日提醒
6. **AI规划页**：天气选择、智能规划生成、分享卡片

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 注册页 | 用户信息表单 | 昵称（必填）、生日、城市、兴趣标签多选 |
| 注册页 | 提交按钮 | 数据存入localStorage，跳转首页 |
| 首页 | 欢迎语 | 显示用户昵称的个性化问候 |
| 首页 | 今日卡片 | 显示3天内即将过生日的好友提醒 |
| 首页 | 快捷入口 | 添加好友按钮 |
| 首页 | 底部导航 | 首页、好友、日子、我的四个Tab |
| 添加好友页 | 好友信息表单 | 姓名、关系类型、生日、城市、兴趣标签、特别关心开关 |
| 添加好友页 | 头像生成 | 使用DiceBear API根据姓名生成头像 |
| 好友列表页 | 分类Tab | 全部/伴侣/家人/朋友/特别关心 |
| 好友列表页 | 好友卡片 | 头像、姓名、关系标签、生日倒计时 |
| 好友详情页 | 信息展示 | 大头像、姓名、生日、城市、兴趣标签 |
| 好友详情页 | 纪念日 | 显示在一起天数 |
| 好友详情页 | 浮动按钮 | 为Ta做规划入口 |
| AI规划页 | 天气选择器 | 晴天/雨天/阴天三种天气选项 |
| AI规划页 | 规划生成 | 6张规划卡片（穿搭、礼物、餐厅、活动、路线、祝福语） |
| AI规划页 | 分享卡片 | 生成精美分享卡片并下载 |

## 3. 核心流程

用户注册 → 添加好友 → 首页提醒 → 查看好友详情 → 生成AI规划 → 分享卡片

```mermaid

    "注册页" --> "首页"
    "首页" --> "添加好友页"
    "添加好友页" --> "好友列表页"
    "首页" --> "好友列表页"
    "好友列表页" --> "好友详情页"
    "好友详情页" --> "AI规划页"
    "AI规划页" --> "分享卡片"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：温暖橙色系 #FF8C42
- 背景色：柔和米色 #FFF9F0
- 按钮风格：圆角按钮，柔和阴影，温暖治愈风
- 字体：系统字体，标题加粗，正文常规
- 布局风格：移动端优先，卡片式布局，max-width: 430px居中
- 动画：页面淡入淡出300ms，卡片依次上浮动画

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 注册页 | 表单区域 | 温暖橙色主题，圆角输入框，多选标签按钮 |
| 首页 | 今日卡片 | 白色圆角卡片，柔和阴影，生日蛋糕emoji |
| 添加好友页 | 关系类型 | 单选按钮组，带emoji图标 |
| 好友列表页 | 好友卡片 | 左侧头像，右侧信息，圆角卡片 |
| AI规划页 | 规划卡片 | 6张彩色图标卡片，依次淡入动画 |
| AI规划页 | 分享卡片 | 暖橙渐变背景，白色文字，375×667px |

### 4.3 响应式设计
- 移动端优先设计，所有元素适配375px-430px宽度
- 文字不溢出，按钮可点击区域足够大
- 卡片间距合理，触摸友好

### 4.4 动画效果
- 页面切换：淡入淡出300ms
- 卡片入场：依次上浮动画，延迟递增
- Loading动画：跳动圆点 + 思考过程文字
- 按钮交互：点击反馈，hover状态
