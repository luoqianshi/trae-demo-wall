💡 选择题
在 Python 中，以下哪个函数可以用于获取列表的长度？
A
size()
B
len()
C
length()
D
count()
🤖
AI 智能解读

在 Python 中，len() 是内置函数，用于返回对象的长度或项目数量，适用于字符串、列表、元组、字典等多种数据类型。size() 不是 Python 内置函数；length() 不存在；count() 是列表方法，用于统计某个元素出现的次数，而非获取列表总长度。

✅ 判断题
Python 中的列表（list）和元组（tuple）都是有序序列，但列表是可变的，元组是不可变的。
🤖
AI 智能解读

这个说法是正确的。列表（list）使用方括号 [] 定义，支持增删改查操作，是可变数据类型。元组（tuple）使用圆括号 () 定义，一旦创建就不能修改，是不可变数据类型。两者都支持索引访问和切片操作，且元素是有序的。

💬 问答题
请简述 Python 中列表推导式（List Comprehension）的基本语法，并给出一个示例。
🤖
AI 智能解读

参考答案：列表推导式的基本语法为 [表达式 for 变量 in 可迭代对象 if 条件]。例如：squares = [x**2 for x in range(10) if x % 2 == 0] 会生成 [0, 4, 16, 36, 64]，即 0 到 9 中偶数的平方。列表推导式比传统的 for 循环更简洁、更高效。
