# 菇管家小程序 — 产品需求文档（PRD）

## 1. 产品概述
菇管家是一款面向 50-65 岁中老年菇农的智慧种植管理小程序，以"大字、大图、语音优先、一键直达"为核心设计原则，帮助菇农一眼掌握菇棚环境、随手拍病害诊断、语音问农技问题、记录每一批次菌棒生命周期。
- **目标用户**：50-65 岁传统菇农，智能手机操作能力有限、视力下降、不擅长文字输入。
- **产品价值**：降低智慧农业使用门槛，让菇农 30 秒内学会看数据 / 拍病害 / 问问题，提升产量、减少病害损失。

## 2. 核心功能

### 2.1 信息架构
采用 **4 Tab + 6 子页面** 的扁平架构，核心功能 3 步内可达，底部 Tab 固定 4 个不扩展。

## 3. 核心流程

**看数据流程**：打开小程序 → 首页实时看板 → 一眼知全局，异常点"去处理"或"一键通风/喷雾"。

**拍病害流程**：首页/AI助手点"拍病害" → 相机拍照 → 上传识别（3秒） → 诊断结果（症状/用药/预防） → 语音播报 / 问专家。

## 4. 用户界面设计

### 4.1 设计风格
- **风格定调**：微信小程序原生质感，老人友好的"大字大图 + 状态即颜色"，克制留白、卡片化布局、扁平层级。
- **主色**：品牌绿 `#07c160`（正常/成功）；警告 `#fa9d3b`；危险 `#fa5151`；信息蓝 `#10aeff`。
- **背景**：页面 `#f5f5f5`，卡片 `#ffffff`，深底 `#f0f0f0`。
- **字体**：PingFang SC / Microsoft YaHei 系统字体栈；正文最小 15px，关键数据 48px/32px，按钮高度 ≥48px，图标点击区 44×44px。
- **布局**：移动端 375px 容器居中，桌面端自适应展示手机壳；卡片圆角 12px，间距 10px，页面边距 12px。
- **按钮**：主按钮绿底白字、警告橙、危险红，圆角 12px，48px 高；圆角胶囊用于标签与次要操作。

### 4.3 多端适配与兼容性
- **移动端优先**：375px 设计基准，手机壳容器居中模拟小程序。
- **桌面/平板自适应**：≥768px 时居中显示手机壳并展示左右说明留白，保证可读。
- **触摸优化**：所有可点击元素 ≥44px，按钮 ≥48px；支持 `prefers-reduced-motion`。
- **兼容性**：使用 React 18 + Vite + Tailwind，现代浏览器（Chrome/Safari/Edge/Firefox 最新两个版本）+ 微信内置浏览器；CSS 变量降级 + 语义化 HTML + ARIA。

### 4.4 交互规范
- 下拉刷新（旋转 Loading）、左右滑动切棚、长按语音、上拉加载更多、双击温度语音播报。
- 状态反馈：骨架屏、Toast 2s、空数据插画、网络异常横幅、操作成功 Toast。
- 动效：页面切换 300ms ease-out、Tab 切换 200ms、卡片淡入上移、数据刷新旋转。
