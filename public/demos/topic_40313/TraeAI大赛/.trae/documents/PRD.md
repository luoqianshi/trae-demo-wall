# 心理科普自助网站 - 产品需求文档

## 1. 产品概述

一款面向有情绪困扰的普通用户的公益心理科普自助网站。提供情绪认知、自测评估、放松工具和求助资源，帮助用户理解自我、获得平静。整体风格温暖柔和、低压迫感，强调共情与支持。

## 2. 核心功能

### 2.1 页面结构（单页应用，标签切换）

| 标签 | 页面名称 | 核心功能 |
|------|---------|---------|
| 首页 | 首页 | 主标题、共情副标题、4个情绪状态互动按钮、4张功能入口卡片 |
| 情绪科普 | 情绪科普 | 6种情绪困扰可展开卡片，生活化语言，去病耻感 |
| 自我觉察 | 自我觉察 | PHQ-9和GAD-7量表自测，逐题显示，柔和进度条 |
| 放松工具 | 放松工具 | 478呼吸法、5-4-3-2-1感官着陆、白噪音、温柔便签 |
| 求助资源 | 求助资源 | 紧急热线、常规渠道、自助书籍三类资源 |

### 2.2 首页模块

- **主标题**：「看见情绪，也看见你」
- **副标题**：共情引导文案，传达理解与支持
- **情绪状态按钮**：4个可点击的情绪状态（焦虑不安、情绪低落、心烦意乱、疲惫无力），点击后轻微动画反馈
- **功能入口卡片**：4张磨砂玻璃质感卡片，分别跳转至科普、自测、放松工具、求助资源

### 2.3 情绪科普模块

6种可展开卡片（默认收起，点击展开）：

1. **抑郁情绪** - 描述常见的情绪低落、兴趣减退体验
2. **焦虑状态** - 描述过度担忧、心跳加速等身心反应
3. **社交焦虑** - 描述在社交场合的紧张与不安
4. **睡眠困扰** - 描述入睡困难、睡眠浅等问题
5. **强迫倾向** - 描述反复检查、重复行为等体验
6. **倦怠耗竭** - 描述长期压力下的身心疲惫

每张卡片包含：
- 简要描述（展开前）
- 详细说明：表现、日常场景、温和的自我关怀建议
- 末尾安抚文案：「这些感受是可以被理解和接受的，你并不孤单」

### 2.4 自我觉察模块（PHQ-9 + GAD-7）

**PHQ-9（患者健康问卷）**：9道题，评估抑郁情绪
**GAD-7（广泛性焦虑障碍量表）**：7道题，评估焦虑水平

功能设计：
- 题目逐题显示，每次一道
- 柔和进度条（雾蓝色）
- 选项采用柔和的按钮样式
- 答题后显示结果页面
- 结果页设计：
  - 先显示安抚文案（如：「感谢你今天为自己抽出时间」）
  - 再显示分级结果（轻度/中度/重度）与简要解读
  - 全程标注「结果仅供参考，不作为诊断依据」
- 数据本地存储（localStorage），不上传服务器

### 2.5 放松工具模块

4个工具，标签页切换：

1. **478呼吸法**
   - 圆形光晕缩放动画（吸气4秒→屏息7秒→呼气8秒）
   - 文字引导（显示当前阶段和剩余秒数）
   - 可开关的轻提示音（柔和的提示音）
   - 开始/暂停按钮

2. **5-4-3-2-1感官着陆练习**
   - 分步引导：先看到5样东西→听到4种声音→触摸3件物品→闻到2种气味→尝到1种味道
   - 每步有文字和视觉引导
   - 可重置/返回上一步

3. **白噪音播放器**
   - 4种音源：细雨、森林、海浪、柴火
   - 极简界面，单一播放/暂停控制
   - 音量滑块
   - 循环播放

4. **温柔便签**
   - 随机显示自我关怀短句
   - 点击刷新获取新便签
   - 柔和的淡入动画

### 2.6 求助资源模块

**开头引导语**：「如果此刻你感到困难，寻求帮助是一种勇气」

三类资源：

1. **紧急热线**
   - 全国心理援助热线：400-161-9995
   - 北京心理危机研究与干预中心：010-82951332
   - 生命热线：400-821-1215

2. **常规渠道**
   - 各地精神卫生中心
   - 心理咨询平台（简单心理、壹心理等）
   - 学校/单位心理咨询室

3. **自助书籍推荐**
   - 《蛤蟆先生去看心理医生》
   - 《也许你该找个人聊聊》
   - 《当下的力量》
   - 《正念：此刻是一枝花》

## 3. 全局设计规范

### 3.1 视觉风格

- **背景色**：奶米白 `#FDF8F3`
- **主色调**：雾蓝 `#A8C0D6`
- **辅助色**：浅芋紫 `#D4C4E3`
- **圆角**：全圆角（border-radius: 20px 或更大）
- **卡片风格**：磨砂玻璃质感（backdrop-filter: blur）
- **过渡动画**：0.6秒平缓过渡，无弹跳无闪烁（ease-out 或 linear）
- **阴影**：柔和阴影，颜色偏暖

### 3.2 排版

- **字体**：系统默认圆体中文字体，确保可读性
- **标题层级**：主标题大而醒目，副标题柔和
- **正文行高**：1.8-2.0，确保阅读舒适

### 3.3 全局组件

- **顶部常驻免责声明**：「本站内容仅供参考，不能替代专业诊断和治疗」
- **右下角常驻按钮**：「回到平静」- 点击平滑滚动至首页
- **标签导航**：顶部固定，毛玻璃效果，切换时内容平滑过渡

## 4. 响应式设计

- **电脑端**：卡片网格布局，舒适留白
- **手机端**：单列布局，按钮和卡片适当放大，触摸友好
- **断点**：768px 以下切换为移动端样式

## 5. 文案语气要求

- 共情温柔，不说教
- 去病耻感，不过度医学化
- 如同朋友间的温暖对话
- 避免：「你有问题」「你应该」「必须」等表述
- 推荐：「这些感受很正常」「你可以」「也许可以试试」
