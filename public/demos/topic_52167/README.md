# 成长印记 Growth Mark · Demo 体验包

## 体验方式

### 方式一：直接打开（推荐）
双击 `index.html` 文件，用浏览器（Chrome/Edge/Firefox）打开即可体验。

### 方式二：本地服务器
```bash
# Python 3
python -m http.server 8080
# 然后浏览器访问 http://localhost:8080
```

## 功能模块

| 模块 | 功能说明 |
|------|----------|
| 🕒 时间线 | 20 条作品，按分类筛选，查看 AI 识别结果 |
| 🏆 荣誉墙 | 10 条荣誉，按级别筛选（国家/省/市/校级） |
| 🏠 家庭 | 家庭信息、邀请码、孩子档案 |
| 📊 报告 | AI 生成的季度成长报告、兴趣分析、成长建议 |
| 🔗 分享 | 分享卡片预览、分享记录 |

## 数据说明

- 所有数据为预设种子数据，无需后端服务
- 离线可用，无需网络连接
- 数据与实际项目后端 `seed.py` 保持一致

## 技术说明

- 纯前端单文件 HTML（无依赖）
- 模拟手机外框，还原移动端 App 体验
- 包含登录态切换、Tab 导航、筛选、详情弹窗等完整交互
