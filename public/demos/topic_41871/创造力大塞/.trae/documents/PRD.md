# 高中学习管理系统 - 产品需求文档

## 1. 产品概述

一款专为高中学生设计的知识掌握度管理系统，帮助学生和家长追踪高考各科目的学习进度与掌握程度。通过可视化的评级系统，让学生清晰了解自己对每个知识点的掌握情况，实现学习无死角排查。

### 目标用户
- 高中学生（高一、高二、高三）
- 学生家长（监督辅助使用）

---

## 2. 核心功能

### 2.1 选科系统
用户首次进入时进行选科操作：
- **年级选择**：高一、高二、高三
- **科目选择**：文/理科目组合
  - 必修科目：语文、数学、英语
  - 物理方向：物理、化学、生物
  - 历史方向：历史、政治、地理
- 选科后可随时修改

### 2.2 知识点展示
- 按学科展示知识点树状结构
- 每个知识点包含：
  - 知识点名称
  - 核心内容（详细说明）
  - 当前评级状态
- 知识点分组（章/单元）

### 2.3 评级系统
采用5级评级制度，用不同颜色直观显示：

| 等级 | 颜色 | 含义 |
|------|------|------|
| ⭐ 完全掌握 | 深绿色 | 能够熟练运用，举一反三 |
| ⭐⭐ 较好掌握 | 浅绿色 | 理解正确，能独立完成 |
| ⭐⭐⭐ 基本理解 | 黄色 | 有印象，需进一步巩固 |
| ⭐⭐⭐⭐ 初步了解 | 橙色 | 有印象，理解不深 |
| ⭐⭐⭐⭐⭐ 尚未学习 | 灰色 | 未开始学习 |

### 2.4 每日学习追踪
- 记录每日学习的知识点
- 按日期展示学习进度
- 统计每日新增掌握的知识点数量

---

## 3. 页面结构

### 3.1 页面列表
1. **选科页面** - 首次进入/修改选科
2. **学科总览页面** - 展示已选学科列表
3. **学科详情页面** - 展示单科学所有知识点
4. **知识点详情页面** - 展示知识点核心内容与评级

### 3.2 页面流程
```
进入系统 → 选科（如未选） → 学科总览 → 学科详情 → 知识点详情
                ↓
          修改选科
```

## 4. 科目知识点结构

### 4.1 高一科目
**语文**：现代文阅读、文言文阅读、诗歌鉴赏、写作表达
**数学**：集合与函数、三角函数、数列、向量
**英语**：词汇语法、阅读理解、写作
**物理**：运动学、力学、能量守恒
**化学**：物质的分类、氧化还原反应、离子反应
**历史**：中国古代史、近代世界史
**政治**：经济生活、政治生活
**地理**：自然地理、人文地理

### 4.2 高二科目
**语文**：散文阅读、小说阅读、语言应用
**数学**：解析几何、概率统计、导数
**英语**：完形填空、阅读技巧、书面表达
**物理**：电磁学、光学、热学
**化学**：有机化学、化学反应速率
**历史**：中国现代史、世界现代史
**政治**：文化生活、哲学生活
**地理**：区域地理、环境问题

### 4.3 高三科目
**语文**：高考专题复习、综合训练
**数学**：函数与导数、解析几何、综合应用
**英语**：高考题型专练、真题演练
**物理**：综合应用、创新实验
**化学**：综合应用、实验专题
**历史**：中外关联、热点专题
**政治**：时事政治、答题技巧
**地理**：综合分析、答题模板

## 5. 技术实现

### 5.1 技术栈
- **前端**：HTML5 + CSS3 + JavaScript（原生）
- **数据存储**：LocalStorage（本地存储）
- **无后端**：纯前端单文件应用

### 5.2 数据结构
```javascript
{
  user: {
    grade: "高一" | "高二" | "高三",
    subjects: ["语文", "数学", "英语", ...]
  },
  knowledgePoints: {
    "语文-高一": [
      { id, name, content, rating: 1-5, lastReview: date }
    ]
  }
}
```

---

## 6. 界面设计

### 6.1 设计风格
- **风格**：清新学术风，适合学生使用
- **主色调**：蓝色系（沉稳、专注）
- **辅助色**：各评级对应颜色
- **字体**：思源黑体/微软雅黑
- **布局**：卡片式布局，清晰易读

### 6.2 交互设计
- 点击学科卡片进入详情
- 点击知识点展开/收起核心内容
- 评级通过星级选择器操作
- 进度条展示整体掌握度
