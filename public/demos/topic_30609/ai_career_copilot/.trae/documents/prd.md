# AI Career Copilot - 产品需求文档

## 1. 产品概述

AI Career Copilot 是一款AI职业成长与面试辅助系统，帮助正在找工作、准备面试或职业迷茫的用户优化简历、准备面试并规划职业成长路径。

## 2. 核心功能

### 2.1 简历与岗位输入模块
- 简历输入框（多行文本）
- 目标岗位输入框
- "开始分析"按钮

### 2.2 AI职业分析模块
- 岗位匹配度评分（50-95分）
- 优势分析（3-6条，80%固定+20%动态）
- 短板分析（3-5条，动态补充未命中技能）
- 支持4大岗位动态生成（前端/后端/产品经理/运营经理）

### 2.3 AI面试模拟模块
- 每个岗位10题题库，按年限抽取3题
- 难度分级：初级（0-2年）/中级（2-5年）/高级（5年+）
- 每个问题包含参考答案和追问

### 2.4 职业成长路径模块
- 按4周拆解成长计划（Week1~Week4）
- 每周包含主题、描述、技能标签

### 2.5 AI职业快照模块（新增）
- 渐变色卡片，一眼展示核心结论
- 三卡片职业报告风格：
  - 岗位匹配度（0-100分）
  - 面试准备度（0-100分）
  - 职业竞争力等级（A+ / A / B+ / B / C+ / C）
- 下方展示核心技能 + 提升建议

### 2.6 加载动画（新增）
- 点击分析后显示多步骤进度动画
- 4个步骤依次完成：识别岗位方向 → 提取核心技能 → 计算匹配度 → 生成报告
- 模拟AI思考过程（总计约1.8-2秒）

## 3. 用户界面设计

### 3.1 设计风格
- Modern SaaS Dashboard 风格
- 简洁专业，类似 LinkedIn Career Report
- 结构清晰、模块分区明显
- 不使用聊天界面风格

### 3.2 布局结构
- 顶部：产品标题和简介
- 主体：四个独立功能模块卡片
- 模块之间清晰分隔

### 3.3 响应式设计
- Desktop-first 设计
- 移动端自适应

## 4. 技术实现

- 单HTML文件，无需后端
- 使用现代CSS（Grid/Flexbox）
- 原生JavaScript实现交互逻辑
- 使用示例数据进行演示
