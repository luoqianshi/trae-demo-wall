1. 项目概述

1.1 项目名称

非遗新语 · Heritage Reimagined

1.2 参赛赛道

社会公益 / 造份新善意 / 非遗创新创意开发

1.3 项目背景

中华五千年文明孕育了丰富的非物质文化遗产，截至目前：

1557
国家级非遗项目
3068
省级非遗项目
43
入选UNESCO

然而，随着时代变迁，许多珍贵的非遗技艺面临着传承断层的困境。年轻一代对传统文化缺乏了解渠道，非遗技艺与现代生活的连接日渐薄弱。

1.4 项目目标

以数字科技赋能非遗传承，打造一个集展示、体验、创造、传播于一体的互动平台，让古老的非遗在新时代焕发新生。

1.5 目标用户
  • 青少年群体：通过游戏化方式了解非遗知识
  • 文化爱好者：深度体验非遗魅力
  • 文创从业者：获取设计灵感
  • 普通大众：便捷接触传统文化

2. 设计理念

2.1 核心理念
"科技致敬传统，创意连接未来"
2.2 设计原则
原则描述
国潮美学以中国传统色彩体系（朱砂红、金黄、宣纸米黄）为基调，融合现代设计语言
互动体验通过游戏化、沉浸式交互，让用户"玩"起来，而非被动观看
文化内涵每一个交互元素都承载非遗文化寓意，寓教于乐
AI赋能利用人工智能技术，提供个性化文创设计生成服务
极简技术零外部依赖，纯原生技术实现，确保广泛可访问性
2.3 视觉设计体系

色彩方案：

  • 主色：朱砂红 #8b2c2c — 象征热情与传承
  • 辅色：金黄 #d4a857 — 象征辉煌与尊贵
  • 背景：宣纸米黄 #f4e4c1 — 象征文化底蕴
  • 文字：墨色 #2c1810 — 象征传统书法

视觉元素：

  • 水墨纹理背景
  • 灯笼摇摆动画
  • 祥云漂浮效果
  • 篆体印章装饰
  • 龙纹旋转图案

3. 核心功能模块

3.1 百艺馆 · Heritage Pavilion

功能定位：非遗知识展示与浏览

非遗项目类型特色描述
华县皮影戏传统戏剧"一口道尽千古事，双手对舞百万兵"
景德镇陶瓷传统技艺"白如玉，明如镜，薄如纸，声如磬"
苏绣艺术传统美术"银针走线绘丹青，双面异色夺天工"
剪纸艺术传统美术"一剪一世界，一纸一乾坤"
中国书法传统美术"笔走龙蛇，墨舞千年"
昆曲艺术传统戏剧"百戏之祖，雅韵千年"

交互设计：卡片悬浮动画效果、独特的CSS动画（摆动、旋转、跳动、脉冲）、分类标签与关键词展示

3.2 🎭 皮影戏互动游戏 · Shadow Puppet Game

功能定位：游戏化非遗体验

游戏机制：

  • 操控方式：键盘（方向键/WASD）、鼠标拖动、触屏滑动
  • 目标物：收集祥云 ☁（+10分）、躲避煞风 🌪（-15分）
  • 限时挑战：60秒内获取最高分
  • 本地存储：自动保存历史最高分

文化内涵：皮影人偶造型源自传统皮影戏、手势摇摆动画模拟皮影操控、祥云/煞风元素源自传统戏曲故事

3.3 ✨ 文创坊 · AI Design Lab

功能定位：非遗元素文创设计生成

核心功能：

  • 非遗工艺选择（8种）：青花瓷、苏绣双面绣、皮影纹样、剪纸吉祥图案、书法墨韵、京剧脸谱、唐三彩、云锦织造
  • 配色方案选择（6套）：国潮经典配色，实时预览
  • 应用场景选择（6种）：文创周边、时尚服饰、家居软装、数字产品、食品包装、展览装置
  • 一键生成：基于用户选择，AI生成专属设计方案

输出内容：可视化设计预览图、设计概念描述、三大设计亮点

3.4 📜 问答堂 · Heritage Quiz

功能定位：非遗知识互动问答

题目序号非遗主题知识点
1皮影戏起源西汉时期，世界最早影画艺术
2景德镇陶瓷位于江西省，千年瓷都
3苏绣艺术双面绣为代表技艺
4中国书法王羲之被誉为"书圣"
5昆曲艺术发源于江苏昆山，百戏之祖

称号体系：

  • 🏆 满分（5/5）：非遗传承大师
  • 📜 3-4分：文化爱好者
  • 🌱 2分：文化探索者
  • 🌿 0-1分：文化初学者

4. 技术实现方案

4.1 技术栈
技术版本用途
HTML5标准页面结构与语义化
CSS3标准视觉样式与动画效果
JavaScriptES6+交互逻辑与游戏引擎
4.2 项目结构
非遗新语/
├── index.html          # 主页面（338行）
├── style.css           # 样式文件（750+行）
└── script.js           # 交互逻辑（500+行）
4.3 核心技术实现

动画系统：数字递增动画（统计数据）、CSS Keyframes 动画（灯笼摇摆、祥云漂浮、龙纹旋转）、卡片悬浮动画、皮影手势摇摆动画

游戏引擎：键盘/鼠标/触屏多端适配、碰撞检测算法、目标物生成与管理、计时与计分系统、本地存储（最高分记录）

AI文创生成器：非遗元素数据库、设计灵感知识库、配色方案系统、实时预览渲染

响应式设计：移动端适配、平板适配、桌面端适配

4.4 性能优化
  • 纯原生技术，零外部依赖
  • CSS动画使用GPU加速
  • 事件委托优化交互性能
  • 本地存储减少网络请求

5. 创新亮点

5.1 技术创新
创新点说明
全原生实现不依赖任何第三方框架或库，确保跨平台兼容性与加载速度
多端交互适配同时支持键盘、鼠标、触屏三种操作方式，覆盖不同使用场景
AI辅助设计内置设计灵感数据库，模拟AI生成个性化文创方案
游戏化体验将非遗文化融入互动游戏，让学习变得有趣
5.2 设计创新
创新点说明
国潮视觉体系完整的中国传统色彩与视觉元素设计系统
动态水墨背景CSS实现的水墨纹理效果，营造东方意境
非遗专属动画每个非遗卡片都有独特的动画效果，贴合文化特色
沉浸式舞台皮影戏游戏采用舞台式设计，增强代入感
5.3 文化创新
创新点说明
寓教于乐通过游戏和问答，让用户在娱乐中学习非遗知识
文化基因库系统化整理非遗元素，为文创设计提供灵感源泉
互动传承用户不仅是观看者，更是参与者和创造者
现代连接将传统非遗与现代生活场景（文创、服饰、家居）相结合

7. 作品展示

7.1 页面预览

首页 Hero 区域：大气的标题展示「千年技艺，薪火相传」、非遗数据统计（国家级1557项、省级3068项、UNESCO 43项）、动态灯笼与龙纹装饰、悬浮祥云动画

百艺馆区域：6个非遗卡片，悬浮展示、每个卡片独特的CSS动画效果、非遗分类标签与关键词

皮影戏游戏区域：舞台式设计，红帘装饰、可操控皮影人偶、得分与时间实时显示

文创坊区域：左侧参数选择面板、右侧设计预览与方案生成

问答堂区域：答题卡片设计、进度条与得分展示、答题反馈与解释

7.2 交互演示

游戏操作：

  1. 点击「开始演出」按钮
  2. 使用方向键/WASD或拖动鼠标移动皮影
  3. 收集飘落的祥云，躲避煞风
  4. 60秒内获取最高分

文创生成：

  1. 选择非遗工艺（如青花瓷）
  2. 选择配色方案（如红金配色）
  3. 选择应用场景（如文创周边）
  4. 点击「生成设计灵感」
  5. 查看设计预览与创意亮点

知识问答：

  1. 阅读题目并选择答案
  2. 查看即时反馈与文化解释
  3. 完成5题后查看最终得分与称号
