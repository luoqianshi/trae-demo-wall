# 工业工程自动化计算系统

前后端分离的工业工程计算Web应用，包含11个计算模块。

## 启动方式

```bash
cd engineering-calculator
npm install
npm start
```

访问 http://localhost:3000

## 计算模块

| 模块 | 说明 |
|------|------|
| 除尘器选型（实战） | 渣仓风量、过滤面积、滤袋数量 |
| 除尘器选型（理论） | 排气口面积法 + 物料系数法 |
| 除尘器选型（实战03） | 风量 + 管道阻力计算 |
| 高温闸阀扭矩 | 介质推力、蠕变修正、电机功率 |
| 金属软管压力 | 进出水推力，含设计院对比 |
| 安全阀选型 | 流道面积、流道直径 |
| 链条抗拉概算 | 链板/销轴抗拉强度 |
| 风冷渣量风量 | 理论/设计负压风量 |
| 冷渣机负压风量 | 两种规格对比 |
| 煤节能效益 | 年节能效益额 |
| 扶梯踏步参数 | 2r+g 安全校验 |

## 技术栈

- 后端: Node.js + Express
- 前端: 原生 HTML/CSS/JS
