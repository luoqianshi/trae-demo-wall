## 一、Demo 简介

**是什么：**
知识衰减预警器是一个基于 Web 的 AI 知识管理应用，基于艾宾浩斯遗忘曲线算法，自动追踪每个知识点的遗忘进度，通过游戏化对战机制激励复习。

**面向谁：**
- 在校大学生（尤其是工科、理科、医科等需要大量记忆的学科）
- 考研、考公、考证的备考人群
- 终身学习者、自学者
- 需要定期复习巩固专业知识的技术从业者

**主要功能（5 个核心页面）：**

| 页面 | 功能 |
|------|------|
| **仪表盘** | 知识健康度环形图、等级经验系统、待复习列表、本周衰减趋势 |
| **知识采集** | 拖拽上传截图/PDF/Word，AI 自动识别提取知识点，支持编辑和难度标记 |
| **保卫战** | 游戏化对战复习：60 秒倒计时、BOSS 血条、4 选 1 答题、连击共鸣特效 |
| **知识图谱** | 力导向知识网络可视化，节点颜色标识健康状态，拖拽缩放，点击查看详情 |
| **知识库** | 搜索筛选、批量操作（标记已掌握/导出/删除）、衰减曲线展开、关联知识推荐 |

## 二、Demo 创作思路

**灵感来源：**
作为一名工科学习者，我在学习电路、信号处理、嵌入式等专业知识时，频繁遇到「学完回头看全忘了」的困境。艾宾浩斯遗忘曲线告诉我们，如果不及时复习，一个月后大约 80% 的内容都会被遗忘——但大多数人根本不知道自己哪些知识正在快速流失。

**想解决的问题：**
1. **遗忘不可见**——不知道自己哪些知识点正在快速流失，等到发现时已经忘光了
2. **复习无动力**——传统的闪卡复习枯燥乏味，难以坚持
3. **整理效率低**——手动录入知识点费时费力，拍照/文档无法自动提取
4. **知识无关联**——知识点之间是孤立的，缺少关系图谱来帮助理解记忆

**为什么做这个方向：**
- 手动做复习计划太麻烦，纸质笔记无法追踪遗忘进度
- 市面上的记忆软件（如 Anki）缺少对工程类知识（公式、定理、代码规则）的友好支持
- 游戏化机制（对战、连击、BOSS 血条）能有效提升复习动力，将复习完成率从传统工具的不足 30% 提升到 70%+
- 知识图谱让知识从孤岛变成网络，培养体系化思维

## 五、技术架构

| 层级 | 技术 | 说明 |
|------|------|------|
| 框架 | Next.js 14 (React) | 主应用框架 |
| 动画 | Framer Motion | 页面过渡、节点入场、粒子特效 |
| 图表 | Recharts | 衰减趋势图、健康度环形图 |
| 样式 | Tailwind CSS + 玻璃态设计 | 暗色赛博朋克主题 |
| 图标 | Lucide (开源图标库) | 全部内联 SVG，无 emoji |
| 语言 | TypeScript | 类型安全 |
| OCR | Tesseract.js | 图片文字识别 |
| 文档解析 | pdfjs-dist + mammoth | PDF/Word 文档解析 |
| 字体 | Orbitron + Noto Sans SC | 科技感展示字体 + 中文正文 |

### 核心技术亮点

1. **力导向图模拟** — 排斥力 + 弹簧吸引力 + 中心引力 + 阻尼，每 3 帧更新一次，requestAnimationFrame 驱动流畅动画
2. **艾宾浩斯衰减模型** — 基于时间间隔自动计算每个知识点的遗忘率，智能推荐复习时间
3. **游戏化对战机制** — 60 秒倒计时 + BOSS 血条 + 连击共鸣，将枯燥复习变为趣味对战
4. **纯原生 Demo** — 展示页使用纯 HTML/CSS/JS 实现，零依赖，单文件即可运行

## 六、开发心得

**踩过的坑：**
- 保卫战答案总是 A：shuffle 打乱选项后 answerIndex 硬编码为 0，改为 `options.indexOf(item.content)` 动态定位
- 答对题目重复刷新：干扰项是千篇一律模板文字，改为从其他知识点随机抽取
- 知识图谱连接线脱离节点：节点用 CSS transition 但连线由模拟驱动不同步，改为节点和连线都由同一力模拟直接驱动
- 内存占用 240M：SpotlightCard 用 requestAnimationFrame 节流 + 直接操作 DOM style，Sparkles 从 framer-motion 改为纯 CSS

**TRAE 使用感受：**
TRAE IDE 在整个开发过程中表现出色，从项目初始化到功能实现、Bug 修复、性能优化，都能快速理解需求并给出高质量的代码方案。特别是知识图谱力模拟和保卫战游戏逻辑这类复杂功能，TRAE 能够准确理解物理模型和游戏机制，大幅提升了开发效率。
