## [1.16.0] - 2026-05-24

### ✨ 新增功能

- **配置导入导出** - 支持将所有配置导出为备份文件，也可从备份文件导入
  - 导出内容：所有提示词、分类列表、启用网站、AI API 配置
  - 导入前确认提示，防止误操作覆盖数据
  - 导出文件包含版本号和导出时间，方便后续兼容处理
  - 配置入口放在 GitHub 和 AI 生成按钮之间，点击弹出模态框

## [1.15.0] - 2026-05-15

### ✨ 新增功能

- **Web 端自动填入提示词** - 悬浮窗点击提示词后自动填入 AI 网站聊天输入框
  - 支持 15 个主流 AI 网站：ChatGPT、Claude、Gemini、Copilot、Perplexity、DeepSeek、Kimi、文心一言、通义千问、讯飞星火、豆包、智谱清言、天工AI、腾讯元宝、Poe
  - 智能匹配输入框（textarea / contenteditable / input）
  - 兼容 React、Vue、ProseMirror、TipTap、Quill 等框架
  - 填入失败时自动降级为复制到剪贴板
  - 配置文件 `config/input-selectors.json` 支持自定义选择器

### 🐛 Bug 修复

- 修复 Kimi 输入框无法自动填入的问题（增强 Vue.js 兼容性）
- 修复元素可见性检测对 fixed 定位元素误判的问题

## [1.14.0] - 2026-05-15

### ✨ 新增功能

- **API 连通性检测** - 保存配置后自动发送测试请求验证 API 可用性
  - 连通成功：标题栏显示绿色「已配置 API」
  - 连通失败：标题栏显示红色「配置 API 错误」
  - 检测中：显示灰色「检测中...」

- **悬浮窗分类同步** - 悬浮窗读取主界面的分类筛选状态
  - 主界面选择某分类后，悬浮窗只显示该分类的提示词
  - 选择"全部分类"时显示所有提示词
  - 提示词按主界面分类配置顺序排列
  - 分类筛选状态持久化保存，重新打开扩展后恢复

### 🐛 Bug 修复

- 修复"使用此提示词"跳转后标题栏未填充内容的问题（延迟 focus 避免值被清空）
- 标题填充内容改为用户需求原文前20字

## 如何更新

1. 下载新版本代码
2. 打开 `edge://extensions/` 或 `chrome://extensions/`
3. 找到 AI Prompt Manager，点击「刷新」按钮
4. 如需使用新的默认数据，请在扩展选项中「重置所有数据」
