# 截图说明

此目录用于存放 README 中引用的界面截图。

## 需要的截图文件

| 文件名 | 描述 | 尺寸建议 |
|--------|------|---------|
| `installed.png` | 安装完成 - 扩展图标出现在工具栏 | 360px 宽 |
| `main-interface.png` | 主界面 - 提示词列表 | 360px 宽 |
| `edit-prompt.png` | 添加/编辑提示词弹窗 | 360px 宽 |
| `ai-generate.png` | AI 生成提示词面板 | 360px 宽 |
| `settings.png` | 设置面板 | 360px 宽 |
| `floating-button.png` | 悬浮在网页上的图标 | 400px 宽 |
| `quick-popup.png` | 点击悬浮图标后的快速选择面板 | 280px 宽 |

## 截图方法

1. 在 Edge/Chrome 中加载扩展
2. 使用系统截图工具（Windows: Win+Shift+S / macOS: Cmd+Shift+4）
3. 截取对应界面，保存为 PNG 格式
4. 放入此目录即可

> 提示：截图时建议保持 Apple 风格的浅色主题，与 README 风格一致。
