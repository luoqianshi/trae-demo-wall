## 一、产品概述

### 1.1 产品名称
**开窍** — AI 智能题库入库与分析系统

### 1.2 产品定位
面向中学生的 AI 驱动学习工具，通过拍照识别、智能标签、错题分析、一键组卷等功能，帮助学生高效管理错题、理解错误原因、针对性巩固练习。

### 1.3 核心价值
- **降低录题门槛**：拍照即可入库，告别手动抄题
- **AI 深度分析**：不只给答案，更分析"为什么错"
- **智能组卷**：根据薄弱知识点自动出卷，精准提分
- **数据持久化**：本地存储，随时回顾

### 1.4 目标用户
- 初中/高中学生（核心用户）
- 家长（辅助管理）
- 教师（组卷参考）

### 1.5 使用场景
| 场景 | 描述 |
|------|------|
| 考后复盘 | 学生拿到试卷后，拍照录入错题，AI 分析错因 |
| 日常练习 | 从题库中筛选薄弱知识点，做巩固练习 |
| 考前冲刺 | 一键组卷，针对薄弱环节生成模拟卷 |
| 错题回顾 | 翻阅错题本，查看历史错误记录 |

### 2.2 模块详细说明

---

#### 模块一：📷 拍照入库

**功能描述**：用户上传试卷/练习题图片，系统模拟 AI 识别过程，自动提取题目内容并打上智能标签，入库保存。

**用户故事**：
> 作为一名学生，我想拍一张试卷照片就能把错题录入系统，这样我就不用手动抄题了。

**功能点**：

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F1.1 | 拖拽上传 | P0 | 支持拖拽图片到上传区域 |
| F1.2 | 点击上传 | P0 | 点击按钮选择本地图片文件 |
| F1.3 | 图片预览 | P0 | 上传后显示图片预览，可移除 |
| F1.4 | OCR 识别动画 | P0 | 4 步扫描动画（识别文字→提取选项→匹配答案→生成标签） |
| F1.5 | 识别结果展示 | P0 | 显示识别出的题目文本 |
| F1.6 | 智能标签 | P0 | 自动生成学科、知识点、难度标签（彩色胶囊样式） |
| F1.7 | 保存到题库 | P0 | 将识别结果连同标签存入题库 |
| F1.8 | 题库列表 | P0 | 展示已入库的题目卡片（最近 5 道） |
| F1.9 | 题库操作 | P1 | 每道题支持"分析"和"删除"操作 |

**交互流程**：
```
用户上传图片 → 显示预览 → 点击识别 → 扫描动画(2.4s)
→ 显示识别结果 + 智能标签 → 用户确认 → 保存到题库
```

**智能标签规则**：
| 标签类型 | 颜色 | 示例 |
|----------|------|------|
| 学科标签 | 蓝色渐变 | 数学、物理、化学 |
| 知识点标签 | 紫色渐变 | 二次函数、牛顿定律、氧化还原 |
| 难度标签-简单 | 绿色渐变 | ⭐ |
| 难度标签-中等 | 橙色渐变 | ⭐⭐ |
| 难度标签-困难 | 红色渐变 | ⭐⭐⭐ |

---

#### 模块二：🔍 错题分析

**功能描述**：学生提交做错的题目和自己的答案，AI 进行深度分析，给出错因、正确解法、出题意图和同类巩固题。

**用户故事**：
> 作为一名学生，我做完题对完答案发现错了，我想知道自己到底哪里想错了，正确答案的思路是什么。

**功能点**：

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F2.1 | 来源切换 | P0 | 支持"手动输入"和"从题库选择"两种模式 |
| F2.2 | 题库联动 | P0 | 选择"从题库选择"时，下拉列表展示题库题目 |
| F2.3 | 题目输入 | P0 | 文本框输入/自动填充题目内容 |
| F2.4 | 学科选择 | P0 | 下拉框选择：数学/物理/化学 |
| F2.5 | 答案输入 | P0 | 文本框输入学生的错误答案 |
| F2.6 | 分析加载动画 | P0 | 打字机效果 + 旋转加载动画（1.5s） |
| F2.7 | 分析结果展示 | P0 | 5 个彩色区块展示分析结果 |
| F2.8 | 同类题交互 | P0 | 查看答案（隐藏/显示）+ 换一题 |
| F2.9 | 自动记录 | P0 | 分析完成后自动保存到错题本 |

**分析结果结构**：
| 区块 | 配色 | 内容 |
|------|------|------|
| 答案对比 | 橙色 | 学生答案 vs 正确思路的差异 |
| 错误原因 | 红色 | 错误类型标签 + 原因描述 |
| 正确解法 | 绿色 | 完整的解题步骤 |
| 出题意图 | 蓝色 | 考查的知识点和能力 |
| 同类巩固题 | 紫色 | 类似题目 + 答案 + 换题功能 |

**错误类型分类**：
| 类型 | 标签颜色 | 说明 |
|------|----------|------|
| 概念不清 | 橙色 | 基础概念理解有误 |
| 计算失误 | 红色 | 思路对但计算出错 |
| 审题不清 | 紫色 | 没有看清题目要求 |

#### 模块三：📊 错题本

**功能描述**：汇总所有分析过的错题，支持按学科筛选、统计和导出。

**用户故事**：
> 作为一名学生，我想集中查看所有做错的题目，按学科分类，方便考前复习。

**功能点**：

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F3.1 | 统计面板 | P0 | 四宫格展示：总错题/数学/物理/化学数量 |
| F3.2 | 筛选功能 | P0 | 下拉框按学科筛选（全部/数学/物理/化学） |
| F3.3 | 错题列表 | P0 | 卡片式展示，含学科标签、题目、错因、日期 |
| F3.4 | 删除功能 | P0 | 单条删除，需确认 |
| F3.5 | 导出功能 | P1 | 导出为 txt 文件 |

**错题卡片结构**：
```
┌─────────────────────────────────┐
│ [学科标签]              [删除]  │
│ 题目：xxxxxxx                   │
│ 错误原因：xxxxxxx               │
│ ─────────────────────────────── │
│ 📅 2024-01-15                   │
└─────────────────────────────────┘
```

#### 模块四：📝 一键组卷

**功能描述**：用户设置组卷条件，AI 从题库中自动挑选符合条件的题目，生成一套新试卷，支持预览和导出。

**用户故事**：
> 作为一名学生，我想针对"二次函数"这个薄弱知识点出一套卷子专门练习。

**功能点**：

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| F4.1 | 学科筛选 | P0 | 全部/数学/物理/化学 |
| F4.2 | 难度筛选 | P0 | 全部/⭐简单/⭐⭐中等/⭐⭐⭐困难 |
| F4.3 | 知识点筛选 | P0 | 全部/二次函数/牛顿定律/氧化还原 |
| F4.4 | 题目数量 | P0 | 数字输入，1-20 题，默认 5 |
| F4.5 | 智能选题 | P0 | 按条件筛选后随机抽取 |
| F4.6 | 试卷预览 | P0 | 带标题、题号、标签的完整试卷 |
| F4.7 | 导出试卷 | P1 | 导出为 txt 文件 |
| F4.8 | 重新生成 | P0 | 保持条件不变，重新随机选题 |

**试卷格式**：
```
═══════════════════════════════════
       数学 智能测试卷
  共 5 题 | 生成时间：2026-06-21
═══════════════════════════════════

① (数学 · 二次函数)
   已知函数 f(x) = x² - 4x + 3...
   [⭐⭐]

② (数学 · 二次函数)
   已知等差数列 {an} 中...
   [⭐]
```

## 三、非功能需求

| 指标 | 要求 |

## 六、视觉设计规范

### 6.1 配色方案
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #4a90e2 → #63a4ff | 蓝色渐变，头部/按钮/主交互 |
| 背景 | #667eea → #764ba2 | 紫蓝渐变，页面背景 |
| 成功 | #26de81 → #57e8a0 | 绿色渐变，正确/保存 |
| 警告 | #ffa502 → #ffbe42 | 橙色渐变，中等难度/概念错误 |
| 危险 | #ff6b6b → #ff8a8a | 红色渐变，删除/计算错误 |
| 知识点 | #a55eea → #c084fc | 紫色渐变，知识点标签/组卷 |
| 文字 | #333/#444/#555/#666/#999 | 多级灰度 |
| 背景面 | #f0f4f8/#f8fafc/#fafbfc | 浅灰蓝层级 |

### 6.2 设计原则
- **圆角风格**：卡片 12-16px，按钮 10-12px，标签 20px
- **柔和阴影**：`0 2px 10px rgba(0,0,0,0.04)` 到 `0 6px 20px rgba(0,0,0,0.08)`
- **渐变按钮**：所有主操作按钮使用渐变色 + 阴影
- **动效反馈**：hover 上浮、active 缩放、淡入滑入动画
- **手机模拟器**：390×844 iPhone 尺寸，含刘海、侧键、Home Indicator
