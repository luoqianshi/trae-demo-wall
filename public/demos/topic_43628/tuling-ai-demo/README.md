# 途灵 AI - 智能旅行计划生成器

> 一句话生成完美旅行计划 —— 行程 / 预算 / 打卡清单一站搞定

---

## 项目简介

途灵 AI 是一款基于 Web 的智能旅行计划生成器。用户只需输入目的地、旅行天数和偏好标签，AI 即可在数秒内生成一份完整的旅行方案，包含每日行程安排、交通建议、餐饮推荐、预算估算和打卡清单。

本项目是 **TRAE AI 创造力大赛** 的初赛参赛作品，使用 TRAE Work 的 Auto 模式开发完成。

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 智能行程规划 | 根据目的地和天数，AI 自动规划每日景点路线和时间安排 |
| 预算自动估算 | 住宿、餐饮、交通、门票四大类预算明细，支持经济/舒适/豪华三档 |
| 打卡清单生成 | 景点、美食、体验三类清单，支持在线勾选打卡 |
| 城市数据覆盖 | 内置成都、大理、西安、杭州等热门城市数据 |
| 移动端适配 | 完美适配手机浏览器，支持刘海屏、深色模式、触摸交互 |

## 快速开始

### 方式一：直接打开（推荐）

1. 下载项目 ZIP 文件并解压
2. 双击 `index.html` 即可在浏览器中运行
3. 无需安装任何依赖，无需联网

### 方式二：本地服务器

```bash
# 进入项目目录
cd tuling-ai-demo

# 启动本地服务器（Python 3）
python3 -m http.server 8080

# 浏览器访问 http://localhost:8080
```

## 使用指南

1. **打开应用**：在浏览器中打开 `index.html`
2. **开始规划**：点击「开始规划旅程」按钮
3. **填写信息**：
   - 输入目的地（如：成都、大理、西安、杭州）
   - 选择旅行天数（2/3/5/7 天）
   - 选择偏好标签（美食/文化/自然/购物/休闲/冒险）
   - 选择预算档位（经济/舒适/豪华）
4. **生成方案**：点击「生成旅行计划」，等待 AI 生成结果
5. **查看结果**：浏览每日行程、预算明细和打卡清单
