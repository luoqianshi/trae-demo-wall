# 园林智意 - 中国园林设计意图量化与推理系统

## 项目简介

本项目旨在将中国园林设计师的隐性决策转化为可计算的显式规则，并进行空间推理。通过图神经网络（GNN）实现设计效果预测、参数敏感度分析和自动优化建议。

## 前端Demo功能

### 核心功能
1. **园林场景交互** - 可视化展示园林元素布局
2. **设计原则教学** - 借景、框景、对景、序列四大传统造园手法
3. **AI设计评分** - 实时预测设计效果评分
4. **参数敏感度分析** - 调整元素位置查看评分变化
5. **设计约束检查** - 检测设计原则违反情况

### 交互方式
- 点击元素查看详情
- 拖动元素调整位置（模型实时评分）
- 使用滑块微调位置
- 切换网格/标签/关系线显示
- 点击设计原则查看演示

## 技术特点

- **前后端分离** - Flask后端提供API服务
- **中国园林美学风格** - 采用传统配色（竹绿、朱砂红、墨色、金色）
- **响应式设计** - 支持多种屏幕尺寸
- **SVG矢量图形** - 元素图形清晰无损
- **双模式支持** - 优先使用后端API，失败自动切换到本地计算

## 功能说明

### 园林场景区域
- 包含假山、观景柱、水池、古树、观景亭等园林元素
- 点击元素可查看详细信息
- 拖动元素可调整位置，AI实时计算评分变化

### 设计原则教学
- 借景：利用外部景观增强空间层次
- 框景：通过门框、窗框等形成画面
- 对景：在轴线两端设置相互对应的景观
- 序列：空间组织应形成有节奏的序列

### 模型预测面板
- 层次感评分：评估元素布局的层次分明程度
- 空间感评分：评估空间通透度和开阔感
- 整体评分：综合评估设计效果
- AI分析：提供专业的设计建议
