# 大明赛博工坊 · Cyber Workshop of Ming Dynasty

> *以现代数字交互技术深度复刻《天工开物》中的冶铁工艺，将古老的典籍智慧与赛博美学融为一体。*

---

## ⚙️ 交互逻辑

在这个数字熔炉中，你将扮演一名督造钢铁的明代工匠，通过四种核心动作实时影响炉内状态，做出你自己的炼钢决策：

| 动作 | 效果 | 典籍注 |
| :--: | :--: | :--: |
| 🔥 **加大风箱** | 炉温 ↑，进度 ↑ | 凡铁分生、熟，出炉未炒则生，既炒则熟。 |
| ◈ **投放石灰石** | 纯度 ↑↑，进度 ↑ | 凡石灰，经火焚炼为用，能去铁中之硫。 |
| 🌲 **投入松脂** | 炉温 ↑，进度 ↑ | 松脂助燃，焰光乍现，以利火候。 |
| ❄ **退温静观** | 炉温 ↓↓，纯度 ↑ | 火候过则铁脆，稍退温，静观炉内反应。 |

实时影响的三个核心指标：**炉温**、**纯度**、**进度**。当进度达到 100%，钢乃成。

---

## 🏗️ 技术实现

- **前端框架**：**Vue 3**（Composition API + `<script setup>`），构建响应式的冶炼状态管理与交互逻辑
- **样式系统**：**Tailwind CSS** 驱动的玻璃拟态（Glassmorphism）UI，半透明毛玻璃 + 柔和阴影，打造漂浮于宣纸之上的视觉层次
- **动效与滤镜**：CSS `backdrop-filter` 实现毛玻璃、`mix-blend-mode` 融合印章与背景、CSS 关键帧动画实现悬停反馈与环形进度条

## 🚀 开发说明

```bash

npm run dev
```
