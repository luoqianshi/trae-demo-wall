# 声脉方志｜初赛 Demo 使用说明

## 打开方式

解压 Zip 后，双击 `index.html` 即可在浏览器中体验。

## 推荐体验流程

1. 首页点击“开始体验采集流程”。
2. 在“口述采集工作台”选择采集主题。
3. 点击“模拟录音并转写”。
4. 查看 AI 生成的方言原文、普通话释义、关键词和古籍关联。
5. 点击“生成数字馆藏页”。
6. 切换到“运营后台”查看机构端预览。

## 文件说明

- `index.html`：Demo 主页面
- `style.css`：页面视觉样式
- `app.js`：交互逻辑与模拟 AI 数据
- `初赛Demo发布帖模板.md`：可直接复制修改的社区发帖模板

## 适合提交方式

该 Demo 是交互式 HTML 文件，适合按初赛要求打包成 Zip 上传到社区。
