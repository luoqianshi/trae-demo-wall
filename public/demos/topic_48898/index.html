Core Modules

六大核心模块

覆盖浏览器自动化的常见场景，每个模块均内置合规保护，开箱即用。

🤖

RPA 自动化

网页自动化操作引擎，支持表单填写、点击导航、文件上传、弹窗处理等 20+ 动作。

  • 可视化步骤定义，JSON 配置
  • 自动等待 + 导航竞态修复
  • 支持视频录制 / Trace 调试
100 步上限 · 验证码拦截 · 500ms 间隔
📷

视觉输出

批量网页截图、PDF 生成、视频录制，支持整页、视口、元素三种模式。

  • 整页 / 视口 / 元素 三种截图模式
  • 批量处理，最多 20 个目标
  • PDF / PNG / WebM / Trace 多格式
批量上限 20 · 版权合规提醒
🌐

网络控制

请求级别的网络控制能力，支持监控、拦截、Mock、修改响应和 HAR 录制。

  • URL 模式匹配，灵活配置规则
  • Mock 任意状态码和响应体
  • HAR 录制 + 网络日志导出
高风险域名禁 Mock · 规则上限 20
📱

多环境模拟

设备、时区、语言、地理位置、网络条件一体化模拟，还原真实用户环境。

  • iPhone / Android / 平板等 10+ 预设
  • 时区 / 语言 / 地理位置模拟
  • 弱网 / 离线 / 高延迟 网络模拟
防欺诈警示 · 透明标识 Header
🕵️

智能爬虫

合规版数据采集工具，懒加载识别，敏感路径/字段自动拦截，限速保护。

  • 滚动触发懒加载内容识别
  • 预置 4 种字段映射模板
  • 自动生成摘要报告
200 条硬上限 · 敏感路径拦截 · ≥1s 间隔
🎥

步骤录制器

交互式操作录制，一边操作一边记录，一键导出可直接运行的 RPA 步骤 JSON。

  • 命令行交互控制录制流程
  • 支持撤销 / 重做 / 预览
  • 导出即运行，无缝对接 RPA
透明标识 · 操作可追溯
Compliance First

三层合规保护体系

从「被动提醒」到「主动拦截」， 用技术手段确保工具始终在合法边界内运行。

  • 🛡️

    硬限制拦截

    超出数量/频率上限立即终止，敏感路径和字段直接拒绝执行。

  • ⚠️

    软限制警告

    高风险操作前弹出警示，确认后才能继续执行。

  • 👁️

    透明标识

    所有请求自动携带 X-Playwright-Toolkit Header，方便网站识别和管理。

🛡️ 第一层 · 硬限制
超限直接终止，从根本上杜绝滥用
200条爬虫上限 100步RPA上限 敏感路径拦截 高风险域名禁Mock 验证码选择器拦截 请求间隔≥1s
⚠️ 第二层 · 软限制
风险操作前警告，提醒用户确认
Mock风险提示 地理模拟警示 版权内容提醒 弱网模拟提示
👁️ 第三层 · 透明标识
所有请求携带标识Header，方便网站识别
X-Playwright-Toolkit User-Agent 标识 操作日志可追溯
