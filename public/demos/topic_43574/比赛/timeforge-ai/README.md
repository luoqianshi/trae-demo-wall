# TimeForge AI - 智能学习任务管理系统

> 一款面向大学生群体的智能学习任务管理工具，融合 AI 任务拆解、DDL 风险预测、学习战力值等核心功能，帮助学习者高效规划学习任务，告别拖延，提升学习效率。

---

## 功能特性

### 1. 仪表盘总览
提供学习任务的整体概览，一目了然地展示进行中任务、已完成任务、今日专注时长、DDL 风险分布等关键指标。支持快速跳转至 AI 任务拆解、DDL 风险预测等核心功能。

### 2. 任务管理
支持创建、查看和管理学习任务。每个任务可设置标题、类型（实验报告/课程设计/考试复习/英语四级/考研复习/竞赛项目/其他）、截止日期和难度等级。任务状态包括"进行中""已完成""已延期"三种。

### 3. AI 任务拆解
根据任务类型自动拆解为多个子任务，并基于截止日期生成每日学习计划。输入任务信息后，AI 引擎将智能分配子任务工时，生成可视化进度条和每日工作量建议。

### 4. DDL 风险预测
基于截止日期、预估工时和实际完成进度，实时计算每个任务的风险等级（绿色/黄色/红色）。提供风险分布统计和任务完成率分析，帮助学习者及时调整计划，避免任务逾期。

### 5. 学习战力值
将学习行为量化为战力值系统，包含四个维度：专注力（累计学习时长）、执行力（完成任务数量）、持续力（连续打卡天数）、规划力（综合等级评估）。战力值随学习行为动态变化，提供等级晋升机制。

### 6. 数据可视化中心
提供 30 天学习热力图、任务完成率饼图、拖延指数分析、专注趋势折线图、DDL 风险变化图、任务类型分布统计和周度学习报告等全方位数据可视化功能。

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **后端框架** | Flask (Python) | 轻量级 Web 框架，提供 RESTful API |
| **数据库** | SQLite 3 | 轻量级嵌入式数据库，零配置 |
| **前端框架** | Bootstrap 5 | 响应式 UI 框架 |
| **图表可视化** | Chart.js 4.x | 灵活的数据可视化库 |
| **图标库** | Bootstrap Icons | 丰富的图标资源 |
| **AI 引擎** | 规则引擎 + 模板系统 | 基于任务类型的智能拆解与建议生成 |

## 快速开始

### 环境要求

- Python 3.8 或更高版本
- pip 包管理器

### 安装依赖

```bash
cd timeforge-ai
pip install -r requirements.txt
```

### 初始化数据库（填充演示数据）

```bash
python seed_data.py
```

### 启动服务

```bash
python app.py
```

启动后访问：

- 仪表盘首页：http://localhost:5000/
- 任务管理：http://localhost:5000/tasks
- AI 任务拆解：http://localhost:5000/decompose
- DDL 风险预测：http://localhost:5000/ddl-risk
- 学习战力值：http://localhost:5000/battle-power
- AI 课程助手：http://localhost:5000/assistant
- 数据可视化：http://localhost:5000/analytics

## 界面预览

TimeForge AI 采用深色科技风主题设计，以深蓝紫色调为主色，搭配渐变光影效果，营造专业、现代、科技感十足的用户界面。主要视觉特点：

- **深色主题**：降低长时间使用造成的视觉疲劳
- **渐变色彩**：主色调使用紫色到蓝色的渐变，突出科技感
- **侧边栏导航**：分组清晰的导航菜单，支持移动端自适应折叠
- **卡片式布局**：信息模块化展示，层次分明
- **响应式设计**：完美适配桌面端和移动端
