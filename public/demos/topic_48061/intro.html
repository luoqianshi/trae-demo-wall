WHY

为什么需要它？

因为「我等下再起来」从来没有「等下」。

🪑

久坐

连续坐 4 小时不动是常态。腰、颈、肩、视力，一点点透支。

🧠

脑雾

下午 3 点开始注意力涣散，却不知道该做什么来"重启"自己。

😶‍🌫️

没仪式感

知道要多喝水、多走动，但没人给你打节拍，没人给你正反馈。

✨

HEAL CARD 把它做成了一场游戏

抽卡 → 翻牌 → 倒计时 → +HP → 解锁图鉴 → 累计连签。每一次"动一下"，都有声音、有反馈、有进度。

🌱

而你，只要在工位上动一下就够了

5 秒眨眨眼，1 分钟喝口水，3 分钟冥想，8 分钟下楼。坚持比强度更重要。

HOW

三步开始

1

选状态

告诉它你现在是「眼酸」「腰僵」还是「想吃东西」，不选也可以。

2

抽 3 张卡

系统按状态加权，从 28 张卡牌中抽 3 张。SSR 出现时会有惊喜动画。

3

真的去做

点开卡牌看倒计时，做完点"完成回血"，HP +N，图鉴 +1。

4

继续抽

本批完成后，按钮变成「抽第 N 轮」，今日上限 100 HP。

TECH

技术细节

🪶

零依赖

原生 HTML / CSS / ES Modules，不打包、不构建、不联网。打开浏览器就能跑。

🔒

本地存储

所有数据保存在浏览器 LocalStorage 中，不上传、不收集、不分析。

📦

解压即用

下载 zip → 解压 → 双击 app.html，或用任意静态服务器打开（推荐）。

🎨

可拓展

修改 data/cards.js 就能加自己的卡牌，自定义 HP / 时长 / 描述。
