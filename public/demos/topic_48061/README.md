# HEAL CARD · 工位回血卡

> 一款面向久坐办公人群的 **游戏化健康微习惯** 单页应用。抽卡 → 翻牌 → 完成 30-180 秒微小恢复动作 → 累计 HP / 解锁图鉴 / 维持连签。

**v1.0.0** · 纯前端 · 零构建 · 零后端 · 数据存于浏览器 LocalStorage

---

## 🚀 如何使用

### 方式 A：双击打开（最简单）

1. 解压 `heal-card.zip`
2. 双击 `index.html`（会自动跳转 `app.html#today`）
3. 完成！

> 💡 推荐使用 **Chrome / Edge / Safari / Firefox 最新版**

### 方式 B：本地静态服务器（兼容性更好）

如果方式 A 在某些浏览器下出现 ES Module 加载失败（少见），请用以下任一命令在解压目录启动本地服务器：

```bash
# 需 Python 3
python3 -m http.server 8080

# 或 Node.js
npx serve .
```

然后浏览器访问 `http://localhost:8080`

## 🎮 玩法速览

| Tab | 功能 |
|---|---|
| **今日回血** `#today` | 选状态 → 抽 3 张卡 → 翻牌 → 倒计时 → 完成 → 累计 HP |
| **卡牌图鉴** `#library` | 收集进度、按稀有度筛选、点击查看详情 |
| **历史 & 周报** `#history` | 4 周热力图、本周回血曲线、周报数据 |
| **我的** `#me` | 个人数据、回血提醒、卡池偏好、导入导出、关于 |

### 稀有度

| 稀有度 | 数量 | 权重 | HP |
|---|---|---|---|
| N（普通） | 14 | 65% | 8–10 |
| R（稀有） | 8 | 25% | 14–18 |
| SR（史诗） | 4 | 8% | 22–25 |
| SSR（传说） | 2 | 2% | 35 |

### 特殊机制

- **饭点限定卡**：仅 11:30-13:30 / 17:30-19:30 出现（可在「我的」关闭）
- **连签**：每天首次抽卡 +1；隔日无抽则归零
- **跨日重置**：自动检测，无需手动

## 💾 数据管理

- 数据键：`localStorage['heal-app-state-v1']`
- **导出**：「我的」→ 导出，下载 JSON 备份
- **导入**：「我的」→ 导入，选择 JSON 文件（**覆盖**当前数据）
- **重置**：「我的」→ 重置，清空所有数据（含二次确认）

---

## 🔔 关于浏览器通知

「我的」→ 回血提醒，开启后会请求 Notification 权限。**仅当浏览器/页面打开时**会在设定时间弹出提醒（不依赖 Service Worker）。
