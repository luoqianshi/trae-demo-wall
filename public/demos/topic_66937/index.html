Q
应届生没有相关工作经验，简历怎么写？
没有经验也没关系，重点是展示你的潜力！可以从这几个方面入手：

1. 实习经历：哪怕时间短也要写，突出具体做了什么
2. 项目经验：课程作业、毕业设计都可以包装
3. 技能证书：英语、计算机、专业证书等
4. 校园经历：社团、学生会、比赛等体现综合能力

关键是用STAR法则描述，量化成果，让HR看到你的价值。
简历 应届生 求职技巧
Q
面试时如何做自我介绍？
一个好的自我介绍应该控制在2-3分钟，包含以下内容：

1. 基本信息：姓名、学校、专业
2. 匹配亮点：2-3个与岗位相关的经历/能力
3. 求职动机：为什么选择这个公司/岗位
4. 结尾感谢：表达对面试机会的感谢

提前准备好并反复练习，保持自信和微笑！
面试 自我介绍
Q
三方协议和劳动合同有什么区别？
两者有本质区别：

三方协议：
• 主体：毕业生、学校、用人单位三方
• 性质：就业意向约定，不是劳动合同
• 效力：毕业生到单位报到后自动终止

劳动合同：
• 主体：劳动者和用人单位双方
• 性质：确立劳动关系的法律文件
• 效力：受劳动法保护，具有法律约束力
三方协议 劳动合同
Q
试用期被辞退有补偿吗？
试用期被辞退是否有补偿，要看具体情况：

有补偿的情况：
• 用人单位违法解除劳动合同
• 用人单位无正当理由辞退

无补偿的情况：
• 在试用期间被证明不符合录用条件
• 严重违反用人单位规章制度
• 严重失职、营私舞弊给单位造成重大损害

建议保留好相关证据，必要时可以申请劳动仲裁。
试用期 辞退 补偿
Q
刚入职和同事相处要注意什么？
职场新人与同事相处的黄金法则：

1. 主动友好：主动打招呼、微笑，营造良好第一印象
2. 多听少说：先了解团队文化，不要急于发表意见
3. 虚心请教：遇到问题主动请教，但要先自己思考
4. 懂得感恩：别人帮忙后要表示感谢
5. 保持距离：不要过早卷入职场政治和八卦
6. 信守承诺：答应的事情一定要做到
职场新人 人际关系
Q
第一份工作应该看重什么？
对于应届生来说，第一份工作的优先级建议：

1. 平台和成长：大平台或能学到东西的团队 > 高薪
2. 行业前景：选择朝阳行业，未来发展空间更大
3. 直属领导：好的领导能让你少走很多弯路
4. 兴趣匹配：做自己喜欢的事更容易坚持和出彩
5. 薪资待遇：保障基本生活即可，不要太纠结起薪

记住：第一份工作的价值不在于赚多少钱，而在于你能学到多少。
职业规划 应届生
Q
群面（无领导小组讨论）有什么技巧？
群面考察的是团队协作和领导力，关键技巧：

1. 角色定位：不一定非要当leader，timer（计时者）、总结者同样出彩
2. 发言质量：宁可不说话也不要说废话，每次发言要有价值
3. 团队意识：引导沉默的同学发言，体现团队精神
4. 逻辑清晰：发言时用"第一、第二、第三"，条理分明
5. 注意倾听：不要打断别人，先肯定再补充
6. 把握节奏：注意时间进度，适时推进讨论
群面 面试技巧
Q
工作中犯错被领导发现了怎么办？
工作中犯错不可怕，关键是处理方式：

1. 主动承认：不要等领导发现，主动汇报错误
2. 分析原因：找出根本原因，不要只说"我失误了"
3. 提出方案：带上解决方案去汇报，而不只是报问题
4. 及时止损：第一时间采取补救措施，降低影响
5. 总结复盘：事后复盘，建立流程避免重犯

切忌：隐瞒不报、推卸责任、找借口辩解。
职场错误 危机处理
Q
五险一金是什么？缴费比例是多少？
五险一金是法定福利，每个正式员工都应该享有：

五险（个人+单位共同缴纳）：
• 养老保险：个人8%，单位16%
• 医疗保险：个人2%，单位8%
• 失业保险：个人0.5%，单位0.5%
• 工伤保险：单位0.2-1.9%，个人不缴
• 生育保险：单位0.8%，个人不缴

一金：
• 住房公积金：个人5-12%，单位同等比例

注意：缴费基数和比例因城市而异，以上为一般参考。面试谈薪时一定要问清楚五险一金的缴费基数和比例。
五险一金 福利
Q
签了三方协议后想毁约，需要承担什么后果？
三方协议毁约可能面临以下后果：

1. 违约金：协议中通常约定违约金，一般为1个月工资左右
2. 学校手续：需要学校重新出具新三方协议
3. 企业关系：可能影响与该企业的关系
4. 时间成本：重新走签约流程可能错过招聘季

建议：
• 签约前慎重考虑，不要为了"保底"而签约
• 确需毁约时，尽早通知对方，态度诚恳
• 了解学校关于三方违约的具体政策
三方协议 毁约
Q
应届生第一份工作选大厂还是小公司？
大厂和小公司各有优劣，关键看你的职业阶段和性格：

大公司优势：
• 完善的培训体系和晋升通道
• 规范的流程和制度
• 简历背书，跳槽更容易
• 优秀同事多，成长快

小公司优势：
• 接触更全面的工作，锻炼综合能力
• 扁平管理，表现容易被看到
• 成长空间大，可能拿到期权
• 参与核心业务的机会多

建议：第一份工作优先选大厂，打好基础和背书。有了大厂经历后，后续选择更灵活。
职业选择 大厂
Q
应届生入职后如何快速提升自己？
入职后的前3个月是黄金成长期，建议从以下方面入手：

1. 快速学习业务：主动了解公司产品、流程、组织架构
2. 建立人际关系：认识关键协作方，找到可请教的导师
3. 记录复盘：每天记录学到的新知识，每周复盘总结
4. 主动承担：在能力范围内多承担任务，积累经验
5. 学会提问：先自己思考再提问，提问时带着方案
6. 建立知识库：整理工作文档、模板、最佳实践
7. 关注行业：持续关注行业动态和竞品分析

记住：前3个月不要怕犯错，关键是学习速度和态度。
