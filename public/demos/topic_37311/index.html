1

拍照 / 选图

用户点击"立即体验"，调用 wx.chooseMedia 选择或拍摄正面照片

wx.chooseMedia({ mediaType: ['image'], sourceType: ['album', 'camera'], success: res => analyzePhoto(res) })
2

AI 生成阶段

模拟 AI 渐进式生成，6 步骤展示进度，实时追加智能小提示

const steps = [ { progress: 15, text: '识别面部轮廓特征...' }, { progress: 35, text: '分析脸型与五官比例...' }, { progress: 55, text: '匹配海量发型数据库...' }, { progress: 75, text: '生成发型设计效果图...' }, { progress: 92, text: '优化细节与发色搭配...' }, { progress: 100, text: '完成！' } ];

生成多方案

根据用户参数（性别 / 长度 / 风格 / 脸型）从模板库动态组合 6 种方案

const templates = getSchemeTemplates({ length: 'medium', // 长度: short/medium/long style: 'casual', // 风格: casual/professional/fashion gender: 'female', // 性别: male/female faceShape: 'oval' // 脸型: round/oval/square/heart }); const selectedSchemes = pickBestSix(templates, userPhoto);

❓ 常见问题

你可能想了解的问题

点击问题展开查看详细解答。

这款小程序需要联网使用吗？+
核心功能（生日提醒、运势查询、生肖星座查询、本地工具等）100% 离线可用。AI 发型设计功能在首次加载模板时需要联网，后续可离线运行。
我的生日数据会上传到云端吗？安全吗？+
所有个人数据均存储在微信小程序本地存储中，不上传任何服务器。数据采用加密存储，仅在你的设备上可见，完全安全。
AI 生成的发型效果图准确吗？+
AI 发型设计通过 CSS 渐变图形叠加在用户照片上，提供直观的视觉参考。虽然不是 100% 真实的照片级效果图，但能很好地帮助用户判断发型风格是否适合自己，是去发廊前的重要决策参考。
农历生日的转换准确吗？会不会有误差？+
农历转换基于标准的农历算法实现，支持 1900 年至 2100 年范围内的精确转换。同时针对星座边界日（如 1 月 20 日、5 月 21 日等易出错的日期）做了双重判断，避免边界错误。
工具未来会持续更新吗？+
会的。项目采用模块化开发架构，新增工具只需创建一个页面文件夹，接入成本极低。我们计划持续更新更多实用工具，欢迎在反馈区提出你想要的功能建议。
为什么选择微信小程序而不是独立 App？+
微信小程序无需下载安装包，即点即用，用户门槛低。无需处理 iOS/Android 多端适配，开发效率高。适合工具类应用的快速迭代与分发，让更多用户能轻松体验。
能在电脑端使用吗？+
可以。在 PC 端微信中搜索"智能生活小工具"即可打开。此外在微信开发者工具中也能预览全部功能，方便测试和调试。主要推荐在手机端体验以获得最佳效果。
如何将喜欢的方案分享给朋友？+
在方案详情页点击"📤 分享"按钮，可选择发送给朋友或朋友圈。分享卡片包含方案名称、效果图和小程序入口，朋友打开后可直接进入查看完整方案。
