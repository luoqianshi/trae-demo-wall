- **版本类型**：H5 离线版（纯前端，无后端）
- **技术栈**：Taro + React + TypeScript

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 解压 zip 包到任意目录
2. 双击 `index.html` 文件
3. 在浏览器中即可体验

> 建议使用 Chrome / Edge / Firefox 等现代浏览器

### 方式二：本地服务器（如直接打开有问题）

如果直接双击打开出现跨域问题，可以使用以下方式启动本地服务器：

#### Python 方式
```bash
# Python 3
python -m http.server 8080

# 然后浏览器访问 http://localhost:8080
```

#### Node.js 方式
```bash
npx serve .
# 或
npx http-server -p 8080
```

## 🔧 技术说明

### 为什么可以离线运行？

本项目采用以下设计，确保纯前端即可完整运行：

1. **数据层抽象**：所有数据操作通过 `services/` 层抽象，可无缝切换 Mock / 真实后端
2. **本地存储**：Zustand store 使用 `localStorage` 持久化用户数据
3. **Hash 路由**：使用 Hash 模式路由，无需服务器配置
4. **相对路径**：所有资源使用相对路径引用，可在任意目录运行

## 📋 常见问题

### Q: 直接打开 index.html 白屏怎么办？

A: 请检查：
1. 是否使用了现代浏览器（Chrome/Edge/Firefox）
2. 是否解压完整（不要在压缩包内直接打开）
3. 尝试使用本地服务器方式启动（见上方"方式二"）

### Q: 发布的善行数据会保存吗？

A: 会保存到浏览器的 localStorage 中。清除浏览器数据后会重置为初始 Mock 数据。

### Q: AI回应是真实的吗？

A: 离线版中AI回应为预设的示例回复，用于展示功能效果。正式版会接入真实的AI服务。

### Q: 图片上传功能能用吗？

A: 离线版中图片上传使用占位图模拟，用于展示交互流程。
