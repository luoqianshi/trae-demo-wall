面向谁

核心功能
MeCab歌词深度解析（词汇+语法+JLPT等级）、5套AI动漫背景+多层动画引擎、194首歌曲库+三格式输出

技术架构

完整pipeline已跑通，包含5个核心模块

歌曲获取 → 歌词抓取 → 日语分析 → 视频生成 → 多格式输出
歌曲获取
Spotify API + 194首fallback歌曲库，支持手动导入LRC
歌词抓取
Uta-Net / UtaMap / 歌詞ナビ 多源抓取，LRC时间轴解析
日语分析
MeCab形态分析 + JLPT等级标注 + 语法点提取 + tokens动态对齐
视频生成
MoviePy + Pillow，飞入飞出动画 + 粒子系统 + Ken Burns背景
多格式输出
TikTok (1080x1920) / YouTube (1920x1080) / Instagram (1080x1080)
# 处理单首歌曲（带音频） python main.py --song "夜に駆ける" --artist "YOASOBI" --audio assets/audio/yoru_ni_kakeru.mp3 # 批量处理（使用缓存数据） python main.py --video-only --max 5 # 仅生成预览帧 python main.py --video-only --preview
