# 慧识 HR Demo

一个纯前端、离线可运行的产品 Demo，用"HR 招聘流程复盘"演示慧识 HR 的核心闭环。

## 核心创新

**不是用 AI 替代员工，而是用 AI 帮助员工升级工作方法。**

传统绩效系统：管理者给员工打分排名 → 员工被动接受评价 → 复盘流于形式。

慧识 HR：员工自己记录成长 → AI 辅助生成改进建议 → 员工主动选择采纳 → 形成可复用的成长证据和模板 → 管理者只看到团队共性问题。

## 功能闭环

1. **员工复盘输入**：写下真实工作片段，AI 帮你补全结构化复盘。
2. **AI 增益建议**：本地规则引擎生成可解释的 P0/P1/P2 建议，带命中规则和证据。
3. **建议采纳**：员工勾选适用建议，不适用的可以忽略。
4. **v1 改进版**：生成改进方案、沉淀模板、预测指标改善。
5. **证据对比**：v0 原始 vs v1 改进，四维成长雷达图可视化。
6. **管理看板**：团队共性问题、风险信号、成长趋势、能力分布。
7. **TRAE 集成**：导出 Markdown 报告，在 TRAE AI 中生成深度人才发展分析。

## 如何打开

直接双击 `index.html` 即可运行，不需要联网、不需要安装依赖。

## 演示路径（比赛推荐）

1. **首页**：看产品定位、差异化价值、用户证言。
2. **员工复盘**：体验"AI 帮我补全"→ 生成建议 → 勾选采纳 → 生成 v1。
3. **证据对比**：看雷达图、指标变化、v0/v1 差异。
4. **管理看板**：看团队共性问题、趋势图、能力分布。
5. **数据设置**：导出 JSON 或 Markdown 给 TRAE AI。

## 设计亮点

- **员工优先**：心理安全提示、权限边界说明、个人收益表达。
- **现代视觉**：参考 Linear / Lattice / Notion / Stripe Dashboard，深色侧边栏 + 玻璃拟态顶栏。
- **数据可视化**：纯 SVG 内嵌雷达图、面积趋势图、能力分布条形图，无外部依赖。
- **主题切换**：亮色/暗色双主题，完整 token 系统。
- **离线可用**：所有数据保存在浏览器 `localStorage`，支持 JSON 导入导出。

## 技术栈

纯 HTML + CSS + JavaScript，零依赖，零构建步骤。
