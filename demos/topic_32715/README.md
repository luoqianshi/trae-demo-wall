# 小白 AI 内容工作台

## 作品简介

这是一个面向新手创作者、兼职创作者、宝妈内容创作者的轻量内容工作台。我们相信很多人不是不会写，而是卡在选题、结构、初稿、修改、发布检查这些环节。这个工具帮助新手创作者把一个零散选题，整理成可发布的内容方案。

## 解决的问题

很多新手创作者面临以下痛点：

- 想写但不知道从哪里开始
- 工具太多反而不知道怎么用
- 选题、结构、初稿、修改、发布检查散落在不同工具里
- 宝妈或兼职创作者时间有限，需要短流程

小白 AI 内容工作台将这些环节整合在一个页面中，让新手创作者能够快速完成从选题到发布的完整流程。

## 核心功能

1. **输入选题** - 在首页输入一个零散的选题想法
2. **选择用户身份** - 选择新手宝妈、兼职创作者或 AI 写作新手
3. **选题室** - 生成选题分析，包括适合人群、推荐角度、文章结构、标题候选
4. **初稿室** - 生成全文初稿，包括标题、开头、正文段落和结尾
5. **小花审稿室** - 对初稿进行检查，包括选题、逻辑、人味和发布风险
6. **二改室** - 根据审稿建议生成更自然的发布版本
7. **发布台** - 进行发布前确认，包括标题清楚、开头真实、没有夸大、已准备封面
8. **复制方案** - 支持复制完整方案或只复制二改稿
9. **保存记录** - 保存创作记录，方便下次查看

## 体验方式

1. 双击 index.html 打开
2. 输入或选择一个示例选题
3. 点击"开始整理"
4. 依次查看 5 个流程工位：选题室、初稿室、小花审稿室、二改室、发布台
5. 在发布台完成点灯确认
6. 尝试复制二改稿或保存记录
