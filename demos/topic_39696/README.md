# AI 面试官 - 智能面试模拟器 Demo

## 项目简介

AI 面试官是一款面向求职者的智能面试模拟工具，帮助用户随时随地练习面试，获得专业反馈。本 Demo 为 TRAE AI 创造力大赛初赛作品。

## 功能特性

### 1. 面试场景配置
- 4 大行业：互联网、金融、公务员、教师
- 多岗位选择：产品经理、前端开发、银行柜员、公务员、教师等
- 4 种面试官风格：温和型、严厉型、HR 型、技术专家
- 3 种时长：15/30/45 分钟

### 2. 智能面试
- 预设题库：通用题 + 行业题，共 50+ 道真题
- AI 提问：按行业和岗位随机抽题
- 语音朗读：支持 TTS 朗读问题
- 语音输入：支持 Web Speech API 语音识别回答
- 计时器：实时显示剩余时间

### 3. 实时评分
- 表达流畅度：基于回答长度和展开程度
- 逻辑结构：检测逻辑连接词使用
- 专业度：检测专业术语关键词
- STAR 法则：检测情境-任务-行动-结果结构
- 逐题改进建议

### 4. 面试报告
- 总分及评语
- 能力雷达图（SVG 绘制）
- 弱点分析与改进建议
- 历史记录（localStorage 持久化）

## 技术栈

- HTML5 + CSS3 + 原生 JavaScript（无框架依赖）
- Web Speech API（语音识别 + 语音合成）
- localStorage（历史记录存储）
- SVG（雷达图绘制）
- 响应式设计（支持手机/电脑）

## 使用方法

1. 用 Chrome 或 Edge 浏览器打开 `index.html`
2. 在首页选择行业、岗位、面试官风格、面试时长
3. 点击"开始面试"进入面试界面
4. 阅读问题，输入回答（支持语音输入）
5. 提交回答后查看本题评分和反馈
6. 完成所有题目后查看面试报告
7. 报告页可查看能力雷达图、弱点分析和历史记录

## 浏览器要求

- 推荐：Chrome 25+ 或 Edge 79+（支持 Web Speech API）
- 语音输入功能需要浏览器支持 SpeechRecognition
- 语音朗读功能需要浏览器支持 SpeechSynthesis
