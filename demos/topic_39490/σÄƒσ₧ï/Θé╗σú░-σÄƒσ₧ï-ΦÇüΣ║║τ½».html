页面说明

IoT 自动触发 · P7

老人不操作，设备自动检测异常。这是产品最高优先级的安全保障，覆盖"老人突然晕倒/失去意识"等极端场景。

3 种异常识别

1

姿态突变

雷达检测到快速高度变化（可能是跌倒）

2

长时间不动

卧室 30 分钟无活动 + 卫生间长时间无响应

3

多设备交叉验证

卧室+客厅+卫生间同时无活动 → 置信度 ≥ 95%

为什么不碰硬件？

✓ 集成 主流 IoT 平台
✓ 开放 API 接口
✓ 轻量 不生产硬件
✓ 灵活 老人已有设备可直接用
