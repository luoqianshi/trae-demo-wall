DT Navigation 是一个智能书签导航插件，由于是浏览器插件所以不能在 WEB 环境使用核心功能，这里只作基本展示。

除浏览器自身安全的底层协议不开放功能不支持外，大部分浏览器自带的浏览器书签管理功能都支持，并且可以和浏览器自身的书签数据实时双向同步变更，包括：

  • 添加、收藏、删除、编辑、拖拽移动、导出、导入书签
  • 可一键删除全部书签、书签各种打开方式
  • 搜索书签，书签和使用中的各种数据统计

设计初衷是没碰到一款浏览器插件可以实现实时以导航方式展现出来的，直观的操作带来的边际效应无需多说，并支持 MCP。

理论支持 10W+ 书签。

面向所有互联网用户，特别是深度知识工作者。

MCP

MCP (Model Context Protocol) 是连接 AI 助手与外部数据的开放协议。

通过 MCP，AI 可以使用项目中书签管理的主要功能，大白话直接控制书签各种操作：打开、新建、删除、添加、批量删除、重新分类整合，数据统计分析、去重、历史……更多，项目主要功能都使用 MCP 服务来操作，可以让大模型只需要理解意图和下达指令，最大化减少 TOKEN 消耗。

有 AI 的管理，收藏夹才是未来知识库。
