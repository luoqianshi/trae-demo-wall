# AI 错题整理助手

> 拍一张照片,错题自动归类 —— 面向中小学生的免费 AI 错题整理工具

## 产品简介

AI 错题整理助手是一款网页端工具,通过拍照上传试卷错题,AI 自动识别题干并按知识点归类,一键生成可打印的错题集,并提供学情分析报告。解决手工抄写耗时、纸质本易丢失、复习查找低效的痛点。

- **赛道**:学习工作
- **目标用户**:中小学生、初高中家长、学科辅导老师
- **核心价值**:免费、轻量化、零门槛,将错题整理时间从 1 小时压缩至 3 分钟

## 核心功能

| 页面 | 功能 |
|------|------|
| 首页 | 产品介绍、核心流程演示、学情分析预览 |
| 拍照整理 | 拍照/上传试卷、AI 识别题干、知识点自动归类、置信度展示、编辑确认 |
| 错题集 | 多维筛选(学科/难度/知识点)、卡片浏览、批量选择、A4 打印预览导出 |
| 学情分析 | 学科分布环形图、薄弱知识点排行、难度分布、智能复习建议、整理趋势 |

## 技术栈

- **框架**:React 18 + TypeScript + Vite
- **样式**:TailwindCSS(自定义"数字笔记本"设计系统)
- **状态管理**:Zustand + localStorage 持久化
- **路由**:React Router v7
- **图标**:lucide-react
- **图表**:纯 SVG 手绘(零依赖)

## 本地开发

```bash
npm install
npm run dev      # 启动开发服务器
npm run build    # 生产构建
npm run preview  # 预览构建结果
```

## 设计语言

- **主色调**:深墨蓝(#1B2A4E) + 暖米纸色(#F5F1E8),营造"数字笔记本"质感
- **强调色**:荧光黄(#FFE066)高亮重点,珊瑚红(#FF6B6B)标记错题
- **字体**:标题 Fraunces 衬线体(教科书气质),正文 Plus Jakarta Sans
- **质感**:纸张纹理背景、扫描线动画、墨水晕染效果

## 体验流程

1. 点击「拍照整理」→ 选择「用示例照片演示」体验完整 AI 识别流程
2. 核对识别结果(学科、题干、知识点、难度)→ 保存到错题集
3. 在「错题集」中筛选浏览 → 勾选错题 → 打印预览导出 PDF
4. 在「学情分析」查看学科分布、薄弱知识点和 AI 复习建议
