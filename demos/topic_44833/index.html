完整流程不是“生成一段话”，而是生成一套角色资产

          
PersonaForge 的核心是结构化角色 schema，后续可以接图片、语音和游戏引擎。

        


        

          

            输入偏好
            
自然语言、标签、参考角色、目标游戏类型都可以作为输入。

          

          

            语义拆解
            
把模糊审美拆成风格、体态、性格、声线、用途和禁用边界。

          

          

            生成角色包
            
输出人设、捏脸蓝图、视觉提示词、服装、对白和行为钩子。

          

          

            预览微调
            
通过角色舞台和参数面板快速调整方向，保留多个变体。

          

          

            导出分享
            
导出 JSON、Markdown、角色卡和引擎友好的数据格式。

          

        

      


      

        

          
它可以从一个报名原型，长成一个角色内容生态

          
同一个角色包，可以被玩家收藏，被创作者 remix，被开发者导入项目。

        


        

          

            玩家端
            
角色预设商店

            
玩家购买或收藏角色包，一键套用到支持的游戏、头像系统或社区展示页。

          

          

            创作者端
            
角色卡工作台

            
为短视频、互动小说、AI 绘图和虚拟人设提供稳定角色档案。

          

          

            开发者端
            
角色资产导出

            
面向独立游戏团队提供批量角色配置、NPC 模板和引擎导出。

          

          

            社区端
            
UGC 角色库

            
用户分享、评分、二创和 remix 角色包，形成持续更新的内容池。

          

        

      


      

        

          
差异化：不是普通 prompt 生成器

          
普通生成器只能给一句描述，PersonaForge 输出可复用、可导出、可协作的角色结构。

        


        

          

            Schema
            
结构化角色字段

            
把角色拆成可维护字段，方便版本管理、批量生成和引擎导入。

          

          

            Safety
            
安全标签边界

            
避免低俗和侵权方向，把角色生成控制在适合公开平台和游戏发行的范围内。

          

          

            Export
            
多格式导出

            
同时支持角色卡、提示词、声音脚本、JSON 和后续引擎格式。

          

          

            Loop
            
可运营内容循环

            
从单次生成扩展到收藏、分享、付费模板和社区 remix。

          

        

      


      

        

          
落地路线

          
报名阶段展示创意和体验骨架，后续逐步接入真实生成能力。

        


        

          

            报名阶段
            
单文件 HTML 概念原型，展示核心交互、角色 schema 和商业闭环。

          

          

            初赛阶段
            
做成 Web Demo，支持自然语言输入、标签抽取、角色卡生成和本地保存。

          

          

            复赛阶段
            
接入图片生成、声音试听、批量变体和角色卡分享页。

          

          

            决赛阶段
            
加入 UGC 角色库、模板市场和 Unity/Unreal 导出实验。

          

        

      


      

        

          
让角色创建从“调参数”变成“造角色”

          
PersonaForge 的目标是把玩家的审美、创作者的灵感和开发者的数据格式放在同一个工作台里，让每一个角色都能被预览、复用、分享和继续生长。
