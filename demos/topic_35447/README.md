# AI 知识点讲解助手 — 使用说明

## 产品简介

AI 知识点讲解助手是一个 Web 应用，用户浏览/搜索/输入任意知识点，一键生成多风格交互式讲解页面。

## 如何体验

1. 解压 ZIP 包
2. 用浏览器打开 `index.html`
3. 从左侧浏览或搜索知识点，点击即可生成讲解
4. 也可以在左下角"自定义知识点"输入框中输入任意知识点

## 核心功能

- **知识点浏览**：9 学科、541 个知识点，按学科→模块→单元→知识点树形浏览
- **搜索**：输入关键词实时搜索知识点
- **自定义输入**：输入知识库外的任意知识点，使用通用模板生成讲解
- **双风格讲解**：每个知识点提供「常规讲解」和「讲给 Alice 听」（故事化讲解）两种风格
- **常见错误**：汇总每个知识点的常见错误表现
- **典型例题**：配套典型例题练习
- **前置知识**：展示学习路径
- **公式渲染**：MathJax 渲染数学公式（需联网加载 CDN）
- **打印功能**：支持打印讲解页

## 技术说明

- 纯 HTML/CSS/JS，无后端依赖
- 知识库数据内嵌，离线可用（公式渲染需联网）
- 响应式设计，支持桌面和移动端

## 数据来源

知识库包含 541 个知识点，覆盖数学、语文、英语、科学、物理、化学、生物、历史、道德与法治 9 个学科，五年级数据最完整。
