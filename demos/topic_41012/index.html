第2课 基础

变量与计算

学习如何定义变量，并进行简单的数学计算

本课知识点

1 什么是变量？

变量就像一个"盒子"，可以用来存放数据。我们可以给盒子起个名字，以后用这个名字就能取出里面的数据。

年龄 = 18 // 把18存进叫"年龄"的盒子
2 数学计算

变量之间可以进行数学运算：加(+)、减(-)、乘(×)、除(÷)

结果 = 10 + 5 // 结果是15
3 打印变量

用 打印() 可以把变量的值显示出来

打印(年龄) // 显示18
