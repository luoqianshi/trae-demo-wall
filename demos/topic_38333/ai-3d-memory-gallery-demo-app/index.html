核心功能

照片转 3D 模型

上传单张或多张照片，AI 自动识别主体并生成高精度 3D 模型。支持人物、宠物、物品、建筑等多种类型。

数字展柜展示

把生成的 3D 模型放入主题展柜（旅行墙、宠物小屋、家庭书架），按时间或主题整理，打造个人数字回忆空间。

导出建模文件

一键下载为 .OBJ、.GLB、.STL 等标准格式，支持 3D 打印、游戏开发、视频制作、AR/VR 展示等专业用途。

使用说明

模拟模式

无需任何 API Key，直接从照片生成风格化 3D 几何体。适合快速体验完整流程、测试 UI 功能。生成的模型可用于 3D 预览和下载，但精度不如真实 AI 引擎。

腾讯混元 3D

国内领先的 3D 生成大模型，支持图片转 3D、文生 3D，输出格式丰富（OBJ/GLB/STL/USDZ/FBX）。需要腾讯云账号并开通混元 3D 服务。

Meshy AI

全球领先的 AI 3D 生成平台，支持图片转 3D 和文本转 3D，生成速度快（约 1 分钟），纹理质量高。需要订阅专业版或更高版本才能使用 API。

Tripo3D

操作友好的 AI 建模平台，支持图片和文本生成 3D，生成速度快，拓扑结构良好。提供基础免费试用积分。

安全说明

所有 API Key 仅保存在浏览器本地存储（localStorage），不会上传到任何服务器。清除浏览器数据或点击「清除所有 API Key」即可删除。
