# AI Career Copilot - 技术架构文档

## 1. 架构设计

```
┌─────────────────────────────────────┐
│           单页HTML应用                │
├─────────────────────────────────────┤
│  UI Layer: HTML5 + CSS3 + Vanilla JS│
├─────────────────────────────────────┤
│  Data Layer: 内置Mock数据            │
└─────────────────────────────────────┘
```

## 2. 技术栈

- **前端**: 纯HTML5 + CSS3 + Vanilla JavaScript
- **无后端**: 所有功能在前端实现
- **无框架依赖**: 单文件完整实现

## 3. 页面结构

| 模块 | 功能 | 技术实现 |
|------|------|----------|
| 输入模块 | 收集简历和岗位信息 | HTML Form + JS事件处理 |
| 分析模块 | 显示匹配度和优劣势 | CSS卡片 + 动态内容渲染 |
| 面试模块 | 展示面试问题和答案 | 手风琴展开/收起效果 |
| 成长模块 | 4周计划时间线 | CSS Grid时间线布局 |

## 5. 交互设计

- 点击"开始分析"触发模拟分析流程
