# 🤖 公益随手做 VolunteerMate

> 让每一次小小的善意，都成为改变世界的力量

一个基于 AI 的公益行动管理应用，帮助用户在日常生活中发现、记录和分享公益行为。

## 🎯 核心亮点

| 亮点 | 说明 |
|------|------|
| 🤖 **真实 AI 接入** | 通过硅基流动 SiliconFlow 的 `nex-agi/Nex-N2-Pro`（限免模型）提供真实 AI 对话，智能推荐任务、AI 问答、AI 画像分析一站式集成 |
| 📱 **5 个完整页面** | 首页/AI 助手/日历/知识/个人中心，每个页面都有独立的功能、数据和交互 |
| 🎮 **游戏化设计** | 7 日公益挑战、18 个成就徽章、公益排行榜、等级成长系统 |
| 📊 **数据可视化** | 使用 Recharts 展示公益时长趋势、分类分布饼图、能力雷达图 |

## 💭 产品理念

> 「我们无法做伟大的事，只能用伟大的爱去做小事。」—— 特蕾莎修女

### 随手公益：让善意无处不在

VolunteerMate 的核心理念是**「降低公益门槛」**。传统公益常常需要专门抽时间、花费金钱，这让很多人望而却步。我们相信，公益不应该是"一个需要专门计划的大事"，而应该是"融入日常的一种生活方式"。

**把公益拆解成 5-30 分钟的微任务**：垃圾分类、整理旧物、分享一条公益信息——这些事每个人都可以做到，不需要等待"有时间再说"。

### 游戏化激励：让坚持变得有趣

做一次好事容易，持续做很难。这正是游戏化设计发挥作用的地方：

- **成就徽章**：给每一步付出一个看得见的印记
- **等级成长**：让累计投入的时间有量化的反馈
- **连续打卡**：用"不能断"的心理培养习惯
- **排行榜**：用适度竞争激发参与感

这不是为了让公益变得功利，而是为了让善意有一个"不被遗忘"的容器。

### AI 作为桥梁，而非终点

AI 在本产品中的定位是**「助手和鼓励者」**，而非"替你做公益的工具"。它帮你：
- 找到适合你的切入点
- 解答你对公益的困惑
- 用温暖的语言给你持续的动力

真正的行动，永远来自你自己。AI 只是那个"在你需要时递上一把火"的角色。

### 单文件部署：让好作品被看见

为什么坚持做单文件构建？因为我们相信，一个公益产品本身也应该遵循"方便他人"的原则。评委/用户不需要安装环境、不需要配置后端，**双击就能体验完整功能**——这本身就是我们对"降低门槛"理念的践行。

## 🎯 真实场景

现在的公益活动门槛太高：大型慈善活动需要专门抽时间，而"随手公益"的理念虽好，但用户缺乏引导和记录工具。VolunteerMate 让公益可以像记日记一样自然发生。

**核心用户痛点**：
1. 想做公益但不知道从哪里开始，缺乏行动引导
2. 公益行为无法量化和记录，缺少成就感
3. 习惯难以维持，需要鼓励和社交激励
4. 公益知识匮乏，不了解如何有效参与

**VolunteerMate 的解决方案**：
- 把公益拆解成 5-30 分钟的微任务
- AI 根据用户画像智能推荐，匹配最适合的行动
- 打卡 + 成就 + 排行榜，游戏化设计提升参与感
- AI 助手随时解答公益相关问题

### AI 功能设计说明

本项目创新性地采用 **"双轨 AI 架构"**：

1. **规则 AI 轨**：内置公益知识库和推荐算法，无网络也能用；
2. **真实 AI 轨**：配置 API Key 后，自动升级为真实大模型对话；

两种模式共享同一套 UI 和数据接口，用户无感切换。设计兼顾了竞赛作品的"必须有 AI"要求和产品实用性。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 生产构建（单文件输出）
npm run build

# 本地预览
npm run preview
```

打开浏览器访问显示的本地地址即可体验完整功能。

### 配置真实 AI（可选）

1. 进入 APP → 底部"AI 助手" → 右上角"⚙️ 设置"
2. 选择一个 API 提供商
3. 选择模型
4. 粘贴 API Key
5. 点击"测试并启用"

推荐先使用**智谱**（glm-4-flash 完全免费）或**硅基流动**（新用户有免费额度）。
