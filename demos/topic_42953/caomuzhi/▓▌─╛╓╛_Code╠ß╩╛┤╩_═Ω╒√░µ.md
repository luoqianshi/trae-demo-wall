## 🏗️ P0-1：项目骨架与惊艳 UI 设计系统

**提示词：**

```
请帮我搭建一个名为"草木志"的 Web 应用项目骨架，使用纯 HTML + CSS + JavaScript（不依赖任何框架），要求：

1. 项目结构：
   - index.html（首页/识别页）
   - detail.html（知识详情页）
   - collection.html（我的图鉴页）
   - login.html（登录/注册页）
   - profile.html（个人中心页）
   - css/style.css（统一样式，含完整设计系统）
   - css/animations.css（动画库）
   - js/data.js（草木数据，200种常见植物的JSON数组）
   - js/app.js（核心逻辑）
   - js/auth.js（用户认证逻辑）
   - js/ai-mock.js（模拟AI识别）
   - assets/images/plants/（植物图片目录）
   - assets/images/ui/（UI装饰元素）
   - assets/icons/（图标）

2. 设计系统（必须惊艳、丝滑、自然）：
   - 主色调：#2D5A3D（森林绿）#1A3A2A（深绿）#4A7C59（中绿）
   - 辅色：#D4A574（琥珀金）#F5E6D3（暖米色）#C4956A（古铜）
   - 背景色：#FAF7F2（米白）#F0EBE3（浅灰米）
   - 文字色：#1C1C1E（深黑）#6B6B6B（中灰）#A1A1A6（浅灰）
   - 强调色：#E07A5F（珊瑚红，用于收藏/警告）#81B29A（薄荷绿，用于成功）
   
   - 字体：中文使用 "Noto Sans SC" 或系统默认，英文使用 "Inter" 或 "SF Pro Display"
   - 圆角：卡片 16px，按钮 12px，标签 20px
   - 阴影：卡片使用多层柔和阴影（0 4px 20px rgba(45,90,61,0.08)）
   - 间距：采用 8px 网格系统

3. 全局动画效果（必须丝滑）：
   - 页面切换：淡入 + 轻微上移（opacity 0→1, translateY 20px→0, 0.4s ease-out）
   - 卡片悬停：上浮 8px + 阴影加深 + 微缩放 1.02（0.3s cubic-bezier(0.4, 0, 0.2, 1)）
   - 按钮点击：缩放 0.96 + 涟漪效果
   - 加载状态：叶子旋转动画（SVG叶子图标旋转 + 呼吸效果）
   - 滚动触发：元素进入视口时依次淡入（stagger 0.1s）
   - 背景：微妙的动态渐变（米白到浅绿的极缓慢渐变动画）

4. 导航栏设计：
   - 顶部固定导航，滚动时背景变为毛玻璃效果（backdrop-filter: blur(20px)）
   - 左侧：Logo "草木志" + 叶子图标
   - 中间：首页 | 识别 | 图鉴 | 古籍（未登录时隐藏图鉴和古籍）
   - 右侧：未登录显示"登录/注册"按钮；已登录显示用户头像下拉菜单（个人中心 | 我的图鉴 | 退出）

5. 页脚设计：
   - 深绿色背景（#1A3A2A），白色文字
   - 包含：关于我们 | 联系方式 | 隐私政策 | 版权声明
   - 底部："用 AI 探索草木世界" 标语

请生成完整的文件结构和基础代码，确保UI视觉效果惊艳、丝滑。
```

## 🔐 P0-2：用户注册登录系统

**提示词：**

```
请帮我实现 login.html 的完整用户注册登录系统，要求：

1. 页面设计（必须美观）：
   - 左侧：全屏植物插画背景（使用CSS渐变+SVG叶子图案模拟），中央大标题"探索草木世界"
   - 右侧：登录/注册表单卡片（毛玻璃效果背景）
   - 表单卡片有微妙的浮动动画（上下轻微浮动，呼吸感）

2. 登录功能：
   - 字段：手机号（带格式验证）+ 密码（可切换显示/隐藏）
   - "记住我"复选框
   - "忘记密码"链接（模拟，点击提示"请联系客服"）
   - 登录按钮：全宽，渐变背景（森林绿到薄荷绿），悬停有光晕效果
   - 第三方登录：微信、QQ图标（模拟，点击提示"功能开发中"）

3. 注册功能：
   - 切换方式：登录/注册通过滑动切换（卡片翻转或滑动动画）
   - 字段：手机号 + 验证码（模拟，点击发送后倒计时60秒）+ 密码 + 确认密码
   - 昵称字段（注册后自动生成默认昵称"草木探索者"+随机数字）
   - 同意用户协议复选框
   - 注册成功后自动登录，跳转首页

4. 数据存储：
   - 使用 localStorage 存储用户信息（模拟后端）
   - 用户数据结构：{ id, phone, nickname, avatar, password, createdAt, collections: [], history: [] }
   - 登录状态：token（随机字符串）存储在 localStorage
   - 密码使用简单加密（btoa/atob 或自定义混淆）

5. 全局认证状态：
   - js/auth.js 提供全局函数：isLoggedIn(), getCurrentUser(), login(), register(), logout()
   - 所有页面引入 auth.js，导航栏根据登录状态自动切换
   - 未登录用户访问需要登录的页面（图鉴、个人中心）时，自动跳转登录页

6. 视觉细节：
   - 输入框聚焦时有绿色光晕边框动画
   - 表单验证错误时有抖动动画 + 红色提示文字
   - 成功操作有绿色对勾动画反馈

请生成完整的 HTML + CSS + JS 代码。
```
