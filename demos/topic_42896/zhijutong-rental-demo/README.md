# 智居通租房 Demo

这是一个纯前端 HTML demo 包，用来演示“租房范围内的智能房产托管与自助看房平台”。

## 怎么打开

直接打开 `index.html` 即可运行。

如果浏览器限制本地文件，可以在该目录下启动一个简单静态服务：

```bash
python3 -m http.server 8080
```

然后在浏览器访问：

```text
http://localhost:8080
```

## 当前包含的功能

1. 用户可以切换“我要租”和“我要出租”身份。
2. 用户界面只显示当前身份对应的视角。
3. 房主完成身份认证后，可以填写房源标题、区域、价格、面积、标签和描述。
4. 房主可以添加 demo 视频和 demo VR。
5. 摄像头和智能门锁使用 dummy 状态模拟。
6. 平台审核使用自动通过的模拟流程。
7. 租客可以浏览、搜索和筛选已认证房源。
8. 租客和房主默认都是已认证状态。
9. 租客预约时需要选择 30 分钟时间段。
10. 同一房源同一时间段不可重复预约。
11. 租客和房主都能在“预约”中看到预约时间，并可以开启聊天。
12. 预约列表默认包含一个“看房中”和一个“未到时间”的 demo 预约。
13. 所有预约时间都固定在整点或半点边界，例如 10:00-10:30、10:30-11:00。
14. 只有在预约时间内，租客才会看到一次性门锁密码。
15. demo 状态变化会输出到浏览器控制台。
