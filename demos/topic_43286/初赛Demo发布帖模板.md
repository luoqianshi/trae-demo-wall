# 【社会服务】声脉方志：非遗方言口述存档工具 Demo

## 1. Demo 简介

声脉方志是一套面向非遗方言口述采集的交互式 Web Demo，核心目标是帮助普通家庭、非遗机构、学校研学团队和地方文旅部门，把老人讲述的方言民俗、老故事、地方传说转化为可读、可听、可检索、可展陈的数字馆藏。

核心用户包括：

- 非遗保护机构、地方文旅部门、博物馆与档案馆
- 高校民俗学 / 方言学团队、中小学研学团队
- 返乡青年、亲子家庭、地方内容创作者

Demo 已实现的主要功能：

- 口述采集：选择讲述人、地点、方言类型和采集主题，模拟老人方言口述录制。
- AI 转写整理：自动生成方言原文、普通话释义、关键词标签和古籍关联说明。
- 数字馆藏生成：一键生成包含原声线索、转写文本、白话释义和古籍活化内容的馆藏页。
- Web 运营后台预览：展示机构端素材审核、专题运营和数字馆藏管理场景。

## 2. Demo 创作思路

灵感来自“少小离家老大回，乡音无改鬓毛衰”。乡音不是简单的发音差异，而是一代人的生活证词。很多老人会讲地方婚嫁、节令饮食、童谣、传说和手艺故事，但这些内容往往只存在于家庭记忆中，缺少低门槛、标准化、可持续的采集工具。

我想解决的问题是：传统非遗记录依赖专业团队，普通家庭不知道怎么问、怎么录、怎么整理；机构采集到的录音也常常分散在文件夹、表格和文档里，难以形成可检索、可展示、可传播的数字馆藏。同时，地方志、族谱和古籍中的民俗记录很难被年轻人理解和使用。

因此这个 Demo 选择“非遗 + 古籍活化”双方向：一边记录老人原声和方言民俗，一边把口述内容与古籍、地方志、族谱中的相关条目连接起来，让声音成为进入地方文化的入口。

## 3. Demo 体验地址

体验方式：交互式 HTML 文件。

请下载并解压 Zip 包，双击打开 `index.html` 即可体验。

建议体验路径：

1. 进入首页，点击“开始体验采集流程”。
2. 在“口述采集工作台”中选择采集主题，例如“清明做青团”。
3. 点击“模拟录音并转写”，查看 AI 转写、方言注释和古籍匹配结果。
4. 点击“生成数字馆藏页”，查看最终馆藏效果。
5. 切换到“运营后台”，查看机构端管理能力。
