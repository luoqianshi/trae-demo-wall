# AI 同声传译助手 — 前端执行计划

> 更新日期：2026-06-04
> 基于最新技术栈：Electron + React + WebSocket

---

## 一、目标

构建一个跨平台的桌面字幕应用，能够：
- 实时显示翻译字幕（透明叠加层，可悬浮于任何窗口上方）
- 展示自动纠错过程（修正内容以动画高亮提示）
- 提供控制面板（启动/停止、语言切换、样式设置）
- 支持语音播报开关

---

## 二、技术栈

| 技术 | 版本/说明 | 用途 |
|:---|:---|:---|
| **Electron** | 最新稳定版 | 桌面应用框架，透明窗口 |
| **React** | 18+ | UI 组件开发 |
| **WebSocket (原生)** | 浏览器内置 | 与后端实时通信 |
| **CSS3** | 动画/透明/毛玻璃 | 字幕样式与动效 |

---

## 三、模块分解

### 模块 1：Electron 主进程框架

| 项 | 说明 |
|:---|:---|
| **Purpose** | 搭建 Electron 应用骨架，创建透明叠加窗口 |
| **Input** | 无 |
| **Output** | 可运行的 Electron 窗口（透明、无边框、置顶） |
| **Dependency** | 无 |

**关键实现**：
- `BrowserWindow` 配置：`transparent: true`, `frame: false`, `alwaysOnTop: true`, `skipTaskbar: true`
- 支持拖拽移动窗口位置
- 支持鼠标穿透（点击穿透到下方应用）

---

### 模块 2：字幕渲染组件

| 项 | 说明 |
|:---|:---|
| **Purpose** | 实时显示翻译文本，支持流式逐字输出 |
| **Input** | WebSocket 推送的翻译文本片段 |
| **Output** | 可视化字幕展示 |
| **Dependency** | 模块 1 |

**关键实现**：
- 字幕区域：居中底部显示，最大宽度 80%
- 流式效果：新文本平滑淡入
- 自动换行与字体大小自适应
- 支持多行字幕保留（显示最近 2~3 句）

---

### 模块 3：纠错可视化组件

| 项 | 说明 |
|:---|:---|
| **Purpose** | 当后端推送修正内容时，以动画高亮提示用户 |
| **Input** | 后端推送的 `{type: "correction", oldText, newText}` |
| **Output** | 被修正的文字以颜色变化/闪烁动画呈现 |
| **Dependency** | 模块 2 |

**关键实现**：
- 修正内容背景色渐变（黄 → 绿）
- 1.5 秒后恢复正常样式
- 可选：在字幕旁显示"已修正"小标记

---

### 模块 4：控制面板窗口

| 项 | 说明 |
|:---|:---|
| **Purpose** | 提供用户操作界面，管理翻译会话 |
| **Input** | 用户点击/输入 |
| **Output** | 发送控制指令到后端（启动/停止/配置） |
| **Dependency** | 模块 1 |

**关键实现**：
- 独立窗口，常规样式（非透明）
- 功能按钮：启动翻译 / 停止 / 暂停
- 语言选择：源语言（英语）→ 目标语言（中文）
- 字幕样式设置：字体、大小、颜色、位置
- API Key 配置输入框（DeepSeek）
- TTS 播报开关

---

### 模块 5：WebSocket 通信层

| 项 | 说明 |
|:---|:---|
| **Purpose** | 与后端 FastAPI WebSocket 建立实时连接 |
| **Input** | 后端推送的翻译/纠错/状态消息 |
| **Output** | 向前端组件分发数据 |
| **Dependency** | 模块 1 |

**关键实现**：
- 自动重连机制
- 心跳保活
- 消息类型解析：`translation` | `correction` | `status` | `error`

---

### 模块 6：设置与状态管理

| 项 | 说明 |
|:---|:---|
| **Purpose** | 管理用户配置和应用状态 |
| **Input** | 用户设置 / 后端状态 |
| **Output** | 持久化配置（localStorage / electron-store） |
| **Dependency** | 模块 4 |

**关键实现**：
- 配置项：API Key、语言对、字幕样式、TTS 开关
- 状态：连接状态、翻译状态、错误信息

---

## 四、文件结构

```
frontend/
├── package.json
├── electron.js                 # 主进程入口
├── preload.js                  # 预加载脚本（安全通信）
├── src/
│   ├── index.jsx               # React 入口
│   ├── App.jsx                 # 根组件
│   ├── components/
│   │   ├── SubtitleOverlay.jsx    # 字幕叠加层（主窗口）
│   │   ├── SubtitleLine.jsx       # 单条字幕组件
│   │   ├── CorrectionHighlight.jsx # 纠错高亮动画
│   │   └── ControlPanel.jsx       # 控制面板
│   ├── hooks/
│   │   ├── useWebSocket.js        # WebSocket 连接管理
│   │   └── useSettings.js         # 设置状态管理
│   └── styles/
│       └── subtitle.css           # 字幕样式（透明/动画）
└── public/
    └── index.html
```
