Yerpx Pixel Studio - Free Online Pixel Art Editor

Web-based Pixel Art Maker | Frame Animation Tool | GIF / APNG / Sprite Sheet Exporter

Yerpx Pixel Studio is a free, web-based pixel art and frame animation editor for game developers, pixel artists, and animation hobbyists. It supports unlimited layers, onion skin preview, mirror / symmetry brush, color palette management, undo / redo, and exports to GIF, APNG, PNG sequence, and sprite sheet. No registration, no download required - open https://px.yerpx.com/ in any modern browser to start drawing immediately.

Yerpx is similar to Aseprite and Piskel, but runs entirely in the browser and is completely free. Unlike Aseprite, there is no purchase required. Unlike Piskel, no account is needed and analytics are self-hosted (no third-party tracking).

  • Pixel-perfect drawing with adjustable brush size (1-8 pixels)

Yerpx FAQ

Is Yerpx Pixel Studio free? Yes, completely free with no ads and no in-app purchases.

Do I need to register? No registration is required.

Do I need to install anything? No, it runs in the web browser.

Where is my artwork stored? All artwork is stored locally in the browser using IndexedDB. Nothing is uploaded to a server.

What export formats are supported? GIF animation, APNG animation, PNG sequence (zip), sprite sheet with metadata, and Data URI.

What import formats are supported? PNG and JPG images can be imported as reference or starting point.

Is my data private? Yes. The tool does not collect personal information, does not set cookies, and respects the browser Do Not Track header. Only anonymous usage statistics are collected via a self-hosted Umami instance.

What browsers are supported? Any modern browser including Chrome, Edge, Firefox, Safari, and Opera.

Is there a mobile version? The site is optimized for desktop. Mobile touch support is not currently planned.

Yerpx Use Cases

Yerpx is suitable for: 2D game asset creation, pixel art for indie game developers, retro-style animation, character sprite design, UI icon design in pixel art style, social media emojis and stickers, game jam projects, teaching pixel art fundamentals, frame-by-frame animation learning, and creating animated GIFs for messaging apps.

Yerpx vs Other Pixel Art Tools

Yerpx Pixel Studio differs from Aseprite in that it is web-based and free. It differs from Piskel in that it requires no account and uses no third-party tracking. It differs from Pixilart in that it does not require login and respects user privacy. It is built with vanilla JavaScript (no heavy frameworks) and is fully open about its data practices.

Technical Information

Built with HTML5 Canvas, vanilla JavaScript, and CSS. No external JavaScript frameworks. Total bundle size approximately 150 KB. Runs entirely client-side. Compatible with screen readers. Hosted on a Chinese server (ICP-filed) and uses self-hosted Umami analytics for privacy-respecting traffic analysis.
