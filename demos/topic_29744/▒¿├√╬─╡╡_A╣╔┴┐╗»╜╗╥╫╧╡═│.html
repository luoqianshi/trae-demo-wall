智能A股量化交易决策系统

基于多维市场信号矩阵与7状态趋势判断的量化交易策略

1. 创意名称 + 创意介绍

想解决什么问题

A股市场情绪波动剧烈，投资者面临三大核心困境：牛市顶部无法识别导致利润大幅回吐、熊市过早抄底被套、V型反弹行情中仓位被恐慌止损反复打断。

现有量化策略普遍依赖单一指标（如均线、RSI），缺乏多信号互证机制，在极端行情中容易失效。本系统通过6轴市场信号矩阵与7状态趋势判定，实现对市场状态的精准分层与动态响应。

为什么会想到做这个

在A股历史回测中发现：2015年牛市顶部仅靠单一均线指标会导致回撤从9%扩大至22%；2020年V型反弹期间，恐慌止损信号频繁触发，本可盈利的交易被反复打断。这说明单一维度的判断在市场转折点存在系统性盲区。

灵感来源于工程领域的冗余设计思维——航空控制系统不会依赖单一传感器，而是多个独立传感器相互校验。本策略同样采用6个独立市场轴（广度、长线、动量、背离、V反、质量）构建冗余判断体系，任何单一信号失效都不会导致系统误判。

大概是什么产品

一个面向A股市场的量化交易辅助决策系统（Web端展示 + Python回测引擎），通过多维市场信号矩阵实时评估市场状态，自动生成买入/卖出/观望的交易决策建议，帮助投资者在牛市最大化收益、在熊市控制回撤、在V型反弹中不被震荡洗出。

2. 目标用户及痛点

面向哪些用户

散户投资者缺乏系统化交易框架，容易被情绪驱动追涨杀跌

有一定经验的交易者有自己的交易系统，但缺乏量化验证和系统化优化能力

量化学习者希望学习多信号协同决策的系统化方法

在什么场景下使用

开盘前：查看当日市场状态评分（7状态）、各轴信号强弱，决定当日交易策略（进攻/防守/观望）

持仓期间：系统自动监控市场广度、趋势动量等指标，在触发止盈/止损条件时提醒

回测验证：用历史数据验证策略有效性，系统自动生成年度收益率、夏普比率、最大回撤等指标

当前痛点（没有这个产品会遇到什么）

信息过载数十个技术指标不知如何综合判断，缺乏决策锚点
情绪干扰牛市贪婪不止盈、熊市恐惧不止损，没有系统约束
指标失效单一指标在极端行情中失效，没有冗余校验
回撤失控牛市赚的钱在顶部回吐大半，缺乏顶部预警
优化困难参数调优靠"感觉"，缺乏系统化的回测验证框架

3. 价值与意义

效率提升

将数十个独立指标整合为6轴评分体系，交易决策从"凭感觉"变为"看评分"，决策效率提升10倍以上。多轴矩阵的冗余设计确保任一指标失效时系统仍可正常工作。

商业价值

系统历史回测显示：2015年牛市收益率达+67.61%（同期上证指数+9.4%），2018年熊市仅亏损-5.33%（同期上证指数-25%），2020年V反行情通过动态止损优化减少损失。

风险管理

最大回撤控制是衡量交易系统的核心指标。本系统通过7状态市场判定 + 崩盘加速度检测 + sf_velocity动量保护的三层风控体系，实现2015年回撤控制在10.59%。

技术普惠

将机构级别的量化策略逻辑透明化、模块化，让普通投资者也能使用系统化的多信号决策框架，降低散户与机构之间的信息不对称。

4. 系统核心架构

三层决策体系

市场层：7状态趋势判定

STRONG_UP → WEAK_UP → RANGE_BULL → RANGE → RANGE_BEAR → WEAK_DOWN → STRONG_DOWN

基于趋势斜率 + above_ma20双条件 + 5维度修正 + 滞回机制，实现状态平滑切换

矩阵层：6轴信号评分

广度(18%) + 长线(18%) + 背离(20%) + 动量(18%) + V反(16%) + 质量(10%)

每轴2-3个子指标互证，3日EMA平滑，任意单轴失效不影响整体评分

6轴信号矩阵（核心创新）

轴名称权重子指标核心作用
轴1市场广度18%above_ma20(55%) + 趋势方向(45%)衡量有多少股票站上均线
轴2长线基础18%above_ma60(55%) + a60斜率(45%)判断长周期趋势是否健康
轴3背离风险20%div差值(50%) + a60健康度(50%)识别假反弹与熊市陷阱
轴4趋势动量18%a10斜率(55%) + a30斜率(45%)衡量短期动量变化方向
轴5V型反转16%恐慌深度(30%) + 恢复比(35%) + 斜率(35%)识别V反并提供卖出松绑
轴6质量/广度10%ma_width_avg(50%) + 均线宽度(50%)衡量均线收敛/发散程度

崩盘加速度检测（v12.54多窗口设计）

窗口阈值含义应对动作
单日断崖above_ma20降>0.20强制清仓信号立即切SURVIVE状态
3日加速累计降>0.10崩盘进行中扣health_score 0.30
5日确认累计降>0.15高置信崩盘扣health_score 0.35
10日阴跌累计降>0.15中期趋势转弱扣health_score 0.12
60日破位累计降>0.25季度级崩盘强制≤LOCK状态
指数辅助3日跌>5%/5日跌>10%指数与广度共振叠加扣分0.25-0.30

6. 核心创新点总结

创新16轴冗余矩阵：不是简单加权平均，而是每个轴内部2-3个子指标互证，任何单一传感器失效最多导致状态降一级而非崩溃。这借鉴了航空系统的冗余设计理念。

创新2动态阈值而非固定值：V型反弹期间卖出阈值自动放宽（crash_exit忽略、P0从0.15降至0.12），避免在恢复行情中被震荡洗出。阈值基于市场状态动态计算。

创新3多窗口崩盘检测：不依赖单一时间窗口，而是1日/3日/5日/10日/20日/30日/60日多窗口协同判断，结合指数级数据交叉验证，区分"正常调整"与"系统性崩盘"。

创新4V反滞回保护：V型反转信号一旦激活，需连续3天不满足条件才退出，防止单日数据波动导致保护中断。
