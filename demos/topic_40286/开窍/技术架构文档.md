### 3.2 OCR 识别模块

**职责**：模拟图片文字识别过程，生成智能标签

**工作流程**：

```
用户上传图片
    ↓
FileReader 读取图片 → 生成预览
    ↓
startOCR() 启动识别流程
    ↓
┌─────────────────────────────────────┐
│  步骤 1: 识别题目文字 (600ms)       │
│  步骤 2: 提取选项内容 (600ms)       │
│  步骤 3: 匹配正确答案 (600ms)       │
│  步骤 4: 生成智能标签 (600ms)       │
└─────────────────────────────────────┘
    ↓
showOCRResult() 展示识别结果
    ↓
用户确认 → saveToBank() 保存到题库
```

**核心代码**：

```javascript
handleImageUpload(file) {
    const reader = new FileReader();
    reader.onload = (e) => {
        // 显示图片预览
        previewImage.src = e.target.result;
        // 延迟启动 OCR
        setTimeout(() => this.startOCR(), 500);
    };
    reader.readAsDataURL(file);
}

startOCR() {
    const steps = ['ocr-step-1', 'ocr-step-2', 'ocr-step-3', 'ocr-step-4'];
    let currentStep = 0;
    
    const interval = setInterval(() => {
        // 更新步骤状态
        if (currentStep > 0) {
            document.getElementById(steps[currentStep - 1])
                .classList.replace('active', 'done');
        }
        if (currentStep < steps.length) {
            document.getElementById(steps[currentStep])
                .classList.add('active');
            currentStep++;
        } else {
            clearInterval(interval);
            this.showOCRResult();
        }
    }, 600);
}
```

### 3.6 组卷引擎模块

**职责**：根据条件从题库中智能选题，生成试卷

**筛选算法**：

```javascript
generatePaper() {
    // 获取筛选条件
    const subject = document.getElementById('paper-subject').value;
    const difficulty = document.getElementById('paper-difficulty').value;
    const knowledge = document.getElementById('paper-knowledge').value;
    const count = parseInt(document.getElementById('paper-count').value);
    
    // 多级筛选
    let filtered = [...this.questionBank];
    
    if (subject !== 'all') {
        filtered = filtered.filter(q => q.subject === subject);
    }
    if (difficulty !== 'all') {
        filtered = filtered.filter(q => q.difficulty === parseInt(difficulty));
    }
    if (knowledge !== 'all') {
        filtered = filtered.filter(q => q.knowledge === knowledge);
    }
    
    // 随机选题（Fisher-Yates 洗牌算法）
    const shuffled = [...filtered].sort(() => Math.random() - 0.5);
    const selected = shuffled.slice(0, Math.min(count, shuffled.length));
    
    this.renderPaper(selected, { subject, difficulty, knowledge });
}
```

**试卷渲染**：

```javascript

    const subjectText = config.subject === 'all' ? '综合' : config.subject;
```
