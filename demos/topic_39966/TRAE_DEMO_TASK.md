# 给 TRAE 的实现任务

请在当前目录实现一个可运行的前端 Demo，主题是「非遗共创展台」。

## 参赛定位

- 大赛：TRAE AI 创造力大赛
- 主赛道：社会服务
- 附加赛题：社会公益 / 非遗创新创意开发
- 目标用户：非遗传承人、文旅志愿者、校园社团、社区文化馆、小型展览组织者
- 核心价值：降低非遗传播和教育内容制作门槛，让小团队快速生成可讲解、可互动、可传播的微展页面。

## 必须完成的体验

1. 页面打开后就是可操作 Demo，不要做营销落地页。
2. 内置「蓝印花布」样例数据，首屏直接展示生成结果。
3. 用户可以修改表单字段并点击生成。
4. 生成内容至少包含：
   - 三分钟讲解词
   - 儿童版解释
   - 互动问答
   - 文创灵感
   - 短视频脚本
   - 微展故事卡
   - 工艺步骤
   - 小测验
   - 传播海报文案
5. 使用纯前端实现，优先保证可演示、响应式和视觉完成度。

## 视觉风格

- 不要做单一蓝紫渐变风。
- 适合文旅/社区展览：温润、清爽、有纸张和印染质感，但不要过度复古。
- 布局要像一个真正的工具：左侧输入，中间生成内容，右侧微展预览。
- 移动端堆叠显示，文字不能溢出按钮或卡片。

## 技术约束

- 使用 `index.html`、`styles.css`、`app.js` 三个文件。
- 不依赖后端和外部 API。
- 生成逻辑用本地模板模拟 AI，按钮点击后更新页面内容。
- 保持代码清晰，便于之后接入真实大模型 API。
